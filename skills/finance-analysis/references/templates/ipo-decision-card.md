# IPO 打新决策卡 Template

> 用法：招股结束当晚（T-1）填写这张卡，T0 开盘按执行计划操作。
> 关联 framework：`frameworks/hk-ipo-arbitrage.md`

填写时 copy 到 `docs/research/YYYY-MM-DD-<ticker>-ipo-card.md`。

---

```markdown
## EV 计算

```

  σ（中签率） = ...（用官方公告或 1/(1+H)*2）
  P1（首日涨幅预期） = +XX%（基于暗盘 × 保守折算）

  L（孖展杠杆） = XX 倍
  rate（孖展年化） = XX%
  days（占用天数） = XX

计算：
  利息/手 = 入场费 × rate × days / 365 = XX HKD
  EV/手 = σ × 每手盈利 - 利息/手 = XX HKD
  申购 N 手 EV = N × EV/手 = XX HKD

## 使用示例：群核科技 0068.HK（2026-04-17 上市）

```markdown
## 基本信息
- 标的：群核科技 (0068.HK)
- 上市日期：2026-04-17
- 填卡日期：2026-04-14 晚
- 招股价：7.62 HKD（区间 6.72-7.62）
- 每手股数：500（入场费 3848 HKD）

## 信号矩阵评分：129/140 → 等级 S（满融）

| 信号 | 数值 | 分 |
|------|------|---|
| 概念 | 杭州六小龙 + 空间智能第一股 | 19/20 |
| 基石（9 家 / 37% / 险资主导） | 优 | 16/20 |
| 保荐人 | 摩根大通 + 建银国际 | 9/10 |
| 定价 | 上限 7.62 | 9/10 |
| 公开超额 | 1591 倍 | 25/25 |
| 国际超额 | 14.46 倍 | 12/15 |
| 暗盘 | +169.3%（盘中 +201.8%） | 20/20 |
| 流通盘 | 9.45% | +5 |
| 大盘 | Q2 港股科技活跃 | +5/±10 |
| 合计 | | 129/140 |

## EV：10x 孖展申 10 手，EV ≈ +1230 HKD

## 执行：T0 开盘挂 20.50 HKD，全部卖出
```
