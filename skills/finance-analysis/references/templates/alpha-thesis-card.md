# Alpha 投资论点卡 Template

> **核心分析能力**：给定一只股票，结构化分析"为什么买 / 什么价买 / 什么信号卖"。
>
> 关联 framework：`frameworks/alpha-generation.md` + `frameworks/ai-thematic-investing.md`（主题股）
>
> 填写时 copy 到 `docs/research/YYYY-MM-DD-<ticker>-thesis.md`。

---

## Alpha 论点 7 问法（分析能力核心）

一张完整的投资论点卡必须回答 7 个问题：

1. **这是什么生意**（business model）
2. **在产业链哪个位置**（positioning）
3. **谁是对手 + 谁更强**（competition）
4. **预期差在哪**（alpha thesis）
5. **怎么赔钱**（bear case / 风险）
6. **什么价买 / 卖**（valuation + triggers）
7. **什么信号证伪**（kill criteria）

**7 问法原则**：任何一问答不出来 = 论点不成立，不买。

## 问 7：证伪条件（kill criteria，铁律）

任一触发 = **清仓**，不讨价还价：

- [ ] 股价跌破止损线 -20%
- [ ] 连续 2 个季度财报 Miss + 指引下调
- [ ] 预期差 1/2/3 中 ≥ 2 条被证伪（事件未兑现 / 证据反了）
- [ ] 核心护城河被突破（如数据飞轮被对手复制）
- [ ] CEO / CFO 离职 + 无合理继任者
- [ ] 财务造假传闻 / 审计事务所辞任 / 做空机构深度报告
- [ ] 主题论点崩塌（如 AI 泡沫破 / 监管重击）

**顶部信号**（逐步减仓）：
- [ ] 打车师傅问这只股（散户 FOMO 峰值）
- [ ] 媒体头条连续覆盖 5 次 / 月
- [ ] 同概念股全部涨停
- [ ] 卖方一致目标价 > 现价 20%+（共识顶）
- [ ] 分析师覆盖家数翻倍（后进者效应）
- [ ] 机构持仓 > 历史 90 分位

## 使用示例：群核科技 0068.HK（长期持有视角，T+6 解禁期后）

```markdown
# Alpha 论点卡 — 0068.HK 群核科技

## 问 1：生意
空间设计 SaaS + 空间智能数据/世界模型双引擎。
2025 营收 8.2 亿 RMB，毛利率 82.2%（顶级 SaaS 水平），经调整净利 5710 万（首次盈利）。

## 问 2：Layer 6 + Layer 7 双卡位
- 主 Layer 6 数据：5 亿 3D 场景库（全球独家）
- 副 Layer 7 世界模型：SpatialLM 开源登 HF 榜
- 协同：数据飞轮喂世界模型，世界模型吸引更多合作方（智元/银河通用/PICO）

## 问 3：护城河 4/5
- 数据飞轮（5 亿 + 3D 资产，短期不可复制）
- 规模效应（6500 万用户 + 4 万企业）
- 切换成本（设计师工具链锁定）

## 问 4：预期差
1. **空间智能数据独家性被低估**：市场把群核当 SaaS 估值（PS 13x 发行），但它本质是具身智能训练数据层（PS 25-30x 合理）
2. **2026 H2 Optimus 量产 → 群核数据订单爆发**：尚未 price-in
3. **开源策略 = 长期生态锁定**（SpatialLM 成为空间智能事实标准）

## 问 5：Bear Case
- 房地产持续下行（SaaS 主营受损）→ 跌到 7 HKD
- 空间智能"元宇宙化"（概念崩塌）→ 跌到 5 HKD
- **最大可接受**：-40%（跌到 10 HKD） → 止损

## 问 6：买卖价
- 不追首日 +144%
- 目标建仓窗口：**2026-10 基石解禁 + 回撤 -30%**
- 建仓区间：10-14 HKD（首日 18.60 的 50-75%）
- 第一目标：25 HKD（+80%）减 1/3
- 长期目标：40-50 HKD（+200%）

## 问 7：证伪
- SpatialLM 被对手开源模型超越 + 社区流失
- Optimus 量产跳票至 2027+
- 中国房地产触底反弹 → 群核"空间智能"标签稀释

## 仓位：保守 3%，中性 5%
```

## 使用纪律

1. **填不满就不买**：7 问中任一答不出来 = 论点不成立
2. **不要自欺**：bear case 写得越狠，越可能真买对；写得虚浮 = 爱上了这只股，危险
3. **预期差 ≤ 2 不下注**：没有 Alpha 就是 Beta，买指数基金更省心
4. **定期回看**：3/6/12 月强制复盘，论点崩塌立即清仓
5. **卡归档**：平仓后的卡不要删，做成个人投研数据库（长期积累胜率）
