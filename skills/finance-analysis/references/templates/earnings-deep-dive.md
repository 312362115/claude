# 财报深度解读 Template（10-Q / 10-K / 业绩会）

> **核心能力 template**：体现 skill 的"**数据深度 + 透过表象**"两大价值。
>
> 套路 C 事件驱动的核心武器。**读原文不看新闻摘要** 是底线，**从小字段推未来** 是本事。
>
> 关联 framework：`command-driven-analysis.md`（套路 C）+ `alpha-generation.md`
>
> 填写时 copy 到 `docs/research/YYYY-MM-DD-<ticker>-<FYQX>-earnings.md`。

---

## 使用前提：必须读原文，不读新闻

**做不到这点就别用这个 template**：

- ✅ 读原件：SEC EDGAR 10-Q 全文（MD&A + 财务附注）、业绩电话会文字稿、财报 PPT 原件
- ✅ 结构化：从 XBRL 抽取字段，对比 YoY / QoQ / 二阶导
- ❌ 看不得：新浪财经 "NVDA Q1 业绩亮眼" 类摘要、财联社快讯

**新闻摘要只解决 5% 信息量，剩下 95% 的 Alpha 在原文 200 页小字里**。

---

## 核心方法论：3 层信息拆解 + 4 维预判

### 3 层拆解（对每个关键字段跑）

```
表层：财报标题新闻（Beat/Miss、涨跌幅）
  ↓ 看新闻即可
中层：关键 KPI（营收/EPS/毛利率/Capex/guidance）
  ↓ 读财报摘要
里层：数据间关系 + 指引落差 + 沉默字段  ← Alpha 在这
```

### 4 维预判

对关键字段做 4 个时间维度的预判，每个**可证伪**：

| 维度 | 问什么 | 证伪条件 |
|------|-------|---------|
| T+3（下季度） | 下次财报 Beat/Miss/In-line？ | 实际数据对比本次预判 |
| T+6（下半年） | 产业链传导完成？ | 间接标的启动与否 |
| T+12（一年） | 业绩兑现程度？ | 增速趋势是否维持 |
| T+24（周期拐点） | 共识期来了还是衰减？ | 媒体覆盖 / 分析师共识 |

## 4. 里层（透过表象的 Alpha）

> **这是 template 的核心部分**。里层信号决定下一季股价走向，也是 Alpha 所在。

### 4.1 数据间的关系（不看单点）

**必查 5 组关系**：

| 关系 | 本季数值 | 信号 |
|------|---------|------|
| **营收增速 vs 应收账款增速** | YoY 营收 +X% / 应收 +X% | 应收增长 > 营收 20%+ = 压货（警告） |
| **营收 vs 库存** | 营收 +X% / 库存 +X% | 库存下降 + 营收上 = 供应紧张（涨价信号）|
| **CapEx vs 折旧摊销** | CapEx $XB / D&A $XB | CapEx 显著 > D&A = 扩产周期开启 |
| **Deferred Revenue YoY** | +X% | 显著上升 = 未来收入保证（订单先行指标）|
| **Gross Margin 二阶导** | 加速/减速 | 连续 2 季度加速上升 = 定价权提升 |

**异常发现**（写至少 2 条）：
- ...
- ...

### 4.2 指引 vs 实际的落差（预期管理）

判断公司是**沙袋策略**（保守指引）还是**激进指引**：

- **沙袋策略**（好）：本季 Beat > 5%，下季指引保守，过往 3 季持续 Beat 指引
- **激进策略**（警报）：本季勉强达标，下季指引激进，历史上超卖方预期但跟不上自己指引

**本公司属于**：沙袋 / 激进 / 中性

**判断依据**：过去 4 个季度 Beat/Miss 规律：
- Q-4：实际 vs 指引 = ±X%
- Q-3：实际 vs 指引 = ±X%
- Q-2：实际 vs 指引 = ±X%
- Q-1：实际 vs 指引 = ±X%

### 4.3 沉默字段（什么没说）

**电话会 Q&A 关注**：
- [ ] 分析师重复追问但管理层回避的问题 → 问题信号
- [ ] 突然不提的产品线 / 地区 / 客户 → 警告
- [ ] 首次出现的新术语 / 指标 → 新方向（可能好可能坏）
- [ ] 管理层语气变化（前一季 confident，本季 cautious） → 信号

**财报文本对比**（10-Q vs 上一份 10-Q）：
- Risk Factors 新增条目：...（新增风险优先关注）
- MD&A 措辞变化：...（"believe" → "expect" 是弱化，反过来是强化）
- 财务附注新披露：...

**沉默发现**（至少 2 条）：

## Worked Example 骨架：NVDA FY25-Q1 财报（2024-05-22 发布）

（假设使用本 template 在发布后 1 小时内读原文）

```markdown
## 表层
- 营收 $26.04B（共识 $24.5B） +6.3% Beat
- EPS $6.12（共识 $5.60） +9.3% Beat
- 股价盘后 +6%

## 中层
- Data Center $22.56B (+427% YoY，+23% QoQ) ← 核心
- Gaming $2.6B (+18% YoY)
- Gross Margin 78.4% (+14pp YoY)
- **Guidance Q2：$28B（共识 $26.8B）+4.5% 超预期**

## 里层（Alpha 所在）
### 4.1 数据关系异常
- **库存 $6B vs 营收 $26B**：库存周转加快 → 供应紧张持续
- **应收 $13B YoY +180%**：但营收 +262%，应收增速 < 营收 = 健康
- **CapEx YoY +100%**：Blackwell 新品扩产周期
- **Deferred Revenue +80%**：Q2/Q3 订单已锁

### 4.2 指引落差
- Q2 Guidance $28B：连续 4 季度大幅 Beat 自己指引（沙袋策略明显）
- 真实 Q2 可能在 $29-30B

### 4.3 沉默字段
- 电话会首次大量提及 "sovereign AI"（主权 AI）→ 新增国家级客户
- Jensen 措辞从 "expecting demand" → "demand exceeds supply"（强化）
- 未明确提 AMD MI300 竞争 → 无压力（或内部评估中）

## 产业链传导
- Layer 1 HBM：美光 / SK 海力士 / 三星（T+0，已有反应）
- Layer 1 CPO：中际旭创 / 新易盛（T+7 传导）
- Layer 1 PCB：沪电股份（T+14 传导）
- **Layer 8 电力**：首次暗示 AI 耗电挑战 → CEG / VST 长期利好（T+90 持续）

## 4 维预判
- T+3（FY25-Q2）：实际营收 $30B+（指引 $28B 必超）
- T+6：A 股 CPO / PCB 二线标的启动
- T+12：全年 Data Center $100B+（共识 $80B）
- T+24：2025 H2 进入共识期，开始减仓信号

## 决策
- **强加仓 NVDA**（里层全红 + 指引沙袋明显）
- **加仓 Layer 1 PCB**（T+14 传导窗口）
- **关注 Layer 8 电力**（T+90 新主线）
```

**事后验证**（2024-05 到 2024-12）：
- T+3：FY25-Q2 营收 $30B（超预期）✅
- T+6：A 股 CPO 启动 ✅
- T+12：Data Center 全年 $115B ✅
- **所有 4 维预判全中** → 本 template 在 NVDA 这种龙头公司上的方法论有效

## 为什么这个 template 是 skill 核心

**"读财报" 是所有投资分析能力的基础**。做不到这点，所有 framework 都是空中楼阁。

- 套路 A 个股分析 → 必须读财报
- 套路 B 政策事件 → 识别受益公司后需读财报
- 套路 C 公司事件 → 财报本身就是事件
- 套路 D 主题赛道 → 每个 Layer 龙头财报是产业链先行指标

**这个 template 是 "数据深度 + 透过表象" 两大价值的集中体现**。用户能把它用好，skill 就真正有 Alpha 生产能力。
