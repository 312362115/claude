# 自上而下选股框架（Top-Down Selection）

> **从宏观变量出发**，推导出"现在值得重仓哪些行业 / 板块"，产出"推荐赛道池"。
>
> **核心动作**：宏观 5 维扫描 → 周期阶段判断 → 行业轮动映射 → 板块评分 → 推荐赛道池（交棒 `stock-selection`）。
>
> **和 stock-selection 的边界**：
> - `stock-selection.md`：自下而上，**入口是"已知赛道"**，输出 Top-N 个股
> - `top-down-selection.md`（本文件）：自上而下，**入口是"宏观环境"**，输出"推荐赛道池"
> - 两者串联：top-down 给赛道池 → stock-selection 从池子筛个股

---

## 🔥 最高原则：股市炒的是预期，不是当前

> **这条原则**比所有 framework 细节都重要。违反这条 → 所有分析都会向"当前数据"对齐 → 注定滞后市场 → 买在顶部、卖在底部。
>
> **认知源头**：`command-driven-analysis.md` §"股市定价是预期驱动"。本 framework 是这条原则在**宏观 / 行业 / 板块**层级的落地。

### 三条推论

1. **市场定价未来 6-24 月，不是现在**
   - 当前 Fed FFR 5.25% 不决定股价，**"市场预期 6 月后降息几次"**才决定
   - 当前 PMI 48 不决定周期板块，**"市场预期明年 PMI 回到 52"**才决定
   - 当前通胀 3% 不决定能源股，**"市场预期 Capex 周期在哪"**才决定

2. **预期变化是驱动力，不是数据本身**
   - 数据如预期 → 股价已 price in → 可能不涨
   - 数据超预期 → 预期上修 → **真正的上涨动力**
   - 数据不及预期 → 预期下修 → **真正的下跌动力**
   - 所以 "Beat the Street Consensus" 比"绝对数值好"更重要

3. **预期有时效性，需要滚动更新**
   - 典型路径：预期酝酿（T-6）→ 预期定价（T-3）→ 数据验证（T0）→ 新预期形成（T+3）
   - 迟了买 = 追高（预期已 price in）
   - 过早买 = 站岗（预期未形成）
   - **甜蜜点**：T-3 到 T0 区间（预期定价初期到数据验证）

### 本 framework 落地这条原则的方式

- **§4.1.2 加息降息预期**：CME FedWatch / Fed Dot Plot / 利率期货隐含路径（预期的核心观测位）
- **§4.5.1 五年规划**：T-6 发酵期是买入甜蜜点，T+6 业绩兑现后反而钝化
- **§8 Worked Example**：2024 年 AI 板块在高利率下暴涨的核心逻辑就是"Capex 超预期"
- **§6.3 主题 vs 宏观博弈**：主题爆发期本质是"新预期形成速度 > 宏观预期"

### 常见反模式

❌ **用当前数据决定配置**："Fed FFR 5.25%，成长股必压" → 忽略市场已 price in 降息
❌ **等数据公布再买**：NFP 公布 + CPI 公布后才判断 → 价格已反映，追涨
❌ **把"分析师观点"当预期**：卖方研报的共识 ≠ 市场真实预期（期货 / 期权隐含更准）
❌ **只看绝对数值好坏**：营收增速 40% 好坏是看"vs 预期"，不是"vs 30%"
❌ **不跟踪预期变化**：一年定一次配置，错过预期重定价

### 最终法则

> **"你不是在预测未来，你是在评估市场对未来的定价是否合理"** — 买预期低估的资产，卖预期透支的资产。

## 1. 核心定位

### 适用场景

- 不知道该看哪个赛道时，从宏观环境出发判断行业
- 新一轮周期来临前做板块配置（美联储转向 / 央行宽松 / 地缘变化）
- 组合整体权重调整（超配 / 低配哪些行业）
- 宏观事件驱动命题（关税 / 利率 / 通胀 / 政策转向）
- 跨市场大类资产配置（美 / 港 / A 股联动）

### 不适用场景

- **已锁定赛道 / 主题** → 直接走 `stock-selection` 或 `ai-thematic-investing`
- **短期交易 / 日内** → 宏观变量变化慢，不适合日内
- **个股选择 / 估值** → 走 `valuation-relative` / `valuation-dcf`
- **事件驱动单股**（并购 / 财报）→ 走 `event-driven`（0.4.0）
- **纯主题投资**（已知 AI / 机器人主题）→ `ai-thematic-investing` 更精确

### 产出边界

本 framework 产出 **"推荐赛道池 + 配置权重建议"**，不产出：
- 具体买入清单 → `stock-selection`
- 个股目标价 → `valuation-*`
- 择时信号 → 非决策导向，本 skill 不做

## 2. 核心方法论：三段式

```
Step A 宏观 5 维扫描（货币 / 通胀 / 经济 / 地缘 / 政策）
Step B 周期阶段判断（5 档：复苏 / 扩张 / 繁荣 / 衰退 / 滞胀）
Step C 行业轮动映射 → 板块评分 → 推荐赛道池
```

### 关键原则

1. **股市炒预期，不炒当前** — 扫描的不是"现在利率多少"，而是"市场定价的未来 6-24 月路径"（见上方"最高原则"）
2. **不预测宏观**，只判断**当前处于哪个阶段 + 市场预期已到哪一步** — 宏观预测是"赌"，阶段识别 + 预期定位是"观察"
3. **多维交叉验证** — 5 维不一致 → 处于过渡期，降低置信度
4. **不同市场用不同宏观锚** — Fed / 央行 / HKMA 各自独立扫
5. **和主题驱动分开但允许叠加** — 宏观周期 ≠ 主题爆发（AI 主题在高利率下仍然涨就是例证）
6. **输出配置权重，不给精确时点** — "超配 / 平配 / 低配"比"何时买"更可执行

## 3. 决策树

```
命题输入：宏观环境下哪些行业 / 板块值得投？
  │
  ├─ Step 1: 市场定位（A / 港 / 美 / 全球）
  │   ├─ 美股 → 以 Fed / US 经济 / 美元为主锚
  │   ├─ 港股 → HKMA 联系汇率 + 内地政策双锚
  │   ├─ A 股 → 央行 + 政策监管为主锚
  │   └─ 全球 → 美元周期 + 跨市场联动
  │
  ├─ Step 2: 宏观 5 维扫描
  │   ├─ 货币政策（利率 / 流动性 / 资产负债表）
  │   ├─ 通胀 + 大宗（CPI / PPI / 原油 / 铜 / 黄金）
  │   ├─ 经济周期（GDP / PMI / 就业 / 消费）
  │   ├─ 地缘政治（贸易战 / 关税 / 制裁）
  │   └─ 政策监管（刺激政策 / 行业规管 / 股市政策）
  │
  ├─ Step 3: 周期阶段判断（5 档）
  │   ├─ 复苏：利率低 + PMI 回升 + 经济触底
  │   ├─ 扩张：利率稳 + 经济增长 + 通胀温和
  │   ├─ 繁荣：利率升 + 经济增长 + 通胀升
  │   ├─ 衰退：利率降 + 经济差 + 通胀落
  │   ├─ 滞胀：利率高 + 经济差 + 通胀高
  │   └─ 过渡期：5 维不一致 → 标注"低置信度"
  │
  ├─ Step 4: 行业轮动映射（阶段 → 受益 / 受损行业）
  │
  ├─ Step 5: 板块评分（每个行业下的细分板块）
  │   ├─ 估值锚（当前 vs 历史分位）
  │   ├─ 盈利动能（盈利预期修正）
  │   ├─ 政策催化（是否有明确催化剂）
  │   └─ 情绪锚（资金流向 / 仓位）
  │
  ├─ Step 6: 输出推荐赛道池
  │   ├─ 核心超配（30-50% 权重）
  │   ├─ 战术平配（20-30%）
  │   ├─ 防御底仓（10-20%）
  │   └─ 避雷（低配 / 不碰）
  │
  └─ Step 7: 交棒 stock-selection
      └─ 对每个推荐赛道调用 `stock-selection` 筛 Top-N
```

## 4. 宏观 5 维扫描明细

### 4.1 货币政策（最重要）

#### 4.1.1 当前水平

| 变量 | 观测来源 | 判断阈值 |
|------|---------|---------|
| 美联储 FFR | FOMC 决议 / FedWatch | < 2% 宽松 / 2-4% 中性 / > 4% 紧缩 |
| 中国 LPR | 人行公告 | 下调周期 = 宽松 / 稳定 = 中性 / 上调 = 紧缩 |
| HKMA 基准利率 | 香港金管局 | 跟 Fed 同步（联系汇率） |
| 美国 M2 YoY | FRED M2 系列 | > 10% 超宽松 / 5-10% 宽松 / < 0% 紧缩 |
| 中国 M2 YoY | 人行月报 | > 12% 宽松 / 8-12% 中性 / < 8% 紧缩 |
| Fed 资产负债表 | FRED WALCL | 扩张 / 平稳 / QT 缩表 |

#### 4.1.2 加息降息预期（前瞻变量，权重 ≥ 当前水平）

**股市定价未来，不是现在** — 市场交易 3-12 月后的利率路径，不是当前利率。

| 变量 | 观测来源 | 判断 |
|------|---------|------|
| **CME FedWatch 概率** | CME Group | 下次 FOMC 加息 / 降息 / 不变概率 |
| **Fed Dot Plot**（点阵图） | FOMC SEP | FOMC 成员对 2-3 年利率路径中位数判断 |
| **2Y / 10Y 国债收益率** | FRED / Treasury | 2Y 反映 1-2 年政策路径；10Y 反映长期经济预期 |
| **利率期货隐含路径** | CME Fed Funds Futures | 市场定价的未来 12 月加息 / 降息次数 |
| **核心通胀 + 就业数据趋势** | CPI Core / NFP / 失业率 | 数据走势决定 Fed 鸽派 / 鹰派转向 |
| **Fed 主席发言 / 会议纪要** | FOMC Minutes / 演讲 | 鸽派 / 鹰派信号提前 3-6 月 |
| **中国 PBOC 发声 / 央行工作会议** | 人行新闻发布会 / 年度工作会议 | 降准 / 降息预期 |

**核心逻辑（预期驱动）**：

```
当前利率 = 锚
市场预期的未来利率路径 = 定价 → 即期反应

典型场景：
  Fed 暂停加息但市场预期即将降息 → 成长股先涨（price in 降息）
  实际降息但预期已打满 → buy the rumor, sell the news → 板块回调
  数据超预期 → 降息预期推迟 → 成长股回调（"higher for longer"）
```

**观察节奏**：
- 每次非农就业（NFP，月首周五）+ CPI（月中） → 可能重定价
- 每次 FOMC 会议（8 周一次） → 点阵图 + 经济预测 SEP 重置
- Fed 主席 Jackson Hole（8 月）+ FOMC 记者会 → 关键政策信号

**判断对板块的影响**：

| 预期转向 | 最受益 | 最受损 |
|---------|-------|-------|
| 加息预期 → 降息预期 | 成长 / 科技 / 高估值 / 长债 / 黄金 | 金融 / 银行 / 周期 / 能源 |
| 降息预期 → 加息预期（"higher for longer"）| 金融 / 银行 / 能源 / 短债 | 成长 / 高估值 / 房地产 |
| 降息幅度预期加大（从 2 次 → 4 次） | 小盘股 / 生物科技 / 房地产 | 大盘价值股相对落后 |

**执行纪律**：
- 不预测 Fed 下一步做什么，**只判断市场预期在哪个档位**
- 每月 refresh 一次预期（NFP / CPI 发布后）
- 预期极端偏离（如市场定价 3 次降息而 Fed 只会降 1 次）→ 警惕反向风险

#### 4.1.3 核心逻辑

- 利率 ↑ → 折现率 ↑ → 成长股 / 高估值股压制；金融 / 价值股受益
- 利率 ↓ → 折现率 ↓ → 成长股 / 高估值股受益；金融压制
- M2 扩张 → 流动性溢出 → 股市整体受益 + 风险偏好抬升
- **预期先行**：实际降息前 3-6 月，成长股已开始 price in

### 4.2 通胀 + 大宗商品

| 变量 | 观测来源 | 阈值 |
|------|---------|------|
| 美国 CPI YoY | BLS 月报 | < 2% 低通胀 / 2-4% 温和 / > 4% 高通胀 |
| 美国 PPI YoY | BLS 月报 | 领先 CPI 3-6 月 |
| 原油（布伦特 / WTI） | EIA | < $60 偏弱 / $60-90 正常 / > $90 高位 |
| 铜价 LME | LME | 经济活跃度晴雨表 |
| 黄金 | COMEX | 避险 + 美元反向 + 实际利率反向 |

**核心逻辑**：
- 通胀 ↑ → 能源 / 材料 / 金融受益；必需消费 / 公用事业 压制
- 通胀 ↓ → 成长 / 科技 受益；周期股 / 能源 压制
- 油价高 + 铜价弱 = 滞胀信号
- 金价创新高 = 市场避险情绪 + 预期实际利率下行

### 4.3 经济周期

| 变量 | 观测来源 | 阈值 |
|------|---------|------|
| 美国 GDP YoY | BEA | > 3% 扩张 / 1-3% 温和 / < 1% 衰退边缘 |
| ISM 制造业 PMI | ISM | > 50 扩张 / < 50 收缩 |
| 中国官方 PMI | 国家统计局 | 同上 |
| 非农就业（NFP） | BLS | > 200K 强劲 / 100-200K 中等 / < 100K 弱 |
| 失业率 | BLS | 历史低位 = 繁荣尾声信号 |
| 零售销售 YoY | Census | 消费景气度 |

**核心逻辑**：
- PMI 跨越 50 → 经济拐点（向上/向下）
- 失业率触底 → 通胀压力累积 → 加息预期
- 零售 ↓ + PMI ↓ → 衰退风险 → 防御板块

### 4.4 地缘政治

| 事件类型 | 影响板块 |
|---------|---------|
| 关税（中美 / 美欧） | 制造业 / 出口链 / 供应链替代 |
| 制裁（半导体 / 能源） | 被制裁方受损 / 国产替代受益 |
| 战争 / 冲突 | 能源 ↑ / 粮食 ↑ / 军工 ↑ / 航运波动 |
| 供应链重构 | 东南亚 / 墨西哥制造业 / 美国回流 |

**观测来源**：官方公告 + 主流媒体 + 行业协会。

### 4.5 政策监管（A 股尤其关键）

A 股政策驱动力常超过宏观周期。分为 **中长期结构性政策**（5-15 年锁定方向）+ **中短期节奏性政策**（季度级调整）。

#### 4.5.1 中长期结构性政策（最重要，锁定 5-15 年战略赛道）

| 政策文件 | 周期 | 观察要点 |
|---------|------|---------|
| **五年规划**（如十四五 2021-2025 / **十五五 2026-2030**） | 5 年 | 战略新兴产业 / 制造强国 / 科技自立赛道 |
| **中央经济工作会议**（每年 12 月） | 年度 | 下一年宏观基调 + 重点工作 |
| **政府工作报告**（每年 3 月两会） | 年度 | 具体政策目标 + 财政 / 货币基调 + GDP 目标 |
| **三中全会决议** | 5-10 年一次 | 改革方向（如 2013 十八届三中、2024 二十届三中） |
| **中共中央 + 国务院联合发文** | 不定期 | 国家级战略（如 "东数西算"、"新质生产力"、"制造强国 2025"） |
| **行业专项规划** | 5 年 | 半导体 / 新能源 / 生物医药 / 人工智能 等单行业规划 |

**如何在本 framework 里使用**：

1. **赛道锁定清单**：五年规划 + 三中全会明确提的战略赛道 → 本 framework §6 行业轮动中**多年周期性加分**，不因某个宏观阶段降档
2. **权重上限**：即使短期宏观逆风（如利率升），战略赛道也给"防御底仓"保护（不低于 10%）
3. **政策落地跟踪**：从"规划文件"到"专项补贴 / 订单 / 税收优惠"是真正的业绩兑现节奏（常 1-3 年）

**十四五（2021-2025）定位的战略赛道**：
- 数字经济 / 人工智能
- 新能源（光伏 / 风电 / 储能 / 新能源车）
- 生物技术 / 创新药
- 半导体 / 集成电路
- 高端装备制造
- 量子信息 / 前沿科技

**十五五（2026-2030，待发布，预期方向）**：
- 新质生产力（AI / 具身智能 / 商业航天 / 低空经济 / 生物制造）
- 制造业升级（6G / 先进材料 / 高端数控）
- 统一大市场 + 内循环
- 绿色低碳 / 核电 / 氢能
- 粮食 / 能源 / 关键矿产安全

**注意**：十五五最终版（2025-10 五中全会 / 2026-03 两会）发布前，用"征求意见稿 + 高层讲话"前瞻把握方向。

#### 4.5.2 中短期节奏性政策

| 类型 | 典型案例 |
|------|---------|
| **消费刺激** | 以旧换新 / 汽车补贴 / 家电下乡 → 汽车 / 家电 / 大消费 |
| **股市政策** | T+0 / 融券 / 印花税 / 回购贷 → 券商 / 大盘股 / 蓝筹 |
| **行业规管（负面）** | 双减 / 反垄断 / 药品集采 / 平台经济治理 → 教培 / 互联网 / 医药 受损 |
| **新兴扶持（正面）** | 半导体大基金 / 新能源补贴 / AI 试点城市 → 对应赛道 |
| **资本市场改革** | 注册制 / IPO 节奏 / 再融资 / 北交所 / 港股互联互通 → 券商 / 创投 |
| **地产政策** | 限购松绑 / 房贷利率 / 保交楼 → 地产 / 家居 / 银行 |
| **外资管控** | QFII / 沪港通 / 北向资金 → 外资持仓板块 |

#### 4.5.3 政策监管评分加权

在本 framework §7.3 板块评分中，**政策催化**维度（25 分）按以下加权：

| 政策级别 | 加分 |
|---------|------|
| 写入五年规划 / 三中全会 / 中央政治局会议 | +5 长期加分 |
| 写入政府工作报告 / 中央经济工作会议 | +3 年度加分 |
| 具体专项扶持（大基金 / 补贴 / 税收） | +3 短期加分 |
| 明显政策逆风（集采 / 反垄断） | -5 压制 |
| 政策不明朗 | 0 |

#### 4.5.4 A 股独有的"政策前瞻"节奏

```
T-6 月 —— 两会前夜政策预期发酵
T-3 月 —— 中央经济工作会议 / 重要会议释放信号
T0    —— 政策文件发布
T+1 月 —— 解读 + 板块反应
T+3 月 —— 具体补贴 / 订单落地（第一波业绩兑现）
T+12 月 —— 主力受益公司业绩爆发（第二波）
T+24 月 —— 政策红利消化，进入正常周期（第三波）
```

**买入时点**：通常 T-3 到 T+1（预期 + 确认），T+6 后业绩还没出容易踩空

## 5. 周期阶段判断

### 5 档阶段定义

| 阶段 | 利率 | 经济 | 通胀 | 典型持续时间 |
|------|------|------|------|-------------|
| **复苏** | 低 / 降息末 | 触底回升 | 低 | 6-18 月 |
| **扩张** | 稳定 / 低 | 增长中 | 温和 | 12-36 月 |
| **繁荣** | 升 / 高 | 高增长 | 升 | 6-18 月 |
| **衰退** | 降息中 | 收缩 | 落 | 6-18 月 |
| **滞胀** | 高 | 差 | 高 | 少见，12-24 月 |

### 判断方法

**主锚 + 辅助验证**：

| 主锚 | 辅助 | 阶段 |
|------|------|------|
| 利率触底 + PMI 回升至 50 以上 | 失业率仍高 + 通胀低 | **复苏** |
| 利率稳定低位 + PMI > 52 | 失业率下行 + 通胀 2-3% | **扩张** |
| 利率升 + PMI > 55 | 失业率历史低 + 通胀 > 3% | **繁荣** |
| 降息启动 + PMI < 48 | 失业率上行 + 通胀落 | **衰退** |
| 利率高 + PMI < 48 | 失业率上行 + 通胀 > 4% | **滞胀** |
| 5 维不一致 | — | **过渡期（降置信度）** |

## 6. 行业轮动映射表

### 6.1 5 阶段 × 行业受益 / 受损

| 阶段 | 超配（重仓） | 标配 | 低配 / 避雷 |
|------|-------------|------|------------|
| **复苏** | 消费（汽车 / 家电）/ 周期股（工业 / 化工）/ 金融（券商） | 科技 / 医药 | 防御类（公用事业）/ 黄金 |
| **扩张** | 成长股 / 科技 / 高端制造 / 医药创新 | 消费 / 金融 | 能源 / 公用事业 / 黄金 |
| **繁荣** | 能源 / 工业金属 / 银行 / 地产 | 化工 / 建材 | 高估值成长 / 必需消费 |
| **衰退** | 必需消费 / 医疗 / 公用事业 | 黄金 / 长债 | 周期股 / 可选消费 / 高杠杆 |
| **滞胀** | 黄金 / 能源 / 必需消费 / 抗通胀 | 农业 / 军工 | 成长股 / 消费 / 金融 |

### 6.2 不同市场的差异

**美股**：
- Fed 驱动强，所以利率对科技板块影响最显著
- 能源板块和地缘强相关
- 大盘股 vs 中小盘受利率影响幅度不同

**港股**：
- 联系汇率 → 跟 Fed + 受内地政策影响
- 南向资金情绪波动大，容易短期偏离宏观
- 中概股估值受中美关系影响

**A 股**：
- 政策驱动 > 宏观驱动（2021 双减 / 2023 AI 政策）
- 外资 vs 内资情绪容易分化
- 北向资金流向是情绪信号
- 消费刺激 / 制造业转型政策是独有变量

### 6.3 主题 vs 宏观的博弈

**特别注意**：主题爆发期可以**超越宏观阶段**。经典例子：

- 2023-2024 美联储 FFR 5.25%（高利率 = 压成长股）但 NVDA / AI 主题涨 10x → 主题溢价抵消宏观
- 2020 疫情衰退 + 零利率，科技股反而暴涨 → 低利率 + 远程办公主题共振
- 2021 A 股新能源车估值打到 150x PE → 主题 + 政策支持 + 流动性

**所以**：宏观给"基础配置建议"，但**主题爆发期要让路给 `ai-thematic-investing`**。两者配合才能做到"宏观给锚 + 主题给超额"。

## 7. 板块评分模型

### 7.1 四维打分（每维 25 分，总分 100）

**估值锚（25）**：

| 子项 | 分值 | 评分标准 |
|------|-----|---------|
| 历史分位 | 10 | < 20% 分位 = 10 / 20-50% = 7 / 50-80% = 4 / > 80% = 0 |
| 对标国际 | 8 | 显著低估对标 = 8 / 持平 = 5 / 高估 = 0 |
| 预期 PE 降速 | 7 | 盈利增速 > PE = 7 / 持平 = 4 / 低于 = 0 |

**盈利动能（25）**：

| 子项 | 分值 | 评分标准 |
|------|-----|---------|
| 一致预期上修 | 10 | 3M 一致预期上修 > 5% = 10 / -5~5% = 5 / 下修 = 0 |
| 最新季度 beat rate | 8 | > 70% 公司 Beat = 8 / 50-70% = 5 / < 50% = 0 |
| 远期能见度 | 7 | 有催化剂 + 可量化 = 7 / 模糊 = 3 / 无 = 0 |

**政策催化（25）**：

| 子项 | 分值 | 评分标准 |
|------|-----|---------|
| 直接政策支持 | 10 | 明确补贴 / 采购 / 规划 = 10 / 间接 = 5 / 无 = 0 |
| 宏观阶段契合 | 8 | 当前阶段主力受益 = 8 / 标配 = 5 / 受损 = 0 |
| 逆风因素 | 7 | 无 = 7 / 有 1 项 = 4 / 有多项 = 0 |

**情绪锚（25）**：

| 子项 | 分值 | 评分标准 |
|------|-----|---------|
| 资金流向 | 10 | 主动资金 3M 净流入 = 10 / 平衡 = 5 / 净流出 = 0 |
| 拥挤度 | 8 | 机构仓位 < 历史中位 = 8 / 中位 = 5 / 极拥挤 = 0 |
| 媒体热度 | 7 | 温和关注 = 7 / 冷门 = 4 / 过热 = 2 |

### 7.2 总分分档

| 总分 | 档位 | 配置建议 |
|------|------|---------|
| 85-100 | **核心超配** | 30-50% 组合权重 |
| 70-85 | **战术平配** | 20-30% |
| 55-70 | **防御底仓** | 10-20% |
| 40-55 | **观察** | 0-5% |
| < 40 | **避雷** | 不配 |

### 7.3 红线

- 估值分 < 5（历史 95% 分位以上）→ 估值已打满，降一档
- 情绪分 < 5（极度拥挤）→ 情绪已顶，降一档
- 政策分 < 5（明显逆风）→ 直接避雷

## 8. Worked Example：2024-2025 高利率周期美股板块配置

### 8.1 宏观环境（2024-12）

- **Fed FFR**：5.25-5.50%（自 2023-07 维持 15 个月）→ 紧缩
- **M2 YoY**：~2%（2020-2022 爆发式增长后回落）→ 中性
- **CPI YoY**：3.1%（从 2022 峰值 9.1% 回落）→ 温和通胀
- **ISM PMI**：48-50 波动 → 收缩边缘
- **失业率**：4.2%（从 3.4% 低位上升）→ 周期末期
- **地缘**：中美关税博弈 + 特朗普当选预期关税 2.0
- **美元指数**：104-107 → 强势

### 8.2 周期阶段判断

- 利率高（紧缩）+ PMI < 50（收缩）+ 通胀温和 → **繁荣尾部 → 衰退前夜**
- 失业率上行 + 消费韧性仍在 → **过渡期**
- **置信度：中**（利率拐点未明，但经济已显疲态）

### 8.3 行业配置（基础版 + 主题叠加）

**基础版（纯宏观）**：

| 板块 | 估值分 | 盈利分 | 政策分 | 情绪分 | 总分 | 档位 |
|------|-------|-------|-------|-------|------|------|
| 必需消费（Staples） | 18 | 15 | 15 | 18 | **66** | 防御底仓 |
| 医疗健康 | 20 | 17 | 18 | 16 | **71** | 战术平配 |
| 公用事业 | 15 | 14 | 12 | 15 | **56** | 防御底仓 |
| 能源 | 17 | 15 | 15 | 14 | **61** | 防御底仓 |
| 金融（银行） | 22 | 18 | 15 | 12 | **67** | 防御底仓（利率拐点前受益，之后压制） |
| 科技 / 成长（纯估值看） | 8 | 20 | 12 | 10 | **50** | 观察（估值高但盈利强） |
| 可选消费 | 12 | 12 | 8 | 10 | **42** | 观察 |
| 工业 / 周期 | 14 | 10 | 10 | 12 | **46** | 观察 |

**基础版结论**：纯宏观看，**防御 + 医疗**为主，科技因估值被降档。

### 8.4 主题叠加修正（AI 主题 vs 宏观）

**关键发现**：**2024 年 AI 主题完全打破宏观周期规律**
- NVDA / MSFT / META 等 AI 相关大盘股在高利率下仍然大涨
- Capex 驱动是"真订单"，不是"估值扩张"
- 因此需要 **主题加分 + 宏观减分 的叠加**

**修正后配置**：

| 板块 | 宏观档位 | 主题叠加 | 最终配置 |
|------|---------|---------|---------|
| **AI 算力（NVDA / AVGO / AMD）** | 观察 | 核心超配（主题驱动） | 30-40% |
| **AI 应用（MSFT / META / GOOG）** | 观察 | 战术平配（主题） | 15-20% |
| 医疗健康 | 战术平配 | — | 10-15% |
| 金融（银行） | 防御底仓（利率拐点前） | — | 5-10% |
| 必需消费 | 防御底仓 | — | 5-10% |
| 能源 | 防御底仓 | — | 5% |
| 防御合计 | — | — | 25-35% |
| **避雷**：高估值小盘成长（无主题）/ 房地产（利率敏感）/ 生物科技小票 | — | — | 0% |

### 8.5 交棒 stock-selection

对"AI 算力"赛道触发 `stock-selection.md`：
- 池子：标普 500 + 纳斯达克 100
- 4 步筛 + 30/60/100 打分
- 预期 Top-3：**NVDA / AVGO / TSM**

对"医疗健康"：
- 池子：标普 500 医疗子行业
- 4 步筛
- 预期 Top-3：**LLY / UNH / ABBV** 之类

### 8.6 实战复盘（2025-Q2 视角）

**事后验证**：
- 2025 年 Fed 虽未降息但软着陆预期强化 → 成长板块延续
- NVDA 延续强势 → 主题超配判断正确
- 金融板块在 2025-Q1 回调 → 利率拐点未到，不算误判但时点模糊
- 医疗（LLY / UNH）走势分化，减肥药主题继续支撑 LLY

**教训**：
- 宏观给"基础配置锚"有效，但 **主题驱动力强时必须让路给主题 framework**
- 纯宏观思维看 2024 AI 完全错过（纯估值看科技板块评分 50 分以下）
- 所以本 framework 必须和 `ai-thematic-investing` 配合使用

## 9. 执行纪律

### 红线（违反 = 重做）

1. **宏观 5 维只扫 1-2 个** → 必须全部过一遍
2. **主观判断阶段，无数据支撑** → 每个阶段判断必须附具体变量数值
3. **忽视主题因素** → 主题爆发期不修正 = 典型宏观派失败
4. **A 股忽略政策监管** → 政策是 A 股第一驱动力
5. **配置权重 > 50% 给单一板块** → 违反分散原则（除非极度确定）
6. **不做红线检查** → 高估值 + 极拥挤不降档 = 追顶
7. **不做跨市场适配** → 用美股宏观逻辑套 A 股 = 错配
8. **一次定 N 个月配置不回看** → 宏观变，配置也应季度 refresh

### 反模式

❌ **宏观预测陷阱**：试图预测"利率什么时候降"（反复啪啪打脸）
❌ **主题 / 宏观对立**：非此即彼，实际两者叠加最稳
❌ **历史 PE 分位滥用**：板块生命周期变了，历史分位失效（如 A 股白酒 2018 vs 2023）
❌ **看 CNBC / 新闻即决策**：媒体叙事滞后，真正逻辑是数据
❌ **过度调仓**：每月调一次 → 交易成本吞掉收益 + 情绪化
❌ **衰退信号出现全部清仓**：过度防御错过复苏
❌ **只看美股宏观不看中国 / 港股独有因素**：A 股 / 港股 ≠ 美股简单映射

### 核查清单

- [ ] 市场定位明确（A / 港 / 美 / 全球）
- [ ] 宏观 5 维全扫 + 每维有具体数值
- [ ] 周期阶段判断有主锚 + 辅助验证
- [ ] 过渡期是否标注"低置信度"
- [ ] 行业轮动映射匹配阶段
- [ ] 板块评分四维都打了
- [ ] 主题因素是否考虑（和 `ai-thematic-investing` 交叉）
- [ ] 配置权重合计 100%
- [ ] 最高单板块权重 ≤ 50%
- [ ] 红线（估值 / 情绪 / 政策）都检查了
- [ ] 给出季度 refresh 机制

## 10. 失效场景

### 10.1 主题爆发期

**场景**：新主题爆发（如 2023 AI），主题涨幅完全压过宏观周期逻辑。

**退路**：
1. 本 framework 给出"基础配置"
2. 但必须调用 `ai-thematic-investing` 叠加主题溢价
3. 最终配置 = 宏观基础 + 主题超配
4. 纯宏观思维会错过主题超额

### 10.2 宏观变量剧变

**场景**：黑天鹅（战争 / 金融危机 / 疫情）导致宏观变量突变。

**退路**：
1. 所有周期阶段判断失效
2. 切入"危机管理模式"：提高现金 + 防御板块 + 黄金
3. 等待新平衡点形成后再用本 framework

### 10.3 阶段混合期

**场景**：5 维数据严重分歧（如 2023 年通胀高 + 经济强 + 利率高 + AI 主题爆发）。

**退路**：
1. 标注"过渡期 + 低置信度"
2. 降低板块配置集中度
3. 更依赖主题 framework + 个股 framework

### 10.4 A 股政策驱动主导

**场景**：A 股某时段政策驱动力远大于宏观（如 2023 AI 政策支持、2021 双减冲击）。

**退路**：
1. 政策权重提到第一位，宏观退居其次
2. 使用"政策映射表"而非"宏观阶段映射"
3. 本 framework §4.5 政策监管维度加权重

### 10.5 流动性极端

**场景**：金融危机式流动性枯竭（2008 / 2020-03）或流动性泛滥（2020-2021）。

**退路**：
1. 配置逻辑被流动性覆盖，行业轮动失效
2. 流动性紧缩 → 所有风险资产一起跌
3. 流动性泛滥 → 所有资产一起涨，不分行业
4. 标注"beta > alpha 主导期"，降本 framework 权重
