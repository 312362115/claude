# 基金分析框架（Fund Analysis）

> **不研究个股**，研究"**该不该买这只基金 / 哪只基金更值得买**"。
>
> **核心动作**：基金分类判断 → 核心指标扫描（夏普 / 最大回撤 / α/β / 归因）→ 持仓分析（风格 / 漂移 / 集中度）→ 成本 + 流动性 → 决策（买 / 持 / 换 / 退）。
>
> **和 stock-selection 的区别**：
> - `stock-selection`：选个股
> - `fund-analysis`（本文件）：选基金 / 配置组合

---

## 🔥 最高原则：看基金是"买管理人"，不是"买近 1 年收益"

> 大多数人的最大错误：**看短期排行榜买基金** → 买在历史最好的时点 → 均值回归后后悔。
>
> 正确思路：**管理人 + 策略稳定性 + 风险调整后的收益** 比"净值涨跌"重要 10 倍。
>
> 认知来源：A 股公募"冠军基金魔咒"（年度冠军次年往往跑输基准）是 20 年定律。

### 三条推论

1. **看**α**不看绝对收益**：涨 30% 时大盘涨 40% 是反指；跌 10% 时大盘跌 30% 是 alpha
2. **看夏普不看年化**：年化 20% 但波动 40% 不如年化 15% 波动 15%（持有体验决定能否拿住）
3. **看管理人任职期 3+ 年不看短期排名**：6 月冠军大概率是押注成功，3 年稳定超额才是能力

## 1. 核心定位

### 适用场景

- 选基金 / 选 ETF / 组合配置
- 判断"现有基金是否继续持有"
- 对比多只同类基金
- 判断"现在是否是申购 / 赎回时点"
- 被动与主动的选择判断

### 不适用场景

- **个股分析** → `stock-selection` / `valuation-*`
- **基金公司内部研究** → 超出散户视角
- **量化策略深度分析** → 需要历史数据 + 回测，Tier 2 能力
- **定增 / 专户 / 非公开产品** → 门槛高 + 数据少

### 产出边界

本 framework 产出 **"基金决策卡"**：是否买 / 买多少 / 多久持有 / 什么信号退出 / 是否换人。

## 2. 基金分类速览

| 大类 | 子类 | 典型代表 | 评估重点 |
|------|------|---------|---------|
| **股票型** | 主动 | 兴全合润 / 富国天惠 / 易方达蓝筹 | 管理人 + α + 风格稳定性 |
|  | 被动 / ETF | 沪深 300 ETF / QQQ / SPY / SMH | 跟踪误差 + 费率 + 流动性 |
|  | 行业主题 ETF | 半导体 ETF / 新能源车 ETF | 赛道景气 + 跟踪误差 |
|  | Smart Beta | 红利 / 低波 / 成长 / 价值 | 策略有效性 + 回测 |
| **混合型** | 偏股 | 张坤 / 朱少醒管理 | 类股基 + 择时能力 |
|  | 偏债 | 天弘安康 / 富国天锋 | 组合纪律 + 回撤 |
| **债券型** | 纯债 | 中债综合 ETF | 久期 / 信用等级（→ `bond-analysis` 0.3.0） |
|  | 可转债 | 兴全可转债 | 转股溢价率 + 正股分析 |
| **另类** | 对冲 | 桥水 / 元盛 / 幻方 | 策略类型 + 相关性 |
|  | 商品 | 黄金 ETF / 原油 ETF | 现货基差 + 展期成本 |
| **FOF** | 组合基金 | 平安盈越 | 子基金精选 + 大类配置 |
| **QDII** | 境外 | 易方达纳指 100 / 摩根亚洲股票 | 汇率风险 + 底层资产 |

## 3. 决策树

```
命题输入：XX 基金值不值得买？ / 从 N 只同类基金里选哪只？
  │
  ├─ Step 1: 分类识别（股 / 混 / 债 / 另类 / FOF / QDII）
  │   └─ 不同类别用不同评估重点
  │
  ├─ Step 2: 基金经理 / 策略扫描（主动基金必做）
  │   ├─ 任职时长 ≥ 3 年（可观察穿越周期）
  │   ├─ 管理规模（< 10 亿 易漂移；> 300 亿 易钝化）
  │   ├─ 历史代表作 + 策略一致性
  │   └─ 重大变动信号（离职 / 跳槽传闻）
  │
  ├─ Step 3: 核心指标扫描（风险调整后收益）
  │   ├─ 3Y / 5Y 年化收益 vs 基准
  │   ├─ 3Y / 5Y 最大回撤
  │   ├─ 夏普比率（Sharpe）
  │   ├─ Sortino（下行风险调整）
  │   ├─ 卡玛比率 Calmar（年化 / 最大回撤）
  │   ├─ α（超额收益）/ β（市场敏感度）
  │   └─ 跟踪误差（被动）
  │
  ├─ Step 4: 持仓分析
  │   ├─ 前 10 大重仓股（集中度 + 行业分布）
  │   ├─ 风格（大盘 / 中小盘 / 成长 / 价值）
  │   ├─ 风格漂移检测（多期对比）
  │   └─ 换手率（< 50% 稳健 / > 200% 择时型）
  │
  ├─ Step 5: 归因分析
  │   ├─ 选股 α vs 择时 α
  │   ├─ 行业配置 α vs 个股精选 α
  │   └─ β 暴露收益（不算 α）
  │
  ├─ Step 6: 成本 + 流动性
  │   ├─ 管理费 + 托管费（年化总费）
  │   ├─ 申购赎回费 + 持有期折扣
  │   ├─ 净值规模（> 2 亿 合理 / < 5000 万 有清盘风险）
  │   └─ 日均成交额（ETF，是否能正常交易）
  │
  ├─ Step 7: 入场时点判断（结合宏观 + 主题）
  │   ├─ 基金底层资产当前估值（→ top-down-selection）
  │   ├─ 基金经理风格是否匹配当前阶段
  │   └─ 申购是否过热（可能是顶部信号）
  │
  └─ Step 8: 决策输出
      ├─ 买入 / 持有 / 观望 / 赎回 / 换人
      ├─ 仓位建议（占组合权重）
      ├─ 止损触发（最大回撤超阈值 / 管理人变更）
      └─ 持有周期（短期套利 / 中期 1-3 年 / 长期 5+ 年）
```

## 4. 核心指标量化阈值

### 4.1 风险调整后收益

| 指标 | 公式 / 含义 | 优秀阈值 | 合格阈值 |
|------|-----------|---------|---------|
| **夏普比率** | (年化收益 - 无风险利率) / 年化波动 | > 1.5 | > 1.0 |
| **Sortino** | (年化收益 - 无风险利率) / 下行波动 | > 2.0 | > 1.5 |
| **卡玛 Calmar** | 年化收益 / 最大回撤 | > 1.0 | > 0.5 |
| **Information Ratio** | (基金收益 - 基准收益) / 跟踪误差 | > 0.8 | > 0.5 |

**红线**：
- 夏普 < 0.5 → 风险收益比失衡，**避雷**
- 卡玛 < 0.3 → 回撤过大，**不适合重仓**

### 4.2 回撤控制

| 指标 | 阈值 |
|------|------|
| **最大回撤** | 股基 < 30% / 混合 < 20% / 债基 < 5% |
| **回撤持续时间** | < 12 月回血 = 纪律好 |
| **Ulcer Index** | < 10 优秀 / 10-15 合格 / > 20 差 |

### 4.3 α / β

| 指标 | 阈值 |
|------|------|
| **3Y α** | > 5% 优秀 / 2-5% 合格 / < 2% 看不出能力 |
| **β** | 0.8-1.1 跟随 / < 0.7 防御型 / > 1.2 高波动 |
| **α 显著性** | t-stat > 2 才算有效 |

### 4.4 被动基金（ETF）特有

| 指标 | 阈值 |
|------|------|
| **跟踪误差** | < 0.5% 优秀 / 0.5-2% 合格 / > 2% 差 |
| **跟踪差** | 年化 < ±1% 优秀 |
| **费率** | 股票 ETF < 0.5% / 债券 ETF < 0.3% / 主题 ETF < 0.75% |
| **日均成交额** | 美股 ETF > $5M / A 股 ETF > 1 亿元 |

## 5. 评分模型：四维打分（100 分）

### 主动基金

**管理人（25）**：

| 子项 | 分值 | 标准 |
|------|-----|------|
| 任职时长 | 10 | > 5Y 得 10 / 3-5Y 得 7 / 1-3Y 得 4 / < 1Y 得 0 |
| 管理规模 | 8 | 10-200 亿 得 8 / < 10 亿 得 4 / > 500 亿 得 3（规模陷阱） |
| 历史业绩穿越周期 | 7 | 经历牛熊 + 超额显著 = 7 / 单一行情 = 3 |

**收益质量（25）**：

| 子项 | 分值 | 标准 |
|------|-----|------|
| 3Y α | 10 | > 8% 得 10 / 4-8% 得 7 / 0-4% 得 4 / < 0 得 0 |
| 夏普 | 8 | > 1.5 得 8 / 1-1.5 得 5 / 0.5-1 得 2 / < 0.5 得 0 |
| Sortino / 卡玛 | 7 | Sortino > 2 或 Calmar > 1 得 7 / 次档得 4 |

**持仓质量（25）**：

| 子项 | 分值 | 标准 |
|------|-----|------|
| 风格稳定性 | 10 | 多期风格不漂移 = 10 / 轻微漂移 = 5 / 严重漂移 = 0 |
| 集中度合理 | 8 | 前 10 重仓 40-60% = 8 / > 70% 得 4（过度集中）/ < 30% 得 4（过度分散） |
| 持仓和策略一致 | 7 | 描述型 vs 实际持仓一致 = 7 / 矛盾 = 0 |

**风险控制（25）**：

| 子项 | 分值 | 标准 |
|------|-----|------|
| 最大回撤 | 10 | < 20% 得 10 / 20-35% 得 5 / > 35% 得 0 |
| 回撤恢复 | 8 | < 6M 恢复 = 8 / 6-12M = 5 / > 12M = 2 |
| 下行捕获 / 上行捕获 | 7 | 下捕 < 上捕 15%+ = 7 / 相近 = 4 / 下捕更高 = 0 |

### ETF

**跟踪质量（40）**：

| 子项 | 分值 |
|------|-----|
| 跟踪误差 | 15 |
| 跟踪差年化 | 15 |
| 再平衡纪律 | 10 |

**流动性 + 成本（40）**：

| 子项 | 分值 |
|------|-----|
| 日均成交 | 15 |
| 费率 | 15 |
| 折价 / 溢价 | 10 |

**底层资产质量（20）**：

| 子项 | 分值 |
|------|-----|
| 指数 / 策略设计 | 10 |
| 底层流动性 | 10 |

### 总分分档

| 总分 | 档位 | 配置建议 |
|------|------|---------|
| 85-100 | **核心持仓** | 15-30% 组合权重 |
| 70-85 | **战术持仓** | 5-15% |
| 55-70 | **观察** | 0-5% |
| < 55 | **避雷** | 不配 |

## 6. Worked Example：SMH（VanEck 半导体 ETF）vs 主动半导体基金

### 6.1 命题

**比较 SMH（美股半导体 ETF）和某主动半导体基金，哪个更值得配置？**

### 6.2 Step 1 分类

- SMH：被动 ETF，跟踪 MVIS 半导体指数
- 主动基金（假设名为"X 半导体主动"）：主动股票型

### 6.3 对比扫描

**SMH（ETF）**：

| 维度 | 数据（2024 年末） | 评分 |
|------|------------------|-----|
| 跟踪误差 | < 0.1% | 15/15 |
| 跟踪差年化 | -0.3% | 13/15 |
| 日均成交 | > $1B | 15/15 |
| 费率 | 0.35% | 13/15 |
| 前 10 重仓占比 | 70%（TSMC / NVDA / AVGO / AMD / QCOM 等） | 中性 |
| 近 3Y 年化 | ~35% | — |
| 近 3Y 最大回撤 | -40%（2022 加息周期） | — |
| 夏普 | ~0.9 | — |
| **总分** | | **88 / 100** |

**X 半导体主动**：

| 维度 | 数据 | 评分 |
|------|------|-----|
| 管理人任职 | 4 年 | 7/10 |
| 管理规模 | 80 亿 | 8/8 |
| 3Y α vs 半导体指数 | +3% | 5/10 |
| 夏普 | 1.0 | 5/8 |
| 最大回撤 | -45% | 3/10 |
| 下行捕获 | 上下捕捉差 5% | 4/7 |
| 风格稳定 | 轻度漂移（买了非纯半导体） | 5/10 |
| 集中度 | 前 10 重仓 55% | 7/8 |
| **总分** | | **约 72 / 100** |

### 6.4 决策

**优先配置 SMH**（总分 88 vs 72），原因：
1. SMH 跟踪误差极低 + 费率合理
2. 流动性强
3. 主动基金 α 仅 3% 不足以覆盖费率差 + 主动风险
4. 回撤 SMH 仍更小

**例外场景**：若主动基金明显在某细分赛道（如设备 / 材料）有选股优势且 α > 6%，可考虑配置 20-30%。

### 6.5 决策输出

```
动作：买入 SMH 15-20% 组合权重
仓位：15% 核心 + 5% 战术（等回调 $220 以下加仓）
时间：5Y+ 长期持有（半导体是长期成长赛道）
止损：半导体指数跌破 200MA → 减仓 50%
退出信号：
  - TSMC / NVDA 三连季度 Miss
  - 美国对华出口管制全面升级
  - 全球半导体 Capex YoY 连续下滑
```

## 7. 执行纪律

### 红线（违反 = 重做）

1. **只看近 1 年排名买基金** → 最典型的散户陷阱
2. **忽略管理人变动** → 明星经理离职后基金质量通常下降
3. **买年度冠军** → "冠军魔咒"次年大概率跑输
4. **大规模基金视而不见** → 500 亿+ 主动基金很难跑赢基准
5. **不看费率** → 10 年累计差 5%+
6. **ETF 不看流动性** → 买了卖不出去 / 高折溢价
7. **把管理费当小钱** → 年费 1% 长期吃掉 20% 收益
8. **基金 ≠ 股票，不择时** → 基金也可以择时（底层资产估值）

### 反模式

❌ **"晨星 5 星就买"**：5 星是历史评级，未来表现不保证
❌ **"基金经理上电视就买"**：媒体曝光度 ≠ 业绩
❌ **"全是朋友推荐的"**：没做独立分析的盲买
❌ **"老牌基金公司一定好"**：大公司也有烂基金
❌ **"定投就不管了"**：定投也要定期评估管理人
❌ **"收益为正就持有"**：要看 vs 基准超额

### 核查清单

- [ ] 分类识别清楚（股 / 混 / 债 / ETF / 另类）
- [ ] 主动基金看管理人 + 任职 3 年+
- [ ] 核心 4 指标（夏普 / 最大回撤 / α / 跟踪误差）都查了
- [ ] 持仓分析（集中度 + 风格稳定性）
- [ ] 费率合理
- [ ] 流动性（规模 / 成交）够
- [ ] 和当前宏观阶段匹配（调 `top-down-selection`）
- [ ] 止损触发 + 退出信号明确

## 8. 失效场景

### 8.1 历史数据不足（新基金）

**场景**：基金成立 < 2 年。

**退路**：
1. 无法算夏普 / 最大回撤
2. 依赖管理人历史（若有其他基金业绩可查）
3. 新策略 / 新主题基金可谨慎配置 3-5%

### 8.2 风格剧变

**场景**：基金经理跳槽 / 大幅调仓 / 主动转被动。

**退路**：
1. 视为"新基金"重新评估
2. 原持仓考虑赎回 + 换人
3. 观察 1-2 个季度新策略效果

### 8.3 极端市场

**场景**：2020 疫情 / 2008 金融危机 / 2022 加息冲击。

**退路**：
1. 基金评分机制暂停（数据极端失真）
2. 关注相对韧性（跌得比基准少就是 alpha）
3. 分批建仓 / 减仓而不是一刀切

### 8.4 清盘风险

**场景**：基金规模 < 5000 万元 或 连续 60 日低于 5000 万。

**退路**：
1. 立即观望，不新买
2. 已持有的考虑赎回转换
3. 关注清盘公告

### 8.5 跟踪偏离（ETF）

**场景**：ETF 跟踪差连续 6M > 1.5%。

**退路**：
1. 可能换指数编制方法 / 换托管机构
2. 对比同类 ETF 选更好的
3. 换成跟踪误差小的竞品

## 10. Disclaimer

- 基金历史业绩不代表未来
- α 和夏普含统计误差，不是确定性
- 管理人可能离职 / 策略可能变动
- 申购赎回费 / 持有期折扣因渠道而异
- 私募基金数据披露有限，置信度低
- 最终决策需结合个人风险偏好 + 投资期限 + 流动性需求

**"基金投资的秘密是长期持有 + 不看每日净值"** — 波动是成本，不是风险。
