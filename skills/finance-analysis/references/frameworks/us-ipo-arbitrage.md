# 美股 IPO 套利 Framework（US IPO Arbitrage）

> 美股 IPO 规则和港股差异巨大，HK 打法套到 US 上会踩坑。
>
> **核心差异**：无暗盘 / 无孖展 / 散户几乎分不到票 / 开盘由做市商撮合（常见 +20-80% 跳空）/ 180 天 Lockup 锁仓。
>
> **核心命题**：美股 IPO 主要不是"打不打"，而是"首日追 / 等 Lockup 回调 / 长期持有"三路径哪条 EV 最高。

---

## 1. 核心定位

### 适用场景

- 美股 NYSE / NASDAQ 上市的 IPO（传统 IPO，发行价区间形式）
- 直接上市（Direct Listing，如 Coinbase / Spotify）— 部分流程不同，后面标注
- 散户可以通过券商（IB / 富途 / 老虎 / Webull）少量分到 IPO 份额，但大部分机会是首日后买入

### 不适用场景

- **SPAC 合并**（DeSPAC）— 规则完全不同，走 event-driven
- **港股 IPO** → 走 `hk-ipo-arbitrage.md`
- **中概股回港 / 二次上市** → 走 `hk-ipo-arbitrage.md` + 单独评估流通量差
- **Pre-IPO 二级市场** → 非公开市场，本 framework 不适用
- **长期基本面投资已锁定** → 走 `valuation-dcf` + `alpha-generation`

## 2. 美股 IPO 规则速览 vs 港股对照

| 维度 | 港股 IPO | 美股 IPO |
|------|---------|---------|
| **散户打新** | 一人一手制，大部分能中 | 券商分配（IB/富途少量），大部分分不到 |
| **暗盘** | T-1 19:00 左右暗盘（盈透 / 富途 / 老虎国际） | **无暗盘**，直接 T0 开盘 |
| **杠杆** | 孖展 10-20x 常见 | 无专门 IPO 孖展，普通 Margin 2-4x |
| **开盘机制** | 港交所集中竞价 | NYSE / NASDAQ 做市商撮合，常见 20-80% 跳空 |
| **定价** | 招股区间 + 公开 / 国际配售 | S-1 区间 → 路演 → 提价次数 → 定价 |
| **披露文件** | HK 招股书 + 补充资料 | **S-1**（注册声明，最详细）+ **424B4**（定价版招股书） |
| **锁仓期** | 无统一规则（通常约定） | **Lockup 180 天**（标准，内部人 + pre-IPO 投资人不能卖） |
| **基石 / 锚** | 基石投资者（前 10 大，披露） | Cornerstone / Anchor Investor（较少见，不一定披露） |
| **破发率** | 历史约 30-40% | 历史约 30-50%（2022 年后大幅上升） |
| **首日涨幅** | 概念强时 +50-200% | 中位数 +15-30%，但开盘跳空常占大头 |

### 关键启示

- **美股散户"打新"本质是"首日后买入"**，因为分不到票
- **Lockup 抛压**是美股独有的可预测事件（T+180）
- **定价区间调整**是最可观测的需求温度信号

## 3. 核心方法论：三路径 EV 比较

美股 IPO 给散户的"路径"：

```
路径 A 不参与首日，等 Lockup 回调  （多数情况 EV 最高）
路径 B 首日追涨                    （高风险，仅当极热需求时）
路径 C 长期持有（≥ 2 年）           （回归基本面 + 选择权）
```

**核心原则**：

1. **不要默认"路径 B 首日追"** — 美股散户分不到开盘前票，追涨常买在 +80% 跳空高位
2. **Lockup 到期日是可预测事件** — 提前观察内部人 / pre-IPO 投资人是否减持信号
3. **S-1 披露最全** — 比 HK 招股书多（lockup 详情、股权结构、律师 / 承销商风险段）
4. **定价区间调整 = 需求温度**：提价 1 次 = 需求正常 / 提价 2 次 + 超区间 = 需求热 / 低于区间 = 需求弱

## 4. 决策树

```
命题输入：XX 美股 IPO 打不打 / 怎么买
  │
  ├─ Step 1: S-1 核心数据提取
  │   ├─ 营收 / 亏损 / 用户数 / GMV 等关键指标
  │   ├─ 股权结构（founders + pre-IPO 投资人 + ESOP）
  │   ├─ Lockup 条款（180 天标准 / 部分早期 Lockup 90 天）
  │   ├─ 承销商（Top 3: GS / MS / JPM） + 律师（顶级所）
  │   └─ 风险段（Risk Factors，必读）
  │
  ├─ Step 2: 定价区间评估（需求温度）
  │   ├─ 初始区间（路演前）
  │   ├─ 提价次数（1 次 = 正常 / 2+ 次 = 热 / 0 次 + 压价 = 弱）
  │   ├─ 最终定价 vs 初始中值
  │   └─ 超额认购倍数（机构路演，通常不完全披露）
  │
  ├─ Step 3: Lockup 抛压评估
  │   ├─ T+180 流通量 = IPO 流通股 × (1 + Lockup 解锁倍数)
  │   ├─ 内部人持股成本（VC 轮次均价 vs IPO 价）
  │   └─ 解锁日压力评分（解锁规模 / 日均成交量）
  │
  ├─ Step 4: 首日机制风险
  │   ├─ 预计开盘跳空幅度（按 S-1 估值 vs 二级市场预期）
  │   ├─ 历史同类 IPO 首日中位数 / 分布
  │   └─ NYSE vs NASDAQ 差异（NYSE 做市商模式）
  │
  ├─ Step 5: 三路径 EV 比较
  │   │
  │   ├─ 路径 A：T0 不买，T+180 前后买入
  │   │   EV_A = E[T+180 价格] - 成本 + 买入时间价值
  │   │
  │   ├─ 路径 B：T0 开盘追涨，T+N 卖出
  │   │   EV_B = E[T+N 价格] × 概率 - E[止损] × 概率
  │   │
  │   └─ 路径 C：T0-T+30 建仓，持有 ≥ 2 年
  │       EV_C = E[T+24M 价格] - 成本 × 概率
  │
  ├─ Step 6: 选择主路径 + 触发条件
  │   ├─ 默认：路径 A（等 Lockup 机会）
  │   ├─ 路径 B：仅当超额需求（定价超区间 20%+ + 机构路演 10x 超额）
  │   └─ 路径 C：基本面极强 + 价格合理 + 愿意承受 -50% 波动
  │
  └─ Step 7: 决策输出
      ├─ 路径选择 + 触发动作
      ├─ 止损位 / 仓位 / 时间维度
      └─ 决策卡（`templates/ipo-decision-card.md` US 扩展版）
```

## 5. 量化信号

### Step 2 定价区间信号

| 信号 | 含义 | 路径偏向 |
|------|------|---------|
| 提价 2+ 次 + 超初始区间 20%+ | 需求极热 | 路径 B 可考虑（谨慎） |
| 提价 1 次 | 正常热度 | 路径 A 为主 |
| 不提价 / 范围内定价 | 冷静 | 路径 A |
| 压价 / 低于初始区间 | 需求弱 | 路径 A 或不参与 |
| 取消上市 | 极冷 | 不参与 |

### Step 3 Lockup 抛压强度

```
抛压评分 = (解锁股数 × T+180 预估价) / (上市后 6 个月平均日成交额)
```

| 抛压评分 | 含义 | 预估回调幅度 |
|---------|------|-------------|
| < 3 | 无压力 | < 10% |
| 3-10 | 中度压力 | 10-25% |
| 10-30 | 强压力 | 25-45% |
| > 30 | 极强压力 | 45%+（+ 悲观情景触发） |

### Step 4 首日跳空预估

```
预估跳空 = (二级市场隐含估值 - S-1 估值) / S-1 估值
         = 机构需求反映 × 散户抢筹情绪

典型分布：
  中位数 +15-30% 跳空
  热门 +50-80%
  极热 +100%+（罕见，2020-2021 常见，2023 后罕见）
  破发 -10 to -30%（市场疲软时 30-40% IPO 破发）
```

## 6. 三路径 EV 详细分析

### 路径 A：不参与首日，等 Lockup 前后建仓

**时间线**：
- T0-T+30：观察首日 + 第一份财报
- T+90：观察第二份财报 + 机构建仓信号
- T+150-T+180：Lockup 解锁预期 → 价格常提前回调 10-30%
- T+180-T+210：实际解锁 + 抛压落地
- T+210+：基本面 + 估值重新定价

**何时用**：
- 定价区间正常热度（不是极热）
- 首日跳空 > 50%
- Lockup 抛压评分 > 10
- 基本面**值得长期持有**但当前贵

**EV 估算**：
```
EV_A = P(Lockup 解锁后回调 30%+) × (T+180 价格 - T+180 买入价)
     - 等待期机会成本
     + 买入后 1-2 年基本面兑现空间
```

**典型盈亏比**：1.5-3:1（温和风险 + 中等回报）

---

### 路径 B：首日追涨

**时间线**：T0 开盘追 → T+1 到 T+30 卖出

**何时用**（所有条件全满足）：
- 定价提价 2+ 次 + 超区间 20%+
- 预估跳空 < 30%（否则追涨买在高位）
- 承销商 Top 3 + 基石 / Anchor 质量高
- 同类近 3 个月 IPO 首日平均涨幅 > +50%
- 主题极热

**EV 估算**：
```
EV_B = P(T+N 价格 > 开盘价 × 1.2) × (盈利)
     - P(回落至 -30% 以下) × (-30% × 仓位)
     - 手续费
```

**典型盈亏比**：0.5-1:1（高风险，多数情况 EV 为负）

**红线**：
- 预估跳空 > 60% → 追涨 EV 为负
- 定价低于区间 → 不追
- 首日破发 → 不抄底（除非极强基本面）

---

### 路径 C：长期持有（≥ 2 年）

**时间线**：T0-T+60 分批建仓 → 持有 2 年以上

**何时用**：
- 基本面极强（商业模型清晰 + 市场领先 + 成长 + FCF 转正路径）
- 估值：按 `valuation-dcf` 三情景，中性情景下现价隐含 < 100% 中性
- 愿意承受 Lockup 期 -40% 波动
- 属于"赛道龙头"（见 `stock-selection.md` 龙头筛）

**EV 估算**：
```
EV_C = Σ(2 年后情景股价 × 概率) - 分批建仓均价
```

**典型盈亏比**：3-5:1（高耐心要求，需承受 Lockup 波动）

**红线**：
- DCF 估值显示当前已超过乐观情景 → 不建仓
- 商业模型未经历完整周期 → 推迟至 T+12M 后再评估
- 财报造假质疑 → 不碰

## 7. Worked Example：DoorDash IPO（2020-12）

### 7.1 背景

- **发行日**：2020-12-09
- **承销商**：Goldman Sachs / JPMorgan（顶级）
- **初始区间**：$75-85
- **提价次数**：2 次（$75-85 → $90-95 → **最终定价 $102**）
- **超额认购**：机构路演 10x+（披露不完整）
- **S-1 核心数据**：
  - 2020 营收 $2.9B（YoY +200%）
  - 净亏损 $149M（COVID 推动盈利改善）
  - 美国外卖市场份额 50%+（龙头）
  - Lockup 180 天（标准）

### 7.2 Step 2 定价区间评估

- **提价 2 次 + 超初始区间 ~21%** → 需求热（接近极热红线）
- **Goldman + JPM + 高质量基石** → 承销结构稳
- **路演超额 10x+** → 机构强需求
- **结论**：定价区间评分 = 极热

### 7.3 Step 3 Lockup 抛压评估

- **IPO 流通**：33M 股 ×  $102 = $3.4B
- **Lockup 解锁（T+180）**：pre-IPO 投资人 + ESOP，约 170M 股（相对 IPO 流通 5x）
- **内部人成本**：VC 轮次均价 $15-40（vs IPO $102，大幅盈利）
- **解锁规模 / 日均成交**：约 15-25x → **抛压评分 = 强**

### 7.4 Step 4 首日跳空预估

- S-1 估值 $33-35B → 市场隐含估值（超热）可能 $60-70B
- **预估跳空 +70-90%**（实际 +78% 开盘 $182，收盘 $189 +85%）

### 7.5 Step 5 三路径 EV 比较

**路径 A：不参与首日，等 Lockup 回调**
```
预估 T+180 价格：$130-150（Lockup 抛压 + 基本面兑现对冲）
建仓区间：$140（实际解锁前回调到 ~$140）
T+24M 价格估算：$150-220（中性情景）
EV_A ≈ +10% ~ +50%
盈亏比 ≈ 2:1
```

**路径 B：首日追涨**
```
开盘价 $182 → +78%
T+1 到 T+30 价格：$175-255（高波动）
止损：$140（-23%）
T+30 中位数：$195（+7% vs 开盘）
EV_B = 0.5 × 7% - 0.3 × (-23%) - 手续费 = +3.5% + 6.9% ≈ +10%
盈亏比 ≈ 0.8:1（薄利）
```

**路径 C：长期持有 2 年**
```
T0 建仓 $182 → T+24M 预估 $150-220
悲观 30%：$100 → -45%
中性 50%：$175 → -4%
乐观 20%：$250 → +37%
EV_C = 0.3 × (-45%) + 0.5 × (-4%) + 0.2 × (+37%) = -8.3%
盈亏比 ≈ 负
```

### 7.6 Step 6 路径选择

- **首选路径 A**（等 Lockup 回调 $140-150 建仓）
- **不选路径 B**（首日 +78% 跳空太大，EV 太薄）
- **不选路径 C**（$182 建仓长期持有 EV 为负，需要等回调）

### 7.7 Step 7 决策输出

- **T0 动作**：观望，不追
- **T+150-T+180 动作**：Lockup 解锁前价格回调至 $140-145 区间分批建仓
- **仓位**：2-3%（Satellite 性质）
- **止损**：$115（-20% from 建仓价）
- **目标**：T+24M $180-220（+30-55%）
- **时间维度**：2-3 年

### 7.8 事后回溯（2022-2024 视角）

**实际走势**：
- 2021-02 高点 $255（+40% vs 路径 A 建仓价 $140）
- 2022-10 低点 $42（-70% vs 路径 A 建仓价）← 宏观加息 + 外卖赛道估值压缩
- 2024-Q1 回到 $140（路径 A 盈亏平衡）
- 2025 约 $170（+21%）

**教训**：
- 路径 A 在 Lockup 后确实建仓较合理，但没算清"宏观周期"因素（2022 加息）
- 若严格执行 -20% 止损（$115），2022 年被洗出 → 错过 2024 回归
- 本案例展示：**IPO 决策方法论正确不代表短期收益** — 需要耐心和宏观判断
- Coinbase（2021 直接上市）是对照案例：规则不同但结论类似（首日追涨 EV 差）

## 8. 执行纪律

### 红线（违反 = 重做）

1. **不读 S-1 就下决策** → 核心数据来源于 S-1
2. **不评估 Lockup 抛压** → 美股独有的可预测事件，必做
3. **首日直接追涨跳空 > 60%** → EV 几乎必为负
4. **路径 C 用 IPO 价 / 首日价建仓** → 几乎总是贵
5. **不做三路径 EV 比较** → 等于闭眼买
6. **把 HK 一人一手逻辑套到美股** → 散户分不到，方法论错
7. **Lockup 期不止损** → 内部人抛压是可预测的下跌动力
8. **认为"热门 = 一定涨"** → 2022-2023 热门 IPO 破发率 > 50%

### 反模式

❌ **首日 FOMO**：看 +80% 跳空就追，不算 EV
❌ **"基本面好 = 可以买任何价"**：DoorDash $250 买入即使持有 2 年也是 -30%
❌ **跳过 S-1 风险段**：IPO 律师写的 Risk Factors 常常预言破发原因
❌ **用 HK 孖展思维**：美股没有专门 IPO 孖展，用普通 Margin 追 IPO 风险极高
❌ **"等 Lockup"不设具体建仓价**："Lockup 后买"太模糊 → 需要具体触发价
❌ **不看宏观**：2021 下半年 Fed 转鹰派所有 IPO 回调，不看宏观会踩坑
❌ **直接上市当 IPO 看**：Direct Listing 无 Lockup（有的），流通量完全不同

### 核查清单

- [ ] 读了 S-1（至少摘要 + Lockup 条款 + Risk Factors）
- [ ] 定价区间评估 + 提价次数已记录
- [ ] Lockup 抛压评分算出来了
- [ ] 预估跳空幅度 + 对应路径偏向
- [ ] 三路径 EV 各自计算（不能只算一条）
- [ ] 选定主路径 + 备选路径 + 触发条件
- [ ] 止损位 + 仓位 + 时间维度写明
- [ ] 宏观判断加入（利率周期 / 风险偏好）

## 9. 失效场景

### 9.1 宏观剧变

**场景**：IPO 发布后 1-3 个月宏观环境剧变（加息 / 衰退预期 / 地缘战争）。

**退路**：
1. 所有路径 EV 全部失效
2. 必须重算 DCF / 估值基准
3. 优先风险规避，降低仓位或不参与

### 9.2 Direct Listing（直接上市）

**场景**：Coinbase / Spotify / Slack 等直接上市，不是传统 IPO。

**差异**：
- 无 IPO 定价区间
- 参考价由 NYSE 做市商给（Coinbase $250 参考价）
- **无 Lockup 或短 Lockup**（有些只有 90 天或不锁）
- 流通量从 T0 就接近全流通
- 首日价格波动极大（Coinbase 首日 $381 高 / $310 低）

**退路**：
1. 无 Lockup → 路径 A 简化为"等风险偏好变化"
2. 首日 EV 分析方法类似但无 180 天锚
3. 建议仓位上限降低

### 9.3 SPAC DeSPAC

**场景**：通过 SPAC 合并上市（2020-2021 流行）。

**退路**：
1. 完全不同的规则（赎回权 / 股价保护 / PIPE）
2. 不用本 framework，走 event-driven（0.4.0）

### 9.4 中概股回美

**场景**：中概股二次上市 / 回美（不多见但存在）。

**退路**：
1. 关注"美国 ADS vs 港股股权"差价
2. 关注 HFCAA 退市风险
3. 单独评估，不套用常规 IPO

### 9.5 极端破发

**场景**：首日破发 > 30%，市场情绪极度悲观。

**退路**：
1. 不抄底（除非基本面极强且估值已回到 VC 轮次）
2. 观察 T+90 / T+180 稳定后再评估
3. 历史 2022-2023 破发 IPO 多数继续下跌

## 10. 与其他 framework 的组合

### 典型协作流

```
"DoorDash 要 IPO，打不打？"
  │
  ├─ us-ipo-arbitrage（主）：规则 + 三路径 EV
```

## 12. Disclaimer

- 本 framework 基于公开信息和历史 IPO 数据
- 三路径 EV 计算含主观概率，不代表未来可复制
- Worked Example 数据（DoorDash 2020-12）基于公开报道，精确数字请查 S-1 原文 + SEC filings
- 2022 年后市场结构变化（加息 / 风险偏好 / SPAC 退潮），历史经验需要校准
- 税务、资金账户类型（IRA vs Margin）、券商分配机制影响实际执行
- 最终决策需结合个人风险偏好和宏观判断

**IPO 是机制博弈 + 基本面博弈的叠加** — 机制看 framework，基本面看 DCF，缺一不可。
