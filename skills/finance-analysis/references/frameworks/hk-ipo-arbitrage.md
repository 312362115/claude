# HK IPO 打新套利 Framework

> 港股新股首日套利的决策框架。**不研究公司基本面，只研究打新 EV 和执行纪律**。
>
> 适用场景：港股主板 / GEM 创业板 IPO 招股阶段（T-10 到 T+1）。
>
> 不适用：中长期持仓（那是 `stock-selection.md` 的事）。

---

## 核心结论先行

港股 IPO 首日涨幅的 80% 可解释性来自 **3 个可量化信号**，T-1 晚（招股结束当晚）就能判定：

1. **香港公开发售超额认购倍数**（决定性信号，权重 25-35%）
2. **暗盘涨幅**（T-1 晚最后校准，权重 20%）
3. **国际配售超额倍数 + 基石质量 + 概念热度**（组合，权重 40-55%）

**不是"预期差"**——这些信号所有人都看得到。**真正能赚的是纪律差**：
- 谁能在 T-1 晚算清楚 EV 满融
- 谁能在 T0 开盘按计划卖出不追高

## 三阶段决策树

### 阶段一：T-10 到 T-5（聆讯通过到招股启动）

**目标**：判断要不要做这单打新 + 申请孖展额度

```flow
聆讯通过（T-10）
↓
概念是否爆点？→ 杭州六小龙 / AI / 硬科技 / 消费龙头
↓ 是
保荐人是谁？→ 中金 / 摩根大通 / 摩根士丹利 / 高盛 = 头部
↓ 头部
基石投资者是否披露？→ 看招股书提前版
↓
启动准备：申请孖展额度（券商要几天审核）
```

**行动清单**：
- [ ] 确认股票代码 + 招股日期
- [ ] 阅读招股书前几页（了解概念和赛道）——不深读基本面
- [ ] 对标近 3 个月同概念 IPO 首日表现
- [ ] 向券商申请孖展额度（富途/老虎/辉立/耀才等）
- [ ] 准备现金等打新

**止损点**：如果以下任一触发，**放弃该单**：
- 保荐人为二线（无历史打新胜绩）
- 基石全部是"小基石"（单个 < 1000 万美元）
- 概念冷门 + 赛道下行周期

### 阶段二：T-2 到 T-1 晚（招股期到招股截止）

**目标**：根据超额认购倍数决定孖展杠杆

```flow
招股启动
↓
每日监控香港公开 &amp; 国际配售超额倍数（财华/AAStocks 公布）
↓
招股截止（T-1 早）→ 最终超额认购公布（T-1 晚）
↓
套用 EV 决策矩阵 → 决定杠杆倍数
```

### 阶段三：T-1 晚 到 T0 开盘（暗盘到首日）

**目标**：根据暗盘表现设定首日卖出策略

```flow
T-1 晚 暗盘开盘（约 16:15-18:30 香港时间）
↓
暗盘涨幅 > +50%？
↓ 是
设定 T0 开盘卖单（挂单价 = 暗盘收盘价 × 1.0-1.05）
↓
T0 开盘 → 自动执行卖单
↓
未全部成交？→ 10 分钟后改市价单
```

## 核心信号矩阵

### 信号 1：香港公开发售超额认购倍数（权重 25-35%）

**数据源**：招股结束当晚（T-1 19:00 左右）财华/智通/AAStocks 报道

**判断表**（基于港股 2020-2025 年样本）：

| 超额倍数 | 首日大涨（+50% 以上）概率 | 首日中位数涨幅 | 建议杠杆 |
|---------|-------------------------|--------------|---------|
| > 2000 | ~95% | +120% | 满融（10-20x） |
| 500-2000 | ~85% | +70% | 高融（5-10x） |
| 100-500 | ~70% | +35% | 中融（3-5x） |
| 20-100 | ~50% | +10% | 低融（1-3x） |
| 5-20 | ~30% | 平盘 | 不孖展 |
| < 5 | ~15% | 破发风险 | 不打 |

**回拨机制**：公开发售 ≥15 倍超额 → **强制回拨**（公开份额从 10% 提升到 50%），进一步推高中签率但压制暗盘溢价空间。

### 信号 2：国际配售超额倍数（权重 10-15%）

**数据源**：同上

**为什么重要**：散户情绪可以造假（拉亲戚朋友打新），**机构真金白银不会骗**。

| 国际配售超额 | 机构信心 |
|------------|---------|
| > 20 倍 | 极强（稀缺标的） |
| 10-20 倍 | 强（合理定价 + 好故事） |
| 3-10 倍 | 一般（常规消化） |
| < 3 倍 | 弱（发行困难信号） |
| < 1 倍 | 破发警报（包销商压力山大） |

### 信号 3：基石投资者质量（权重 10-15%）

**关键指标**：
- 基石总认购占全球发售比例（通常 20-40%，太高压制流通盘价格发现）
- 基石锁定期（港股惯例 6 个月）
- 基石阵容质量

**质量分级**：

| Tier | 典型机构 | 加分 |
|------|---------|------|
| T1 | 大型险资（泰康/平安/阳光）、GIC/淡马锡、BlackRock、Fidelity、高瓴 | +10 |
| T2 | 国际一线资管（Mirae/Capital/Vanguard）、产业龙头、头部私募 | +5 |
| T3 | 国资背景、普通战略合作伙伴、中小私募 | +2 |

**基石总额 / 全球发售比例**：

| 占比 | 含义 |
|------|------|
| > 50% | 筹码极度锁定，暗盘流通盘小，易暴涨但定价不透明 |
| 30-50% | 最优区间（群核 37.12%） |
| 15-30% | 正常 |
| < 15% | 基石参与度低（需警惕） |
| 0% | 无基石（高风险，多为小盘股） |

### 信号 4：发行定价在招股区间位置（权重 5-10%）

- **上限定价** = 承销商信心足，通常配合高超额 → 强信号
- **中枢定价** = 中性
- **下限定价** = 发行压力（机构不愿高价）→ 减分

### 信号 5：保荐人 + 稳定价格经纪商（权重 5-10%）

| 保荐人 | 历史胜率 |
|--------|---------|
| 摩根大通 / 摩根士丹利 / 高盛 / 中金 | 高（首日破发率 < 20%） |
| 中信证券 / 建银国际 / 华泰国际 | 中高 |
| 二三线 | 打折评估 |

**稳定价格经纪商绿鞋**：
- 绿鞋未动用 = 股价强势，无需托底
- 绿鞋全部动用 = 接近发行价，警告信号

### 信号 6：暗盘表现（权重 20%）— **T-1 晚最后信号**

**数据源**：富途 / 辉立 / 华盛 暗盘系统，T-1 晚 16:15-18:30

**判断**：

| 暗盘涨幅 | T0 首日开盘预期 | 建议卖出挂单价 |
|---------|----------------|--------------|
| > +100% | +100% 到 +200% | 暗盘收盘价 × 0.98-1.02 |
| +50% 到 +100% | +40% 到 +100% | 暗盘收盘价 × 0.95-1.0 |
| 0 到 +50% | 平开到 +40% | 暗盘收盘价 × 0.95 |
| 平盘或负 | 破发风险 | **中签即开盘市价卖** |

**暗盘最高价 vs 收盘价**：
- 收盘价 < 最高价 50% → 盘中已有获利盘抛售，首日开盘可能回落
- 收盘价 接近 最高价 → 买盘持续强劲，首日大概率高开高走

### 信号 7：流通盘 + 概念热度（权重 ±5%）

- **流通盘 < 10%**：筹码结构极紧，弹性大（上涨/下跌都剧烈）
- **杭州六小龙 / AI 第一股 / 消费龙头** 类 narrative：散户情绪加分 +5
- **冷门赛道 + 无故事**：减分 -5

## EV 计算模型（打新的数学题）

### 变量定义

```
H = 香港公开发售超额倍数
σ = 中签率 ≈ 1 / (1 + H) × 2（回拨后）
        实际查：券商官方中签率公告

P1 = 首日开盘涨幅预期（由信号矩阵推算）
P_issue = 发行价
shares_per_lot = 每手股数（如 500 股）
cost_per_lot = P_issue × shares_per_lot × 1.01（含手续费）
profit_per_lot = P_issue × shares_per_lot × P1

L = 孖展杠杆倍数（如 10x）
rate = 孖展年化利率（通常 7-12%）
days = 资金占用天数（T-4 申请 到 T+1 结算，约 6 天）
interest_per_lot = cost_per_lot × (1/L) × rate × days / 365 × L
                 = cost_per_lot × rate × days / 365

EV_per_lot = σ × profit_per_lot - interest_per_lot
EV_total = L × EV_per_lot
```

**关键洞察**：孖展杠杆 L 同时放大中签率（×L）和利息（×L），**最终 EV 与 L 线性相关**，不是"越高杠杆越赚"（有券商额度限制和爆仓风险）。

### 群核科技案例代入

```
H = 1591
σ = 实际公告中签率 3%（融资 1 手申购 ~ 33.3 手中签 1 手）
shares_per_lot = 500
P_issue = 7.62
cost_per_lot = 7.62 × 500 × 1.01 ≈ 3848
首日开盘 20.70 → P1 = (20.70 - 7.62) / 7.62 = 1.717（+171%）
profit_per_lot = 7.62 × 500 × 1.717 ≈ 6541

按开盘卖（首日 5490 HKD 是收盘价计算，开盘卖更多）
取 5490 保守估算

L = 10x
rate = 8%
days = 5
interest_per_lot ≈ 3848 × 8% × 5/365 ≈ 4.2 HKD（单手自有资金部分）
每申购 1 手融资 10 手 → 利息约 42 HKD/手

σ × profit_per_lot - interest = 0.03 × 5490 - 42 = 164.7 - 42 = +123 HKD EV/手
申 10 手 → EV = +1230 HKD
申 100 手（10x 融资）→ EV ≈ +12,300 HKD
```

**对比不融资单手打新**：
```
中签率 0.3% × 盈利 5490 = EV = 16 HKD/手（几乎等于没打）
```

**结论**：群核这种 1591 倍超额，**不满融 = 放弃 EV**。

### 杠杆极限

孖展 ≠ 越高越好，限制因素：
- 券商额度上限（一般 10-20x，超过要补仓）
- 自有资金机会成本
- 破发黑天鹅风险（若破发，杠杆亏损被放大）
- 心理承受能力

**建议**：
- EV 矩阵判断"首日大涨概率 > 85%" → 可用 10x 以上
- 85% > 概率 > 60% → 5-10x
- 60% > 概率 > 40% → 3-5x
- 概率 < 40% → 不孖展或不打

## 执行纪律（最容易翻车的环节）

### 纪律 1：T-1 晚定好卖出计划，不盘中临时起意

**必须** 在 T-1 晚 暗盘收盘后立刻决定：
- T0 开盘挂单价位
- 如果 3 分钟未成交改市价单
- 如果低开破位怎么办

**禁止** 在 T0 上午临时决定"再等等"。

### 纪律 2：不追高（最致命陷阱）

**群核教训**：
- 开盘 20.70（+171%）
- 最高 约 23 HKD（+202%，暗盘顶）
- 收盘 18.60（+144%）

**开盘追高买入的人**：平均亏 10%（开盘→收盘回落）。

**规则**：
- 中签打新者 = T0 开盘卖（不等反弹）
- 未中签者 = **不追高**（彻底回避首日）

### 纪律 3：破发不加仓

如果首日低开 / 破发（<发行价），**不加仓抄底**：
- 港股破发首日的股票，60% 在 1 个月内继续下跌
- 绿鞋托底时间有限（通常 2-4 周）
- 抄底属于"中长期持仓"，应走 `stock-selection.md` 路径

### 纪律 4：6 个月解禁前 1-2 月开始减仓

如果计划持仓 6 个月以上（比如打新中签后留一半长期）：
- 基石锁定期通常 6 个月
- **解禁前 1-2 个月**（即 T+4.5 到 T+5 个月）开始减仓 50%
- 历史统计：港股解禁前后 2 周平均下跌 8-15%

## Worked Example：群核科技 0068.HK（2026-04-17 上市）

### 招股期信号（T-5 到 T-1）

| 信号 | 数值 | 打分（满分） |
|------|------|-----------|
| 保荐人 | 摩根大通 + 建银国际 | 9/10（头部） |
| 概念 | 杭州六小龙 + 空间智能第一股 | 19/20（双概念） |
| 定价 | 7.62（招股上限 6.72-7.62） | 9/10（上限） |
| 基石占比 | 37.12%（4.55 亿 HKD / 9 家） | 16/20（最优区间） |
| 基石质量 | 泰康+阳光（T1险资） + Mirae（T2） + 禾赛（T2产业） | 14/20 |
| 公开超额 | **1591 倍** | 25/25（天花板） |
| 国际超额 | 14.46 倍 | 12/15（强） |
| 暗盘涨幅 | +169.3%（盘中 +201.8%） | 20/20（极强） |
| 流通盘 | 9.45%（极小） | +5（加分项） |

**总分 129/140 ≈ 92 分 → 首日大涨概率 >90%**

### EV 决策（T-1 晚）

```
H = 1591 → σ ≈ 3%
P1 预期 = +150%（暗盘 +169% 保守折算）
单手盈利 ≈ 5490 HKD（按首日收盘价 18.60 计）
建议杠杆：10x（孖展 1074 倍说明市场已融到位）

申购策略：
- 保守：融资 5x 申 5 手，EV ≈ +615 HKD
- 中性：融资 10x 申 10 手，EV ≈ +1230 HKD
- 激进：融资 20x 申 20 手，EV ≈ +2460 HKD（注意券商额度 + 爆仓风险）
```

### T0 卖出执行（4-17 早）

- 暗盘收盘 20.52 → 开盘挂单价 20.50-21.00
- 实际开盘 20.70 → 挂单在 21.00 的可能未成交
- 挂单在 20.50 的秒成交
- 保守派：20.50 挂单全部卖出，落袋
- 激进派：20.50 卖 50%，留 50% 看走势（当日收盘 18.60，实际亏 9%）

**后见之明教训**：群核开盘 = 首日高点之一，**全仓开盘卖是最优解**。

### 本案例的方法论验证

| 信号 | 预判（T-1 晚） | 实际（T0） | 准确度 |
|------|--------------|----------|--------|
| 首日大涨概率 > 90% | ✅ 预判 | +144% 收盘 | ✅ 准确 |
| 开盘 > +150% | ✅ 预判 | +171% 开盘 | ✅ 准确 |
| 高开低走概率 | ⚠️ 警告 | 开盘 20.7 → 收盘 18.6 | ✅ 准确 |
| 打新 10x EV 正 | ✅ 预判 | 实际盈利 | ✅ 准确 |

**方法论有效**。但请注意：这是**上市前可见信号的合理推论**，不是"预测未来"。当任一信号背离时（比如公开超额 1591 倍 + 暗盘破发），模型会失效——那种情况历史罕见。

## 反面情况（方法论失效场景）

本 framework 不是圣杯，在以下场景会失准：

1. **大盘黑天鹅**：T-1 晚港股暴跌 5%+ → 首日开盘情绪崩溃，即使超额高也可能开盘即跌
2. **同期竞品分流**：同一周多只热门新股同时招股，资金面被稀释
3. **发行人负面消息**：T-1 到 T0 之间突发实控人被调查/财务造假传闻
4. **极端小盘股**：发行股数 < 5000 万股的迷你 IPO，流动性差，首日可能冲高即熄火
5. **监管风暴**：行业突发监管收紧（如 2021 教培 / 2022 平台）

遇到以上任一情况：**暂停孖展、降低杠杆、不追高**。
