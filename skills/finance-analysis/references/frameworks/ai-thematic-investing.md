> **定位**：为"AI 是 50 年一遇科技革命"这个判断提供可操作的投资框架。

> 前置：`alpha-generation.md`（Alpha 执行差方法论）+ `hk-ipo-arbitrage.md`（短期套利）。本 framework 是**大主题战略层**。

## 核心判断：AI 是 50 年级范式转换

**历史对照**：
| 时代 | 浪潮 | 代表龙头 | 长期涨幅 | 泡沫期 |
|------|------|---------|---------|-------|
| 1990s | 互联网 | MSFT / AMZN / CSCO | 100x-1000x | 2000-2002 崩 80% |
| 2000s | 移动互联网 | AAPL / TCEHY | 30x-500x | 无显著崩 |
| 2010s | 云 + SaaS | AMZN / CRM | 20x-100x | 2020-2022 回调 |
| **2020s** | **AI** | **NVDA (+ ?)** | **已 140x，仍中期** | **未到** |

**四大驱动叠加**：
1. Transformer 架构（2017）→ 通用智能底座
2. Scaling Law（2020）→ 规模越大性能越好
3. GPU 算力爆发（H100 2022）→ 工程可能性
4. 数据飞轮（ChatGPT 2022）→ 商业化闭环

**判断**：这不是周期股，是**生产力工具革命**。持有期 5-10 年维度。

## AI 产业链完整地图（8 层）

```
┌─────────────────────────────────────────────────┐
│ Layer 8  电力 / 基础设施（核电/电网/变压器）      │ ← 2025-2027 Alpha
│ Layer 7  具身智能 / 机器人（人形/工业）            │ ← 2025-2027 Alpha
│ Layer 6  数据 / 训练素材（Scale AI/合成数据）     │ ← 冷门，关键
├─────────────────────────────────────────────────┤
│ Layer 5  应用层（ToC App/垂直 SaaS）              │ ← 竞争激烈
│ Layer 4  开发者工具（Cursor/LangChain/向量DB）    │ ← Alpha 中
│ Layer 3  基础模型（OpenAI/Anthropic/MiniMax/Zhipu）│ ← MiniMax/Zhipu
├─────────────────────────────────────────────────┤
│ Layer 2  数据中心 / 云（AWS/CoreWeave）           │ ← 已 price-in
│ Layer 1  算力底座（GPU/HBM/CPO/PCB/液冷）         │ ← 2022-24 10x 完成
└─────────────────────────────────────────────────┘
```

- 2025-2027 是**具身智能苗头→共识**窗口

## 趋势启动识别系统（3 个月 KPI）

**硬核 KPI**：任何 AI 子赛道趋势启动后，本 framework 必须在 **3 个月内**识别到。**3 个月没识别 = 框架失败需要反思**。

### "趋势启动" 的量化定义（三条同时满足）

| 维度 | 量化信号 | 数据源 |
|------|---------|-------|
| **行业共识形成** | ≥ 3 家 Tier-1 卖方积极研报 + 主流财经头条 ≥ 5 次/月 + Google Trends 关键词 YoY +50% | 卖方研报库 / Bloomberg Terminal / Google Trends |
| **个股明牌** | 赛道龙头（市占 >20%）+ 最近财报 Beat 2 季度 + 机构调研 ≥ 10 家/季度 | 公司公告 + 机构调研公告（A 股必披露） |
| **开始走趋势** | 股价突破 **200 日均线 +5%** + 成交量 ≥ 50 日均值 **1.5 倍** + 30 天涨幅 ≥ **+15%** | 任何股价软件 |

**三条同时触发** = 加仓信号（不是"预测"，是"确认正在发生"）。

### 趋势启动传导链（从事件到可识别）

```
事件层（NVDA Q3 Capex 超预期 / GPT-5 发布 / Optimus 量产确认）
  ↓ T+0 到 T+7 天
产业层（产业链标的开始启动）
  ↓ T+7 到 T+30 天（A 股滞后 2-4 周）
股票层（龙头 + 二线股价异常）
  ↓ T+15 到 T+60 天
信号层（本 framework 必须捕获）：
  · 龙头突破 200 日均线 + 成交量 1.5x
  · 财经头条覆盖升温 + Google Trends
  · 分析师密集上调
  ↓
月度 Checkup 识别：T+30 到 T+90 天
  ↓
加仓决策（若论点/仓位允许）
```

**兜底机制**：每月 1 日执行"AI 月度巡检"（见下节），强制每月扫描 8 层所有趋势。

### AI 月度巡检 Checklist（90 分钟）

每月 1 日（或月初第一个交易日）执行，sub-skill template 见 `templates/ai-monthly-checkup.md`。

**Part A — 宏观温度计**（10 分钟）
- [ ] NVDA 股价 vs 200 日均线位置
- [ ] 纳指 AI / SMH 半导体 / BOTZ 机器人 指数 YoY
- [ ] 美元指数 + 10Y 美债收益率
- [ ] 美联储利率路径（是否利空成长股？）

**Part B — 产业数据更新**（30 分钟）
- [ ] 最新 NVDA 季度 Capex 指引（加速/减速？）
- [ ] 四大云厂 Capex 指引（AMZN/MSFT/GOOGL/META）
- [ ] HBM 价格周报（TrendForce / DRAMeXchange）
- [ ] 光模块出货（中际旭创/新易盛月度）
- [ ] 机器人量产进度（Optimus / Figure / 智元 / UBTech）
- [ ] 核电 PPA 合约公告（大厂签约频率）
- [ ] 中国 AI 政策 / 监管动态

**Part C — 8 层子赛道巡检**（30 分钟）

对每个 Layer（8 个）跑以下 5 条：

| # | 巡检项 | 触发值 |
|---|-------|-------|
| 1 | 该 Layer 龙头股价位置 | 突破 200 日均线 +5% |
| 2 | 成交量 | 近 30 天均值 > 50 日均值 1.5x |
| 3 | 最近财报方向 | Beat or Miss? |
| 4 | 产业数据 YoY | 加速/减速？|
| 5 | 媒体覆盖度 | Google Trends YoY + 头条次数 |

任一 Layer 若 1+2+3 同时满足 → **标记为"趋势启动中"**。

**Part D — 个股 Watchlist 警报**（15 分钟）

对 pre-defined 20 只 AI 相关标的跑：
- [ ] 近 30 天涨幅 > +15%？
- [ ] 成交量 > 50 日均值 1.5 倍？
- [ ] 机构调研 ≥ 10 家/季度（A 股）？
- [ ] 分析师共识上调？

**Part E — 决策**（5 分钟）
- [ ] 新加仓机会（Part C 中 "三项全中" 的 Layer + Part D 中警报触发的个股）
- [ ] 减仓信号（FOMO 峰值 + 基本面恶化）
- [ ] 新 watchlist 添加
- [ ] 论点崩塌清仓
