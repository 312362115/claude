# 相对估值（Valuation - Relative）

> 用 PE / PS / PB 和**对标池中位数 + 三情景 forward 业绩 + 概率加权**算出"隐含预期位置"。
> 不做"静态 PE 对标"，做**预期驱动的情景估值**。
>
> **主入口认知基础**：参见 `command-driven-analysis.md` §"股市定价是预期驱动" + §"情景分析 + 概率加权" + §"铁律 -0.5 估值重塑归因"。本 framework 是把那些认知**流程化、可执行化**。

---

## 1. 核心定位

### 适用场景

- 商业模型成熟、可比公司多的标的（如 CATL、东山精密、NVDA）
- 有稳定或可预估的营收、利润、净资产
- 赛道内有明确的"龙头 / 次龙头 / 跟随者"分层
- 需要在"买 / 持 / 观望 / 减仓"间做决策，且必须给出目标价区间和盈亏比

### 不适用场景

- 早期亏损期、商业模型未定型（→ 走 `valuation-dcf.md` 三情景）
- 赛道完全没有可比公司（如全球独一份的商业模型）→ 退 DCF + 同类水平扩展
- 主题情绪极端期（市场给出的 PE 完全失真 ±3σ）→ 退 DCF + 警惕"预期透支"
- 单次性重大事件驱动（并购、分拆、资产剥离） → 走 event-driven（0.4.0）

## 2. 核心方法论：三段式

```
Step A 对标池构建
Step B 三情景 forward 业绩 × 合理 PE × 概率加权 → 合理价
Step C 加主题溢价 → 隐含预期位置 → 盈亏比 → 决策
```

**关键原则**：

1. **PE 不是估值锚，未来业绩是锚** — 当前 PE 只是"预期已定价到什么程度"的表象
2. **对标池必须深挖产业链位置**（参见 `command-driven-analysis.md` 铁律 -0.3）
3. **情景必须三档**（乐观/中性/悲观），单一假设 = 直觉 ≠ 估值
4. **概率必须保守** — 主观高估乐观概率是典型反模式

## 3. 决策树

```
命题输入：XX 股相对估值如何？现价值不值？
  │
  ├─ Step 1: 商业模型分类 → 选主估值指标
  │   ├─ 成长股、有利润：PE（forward）
  │   ├─ 早期亏损 / 超高增速：PS
  │   ├─ 重资产（银行 / 保险 / 周期）：PB
  │   └─ 多业务：按业务分拆，各自选指标（铁律 -0.5）
  │
  ├─ Step 2: 对标池构建（3-5 家）
  │   ├─ 同板块发现 5 种方法至少用 2 种（铁律 -1）
  │   ├─ 业务重合度 × 市值接近度 × 数据可得性 排序
  │   └─ 龙头 1 + 直接对手 2-3 + 上/下游 1
  │
  ├─ Step 3: 对标 forward PE 分布
  │   ├─ 取最新财报季 + 卖方一致预期
  │   ├─ 算中位数 + 龙头溢价 + 规模折价系数
  │   └─ 标注每家的"产业链位置 + 护城河"（铁律 -0.3）
  │
  ├─ Step 4: 本标的三情景 forward 业绩
  │   ├─ 乐观：主题 + 业绩全兑现（核心催化剂全部落地）
  │   ├─ 中性：部分兑现（主要客户签单 + 产能落地）
  │   └─ 悲观：仅现有订单 + 竞争加剧 / 替代冲击
  │
  ├─ Step 5: 每情景 × 合理 PE → 合理市值 / 股价
  │   ├─ 乐观用"次线龙头 PE × 成长溢价"
  │   ├─ 中性用"对标中位 PE × 规模折价"
  │   └─ 悲观用"对标下四分位 PE"
  │
  ├─ Step 6: 三情景概率加权
  │   ├─ 乐观 20-40%（保守设定）
  │   ├─ 中性 40-60%
  │   └─ 悲观 10-30%（反面证据强制不低于 20%）
  │
  ├─ Step 7: 加主题溢价因子
  │   ├─ 主线主题（2024-2026 AI 等）：1.3-1.5x
  │   ├─ 细分主题：1.1-1.3x
  │   ├─ 无主题：1.0x
  │   └─ 逆主题 / 政策打压：0.7-0.9x
  │
  ├─ Step 8: 计算隐含预期位置
  │   ├─ 现价 / 加权合理价 = 隐含预期百分比
  │   └─ 映射到决策区间（见 §5 映射表）
  │
  └─ Step 9: 输出决策 + 盈亏比
      ├─ 买入 / 持有 / 观望 / 减仓
      ├─ 目标价区间（乐观上沿 + 中性下沿）
      ├─ 止损价（悲观情景 - 10%）
      └─ 仓位建议 + 时间维度
```

**末端必然产出**：一张 `valuation-card`（见 §10）。

## 4. 量化信号（每节点可观测阈值）

| 节点 | 可观测信号 | 通过阈值 | 未通过处理 |
|------|-----------|---------|----------|
| Step 2 对标池 | 找到 ≥ 3 家对标 + 数据完整 | YES | 退 DCF 或水平扩展 |
| Step 3 分布 | 对标 forward PE 变异系数 < 0.5 | YES | 说明可比性差，加重权重警告 |
| Step 4 业绩 | forward 业绩有公开卖方覆盖 ≥ 3 家 | YES | 自己做分部预测 + 透明化假设 |
| Step 5 PE | 选定 PE 和"产业链位置 + 护城河"匹配 | YES | 重新评估规模折价系数 |
| Step 6 概率 | 反面证据搜索 ≥ 3 条 + 悲观概率 ≥ 20% | YES | 补反面搜索（否则是"选择性乐观"） |
| Step 7 溢价 | 主题溢价可追溯到具体主题（不是"感觉热"）| YES | 降 1 档（细分主题降到 1.0x） |
| Step 8 位置 | 算出的隐含位置落在 §5 映射表五档之一 | YES | 重算中性情景权重 |

## 5. 核心评分：隐含预期位置映射

**估值映射公式**：

```
加权合理股价 = Σ(情景概率_i × 情景股价_i) × 主题溢价因子

隐含预期位置 = 现价 / 加权合理股价

盈亏比 = (乐观上沿 - 现价) / (现价 - 悲观下沿)
```

**决策映射表**：

| 隐含预期位置 | 含义 | 决策 | 目标价 | 仓位 |
|-------------|------|------|--------|------|
| < 60% 悲观情景 | 预期严重低估 / 市场过度悲观 | **强买入** | 乐观上沿 | 5-8% |
| 60-100% 悲观情景 | 预期偏低 | **买入** | 乐观上沿 | 3-5% |
| 100% 悲观 - 100% 中性 | 预期合理 | **持有** / 观望 | 中性 + 10% | 2-3% |
| 100% 中性 - 70% 乐观 | 预期偏高 | **观望 / 小仓试错** | 乐观 70% | 1-2% |
| 70-100% 乐观 | 乐观已打满 | **谨慎 / 等回调** | — | 0（不追） |
| > 100% 乐观 + 主题溢价 | 情绪顶部透支 | **减仓 / 清仓** | 止损 悲观 - 10% | 减到 0-1% |

**盈亏比红线**：

| 盈亏比 | 决策修正 |
|-------|---------|
| > 3:1 | 强信号，可加仓 |
| 1.5-3:1 | 正常买入 |
| 0.6-1.5:1 | 观望 |
| < 0.6:1 | 不买 / 减仓 |

## 6. Worked Example：东山精密（002384, 2026-04-22 分析）

### 背景

- 赛道：PCB + 光模块（AI 服务器双卡位）
- 核心子公司：索尔思光电（光模块）、东山本体（PCB）
- 股价：186 元（2026-04-22）
- Trailing PE 248x → 按静态思维"泡沫 80-120%"

### Step 1：商业模型分类

- **多业务标的** → 必做分拆（铁律 -0.5）
- 分拆结果：
  - PCB 主业：成熟制造业，PE 估值
  - 索尔思光模块：成长期，高增速，PE forward 估值（非 PS，因已有利润）

### Step 2：对标池构建

| 对标 | 定位 | forward PE | 规模对比 | 产业链位置 |
|------|------|-----------|---------|----------|
| 沪电股份 | 主业 PCB（AI 服务器） | 30-35x | 1x | 同业同规模 |
| 鹏鼎控股 | 消费电子 PCB（苹果供应链） | 20-25x | 2x | 偏消费电子 |
| 中际旭创 | 光模块龙头 | 28-32x | 10x | 全球第一（1.6T 主力供应商） |
| 生益科技 | 覆铜板（PCB 上游） | 25-28x | 1.5x | 上游 |
| 永鼎股份 | 光模块二线 | 45-55x | 0.3x | 次线 |

**分布分析**：
- PCB 业务 forward PE 中位数：25-30x
- 光模块 forward PE 中位数：35-45x（龙头中际 28-32，次线 45-55）
- 规模折价系数：索尔思 vs 中际 = 1:10 → 折价 0.5-0.6 或"次线 + 成长溢价"给 PE 40-50x

### Step 3-5：三情景 forward 业绩 + 合理股价

| 情景 | 假设 | 2027 预期业绩 | 合理 PE | 合理市值 | 合理股价 |
|------|------|--------------|---------|---------|---------|
| **乐观 30%** | NVDA + AWS + Meta 全签单，1.6T 订单落地，索尔思 2027 营收 200-300 亿 | 净利 25-35 亿 | 45-50x | 1125-1750 亿 | 90-140 元 |
| **中性 50%** | Meta 继续交付 + NVDA 部分签单，2027 营收 100-150 亿 | 净利 12-18 亿 | 35-40x | 420-720 亿 | 50-80 元 |
| **悲观 20%** | NVDA 没签单 + CPO 方案替代传统光模块，2027 营收 50-80 亿 | 净利 5-8 亿 | 28-32x | 140-256 亿 | 25-35 元 |

### Step 6：概率加权期望价

```
加权合理价 = 0.30 × 115 + 0.50 × 65 + 0.20 × 30
           = 34.5 + 32.5 + 6.0
           ≈ 73 元（基础）
```

### Step 7：主题溢价

- 2024-2026 主线主题：AI 基础设施
- 光模块属于 AI 主线核心细分 → 溢价因子 1.4x

```
加权合理股价（含溢价）= 73 × 1.4 ≈ 100-140 元
```

### Step 8：隐含预期位置

```
现价 186 / 加权合理价 120（中值）= 155%
对应映射表：> 100% 乐观 + 主题溢价 → 情绪顶部透支
```

**更精细分解**：186 元隐含的是"**乐观情景 + 主题溢价已打满 + 25% 额外透支**"（乐观上沿 140 × 1.33 ≈ 186）。

### Step 9：决策输出

- **动作**：减仓 40-50%（锁利润，保留仓位观察 NVDA 订单落地）
- **目标价区间**：上行 -25%～-50%（至 90-140），下行 -55%～-85%（至 25-50）
- **盈亏比**：(140 - 186)/(186 - 50) = -46/136 = **负 1:3**，极差
- **止损价**：150（失守光模块主题溢价锚）
- **时间维度**：1-6 月观察窗（等 Q2 中报 + NVDA 订单进展）
- **关键催化剂**：NVDA 订单签单 / Q2 索尔思营收同比 / CPO 商业化进展

**决策卡产出**：见 `templates/valuation-card.md` 的东山精密填充版。

## 7. 执行纪律

### 红线（违反 = 重做）

1. **未做多业务分拆** → 用集团 PE 下结论 = 违反铁律 -0.5
2. **对标 < 3 家** → 单点没锚，估值分析无意义
3. **未标注产业链位置** → 违反铁律 -0.3，把次线按龙头估
4. **只做单点估值不做三情景** → 违反"预期驱动"核心认知
5. **乐观概率 > 50%** → 主观偏乐观，必须补反面证据
6. **悲观概率 < 20%** → 反面证据不足，反向论证失败
7. **隐含预期位置 > 100% 乐观 + 溢价，还建议买入** → 逻辑错位
8. **主题溢价无触发条件** → "感觉热"不是依据

### 反模式

❌ **静态 PE 对标**："PE 248x vs 行业 30x → 泡沫 800%"（把 PE 当锚，不看未来业绩）
❌ **龙头 PE 硬套次线**：东山索尔思按中际旭创 PE 估（规模差 10 倍）
❌ **单点目标价**："我觉得合理 100 元"（无情景 = 无估值）
❌ **主题溢价吃到 2.0x+**：除非全球独一份，否则溢价 1.5x 已到顶
❌ **概率权重随意凑**："各 1/3"（没压测反面证据）
❌ **隐含位置 -> 买卖决策脱钩**：算出 150% 透支还说"可持有"（逻辑断裂）

### 核查清单

提交决策卡之前过一遍：

- [ ] 多业务做了分拆吗？
- [ ] 对标 ≥ 3 家 + 规模在 ±10 倍内吗？
- [ ] 每家对标标注了产业链位置 + 护城河吗？
- [ ] 三情景 forward 业绩有具体数字吗？
- [ ] 每情景的合理 PE 有"锚"依据吗？
- [ ] 反面证据 ≥ 3 条吗？
- [ ] 悲观概率 ≥ 20% 吗？
- [ ] 主题溢价触发条件写清楚了吗？
- [ ] 隐含预期位置落在哪档 + 决策匹配吗？
- [ ] 盈亏比算出来了吗？
- [ ] 止损价和仓位定了吗？

任何一项 ❌ → 回到对应 Step 修复。

## 8. 失效场景（何时本 framework 不适用）

### 8.1 对标池无法构建

**场景**：标的是"全球独一份"（如港股 pure-play LLM MiniMax，全球无纯 LLM 上市公司对标）。

**退路**：
1. 退 DCF 估值（`valuation-dcf.md`）+ 补产业链同业（上游 / 下游）做交叉验证
2. 如 DCF 也不可行（早期无现金流），标注"估值无法给出置信区间，只做定性判断"
3. 绝对不用"美股同业 + 全球扩展"硬凑 3 家，宁可放弃

### 8.2 主题情绪极端期

**场景**：整个板块 PE 都在历史 3σ 以外（如 2021 年初新能源车 PE 全行业 150x+）。

**退路**：
1. 相对估值本身失效（比"谁更泡沫"没有意义）
2. 退 DCF 做"真正锚"
3. 或走 `alpha-generation.md` 的"顶点识别"模式（判断主题何时转向）

### 8.3 转型期

**场景**：标的从业务 A 转型到业务 B，过去财报不代表未来（如东山精密从消费电子转向 AI）。

**退路**：
1. 对过去 trailing PE 完全无效
2. 必须做业务分拆（铁律 -0.5）+ 每个业务独立选对标
3. 转型尚未验证的业务只能给"低概率情景 + 选择权估值"

### 8.4 财报严重不可信

**场景**：标的财报出现重大质疑（如商誉占比 > 50%、关联交易比例异常、审计师变更）。

**退路**：
1. 相对估值基于财报数字，财报不可信 = 输入污染
2. 标注"估值分析不适用，本质是信用事件而非估值事件"
3. 走风险规避原则，不建议任何买入决策

### 8.5 流动性枯竭

**场景**：标的日均成交 < 500 万元（A 股）或 < 100 万美元（美股），估值没法交易。

**退路**：
1. 即使估值低估，执行层面无法建仓或退出
2. 标注"理论估值 vs 流动性风险"，前者不可作为决策依据

## 11. Disclaimer

- 本 framework 是**方法论工具**，不是投资建议
- 所有 Worked Example 基于公开信息的历史分析，结果不代表未来可复制
- 对标池选择和概率估计含主观判断，需跟踪自我迭代闭环（见 `self-improvement-loop.md`）
- 最终决策需结合个人风险偏好、资金结构、税务情况
- 法规变更、黑天鹅事件、数据质量问题可能导致本方法失效
- 对 2026 年之后的估值假设可能过时，使用时请核对最新财报和卖方一致预期

**估值不是预测，是锚** —— 有锚才有盈亏比，有盈亏比才有纪律。
