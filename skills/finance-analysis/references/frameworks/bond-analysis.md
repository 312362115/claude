# 债券分析框架（Bond Analysis）

> **不研究公司故事**，研究"**这只债券的风险收益比 / 利率敏感度 / 违约概率**"。
>
> **核心动作**：债券分类 → 三维扫描（久期 / 信用 / 利差）→ 宏观锚（利率路径 / 信用周期）→ 决策（买 / 持 / 换 / 避）。

---

## 🔥 最高原则：债券不是"保本"，是"利率 + 信用"的 bet

> 散户最大的债券认知错误：**"债券是保本的"**。错。
>
> 债券实质 = **未来现金流 / (1 + YTM)^t** — 只要 YTM 变动，价格就会大幅波动。
>
> 30 年美债在 2022 年跌 **-39%**（TLT），比股市还剧烈。

### 三条推论

1. **买债券 = 赌利率方向**（至少部分）。久期越长，利率赌注越大
2. **买债券 = 赌信用不违约**。高息 = 信用补偿，不是免费午餐
3. **债券价格和 YTM 反向**。YTM 涨 → 价格跌，绝对不是"保本"

### 散户常见陷阱

❌ "国债 4%年化比银行利息高，稳赚不赔" → 加息 30 年国债跌 30%+
❌ "高息债券 8% 看着挺好" → 实际是垃圾债（BB- 或以下），违约率 3-5%/年
❌ "反正到期就还本" → 中间若急需流动性，卖出可能大幅亏损
❌ "债基稳健" → 债基有久期暴露，2022 年债基跌 10-20% 比比皆是

## 1. 核心定位

### 适用场景

- 散户 / 个人配置债券资产（固收部分）
- 债券 ETF 对比（TLT / IEF / LQD / HYG / 国债 ETF）
- 可转债分析（股债双性特征）
- 组合中的"债券权重 / 久期权重"决策
- 利率周期判断下的债券配置策略

### 不适用场景

- **机构级债券套利** → 超出散户能力（需要 Bloomberg Terminal / 衍生品）
- **非标债 / 信托** → 散户很难评估
- **外汇债 / 主权债深度分析** → 需要宏观 + 汇率专业知识
- **具体到某张债的技术性交易** → 需要做市商报价 / 流动性支撑

### 产出边界

产出"**债券决策卡**"：是否买 / 买什么久期 / 买什么评级 / 什么止损 / 持多久 / 配比多少。

|------|------|----------|---------|

## 3. 三维分析法

```
任何债券决策 = 久期维 × 信用维 × 利差维 的组合判断
```

### 3.1 久期维（Duration） — 利率敏感度

**定义**：久期 = 债券价格对 YTM 变化的敏感度（粗略等于"每变化 1% 利率，价格变化百分比"）。

**公式**：
```
价格变化 ≈ -久期 × YTM 变化

例：久期 18 的 TLT，如果利率上升 1% → 价格下跌约 18%
```

**分类**：

| 久期 | 分类 | 典型代表 | 利率敏感度 |
|------|------|---------|-----------|
| < 2 | 短久期 | SHY / 货币基金 | 低（利率变动几乎不动） |
| 2-5 | 中短久期 | IEI | 中 |
| 5-10 | 中久期 | IEF / LQD | 高 |
| 10-20 | 长久期 | VCLT | 很高 |
| > 20 | 超长久期 | TLT / EDV | 极高 |

**判断要点**：
- 利率见顶 + 即将降息 → **加长久期**（TLT / EDV）
- 利率未见顶 / 仍可能上行 → **缩短久期**（SHY / 货币）
- 利率横盘 → 中久期（IEF / LQD）

### 3.2 信用维（Credit） — 违约概率

**评级 → 违约率**：

| 评级 | 类别 | 历史 1 年违约率 | 息差补偿 | 散户建议 |
|------|------|----------------|---------|---------|
| AAA-AA | 最高等级 | < 0.1% | 50-100 bps | 可买 |
| A | 高等级 | 0.1-0.3% | 100-150 bps | 可买 |
| BBB | 投资级下限 | 0.3-0.8% | 150-250 bps | 可买（LQD 以此为主） |
| BB | 高收益级上限 | 1-2% | 300-500 bps | 谨慎 |
| B | 高收益 | 3-5% | 500-800 bps | HYG 等组合化参与 |
| CCC 及以下 | 垃圾级 | > 10% | 1000+ bps | **避雷** |
| D | 违约 | 100% | — | 已违约 |

**核心公式**：
```
预期回报 = YTM - 违约概率 × 违约损失率

YTM 7% - 违约率 3% × 损失率 40% = 预期回报 5.8%
YTM 5% - 违约率 0.3% × 损失率 40% = 预期回报 4.88%

高息债 7% 实际预期回报仅比投资级多 1%，但波动风险大很多
```

### 3.3 利差维（Spread） — 相对定价

**4 种重要利差**：

| 利差 | 公式 | 信号 |
|-----|------|------|
| **期限利差** | 10Y - 2Y 美债 | 负值（倒挂）= 衰退预警 |
| **信用利差** | BBB - US10Y | > 200 bps 信用担忧 / < 100 bps 过度乐观 |
| **高收益利差** | HY - US10Y | > 800 bps 风险厌恶 / < 300 bps 泡沫 |
| **中美利差** | US10Y - CN10Y | 影响人民币 / 外资 A 股 |

**利差的价值**：
- 利差**走宽** → 市场避险 → 高评级受益 / 低评级压制
- 利差**收窄** → 市场乐观 → 低评级反弹 / 高评级平淡
- **倒挂**（期限利差 < 0）→ 历史上 12-18 月内大概率衰退

## 4. 决策树

```
命题输入：XX 债券 / XX 债券 ETF 值不值得买？/ 现在配多少固收？
  │
  ├─ Step 1: 分类识别（利率债 / 信用债 / 可转债 / HY / TIPS / 混合）
  │
  ├─ Step 2: 久期判断
  │   ├─ 当前 YTM / 久期
  │   ├─ 假设 YTM 变动 ±1%，价格变化多少
  │   └─ 匹配利率预期（调 top-down-selection §4.1.2 加息降息预期）
  │
  ├─ Step 3: 信用判断（信用债 + 可转债必做）
  │   ├─ 主体评级 + 债项评级
  │   ├─ 历史违约率参考
  │   ├─ 财务杠杆 / 现金流 / 利息保障倍数
  │   └─ CDS / 利差当前位置
  │
  ├─ Step 4: 利差分析
  │   ├─ 当前利差 vs 历史分位（5Y / 10Y）
  │   ├─ 走宽 / 收窄预期
  │   └─ 和其他信用利差 / 股市风险偏好 对比
  │
  ├─ Step 5: 宏观锚
  │   ├─ 利率路径（FedWatch / 国内货币政策）
  │   ├─ 信用周期（信贷宽松 / 紧缩）
  │   └─ 通胀路径（对 TIPS 重要）
  │
  ├─ Step 6: 特殊债券额外分析
  │   ├─ 可转债：股性（正股分析）+ 债底 + 溢价率
  │   ├─ TIPS：实际利率 + 通胀盈亏平衡
  │   └─ 市政债：免税后等效 YTM
  │
  ├─ Step 7: 流动性 + 成本
  │   ├─ 场内成交量（ETF 日均 / 单券成交）
  │   ├─ 买卖价差
  │   └─ 基金 / ETF 费率
  │
  └─ Step 8: 决策输出
      ├─ 买入 / 持有 / 观望 / 替换
      ├─ 仓位（固收占组合 %）
      ├─ 久期配比（短中长久期权重）
      └─ 退出信号（利率拐点 / 利差极端 / 违约出现）
```

## 5. 评分模型（投资级债券 / 债券 ETF）

### 四维打分（每维 25 分，总 100）

**久期匹配（25）**：

| 子项 | 分值 | 标准 |
|------|-----|------|
| 久期 vs 利率预期匹配 | 15 | 利率见顶 + 长久期 = 15 / 不匹配 = 0 |
| 久期波动承受度 | 10 | 用户风险偏好匹配 = 10 |

**信用质量（25）**：

| 子项 | 分值 | 标准 |
|------|-----|------|
| 评级 | 10 | AAA-A = 10 / BBB = 7 / BB = 3 / B 以下 = 0 |
| 违约率补偿 | 8 | 利差能补偿违约率 = 8 |
| 发行人财务健康 | 7 | 利息保障 > 5x = 7 / 3-5x = 4 / < 3x = 0 |

**YTM 吸引力（25）**：

| 子项 | 分值 | 标准 |
|------|-----|------|
| 绝对 YTM | 10 | 历史 80% 分位 = 10 / 50% = 5 / 20% = 0 |
| 相对同类 | 8 | 同久期同评级中位以上 = 8 |
| 税后实际收益 | 7 | 考虑税 + 通胀后仍为正 = 7 |

**流动性 + 成本（25）**：

| 子项 | 分值 | 标准 |
|------|-----|------|
| 日均成交 | 10 | 美国 ETF > $50M / 中国 > 1 亿元 |
| 买卖价差 | 8 | < 0.1% = 8 / > 0.5% = 0 |
| 费率 | 7 | < 0.1% = 7 / 0.1-0.5% = 4 / > 0.5% = 0 |

### 总分分档

| 总分 | 档位 | 配置 |
|------|------|------|
| 85-100 | **核心固收** | 15-40% 组合权重 |
| 70-85 | **战术配置** | 5-15% |
| 55-70 | **观察** | 0-5% |
| < 55 | **避雷** | 不配 |

## 6. Worked Example：TLT 2022 → 2024 周期

### 6.1 背景

- **TLT**（iShares 20+ Year Treasury Bond ETF）
- 久期 ~18，几乎无信用风险（100% 美国国债）
- 2021-07 峰值 ~$150 → 2023-10 低点 ~$82（**-45%**）
- 2024 年反弹至 $96-100

### 6.2 2022-10 时点分析（Fed FFR 3.75%，刚开始加息路径）

**Step 1 分类**：利率债 / 长久期 ETF

**Step 2 久期**：18，YTM 当时 ~4.5%
- 假设利率再涨 1% → 价格下跌约 18%（实际 2023 跌更多）

**Step 3 信用**：美国国债，无信用风险

**Step 4 利差**：10Y-2Y 已倒挂 -50 bps → 衰退预警

**Step 5 宏观锚**（当时视角）：
- Fed 仍在加息周期（FFR 3.75% → 5.25% 已 priced in）
- 市场预期 2023 继续加息 + 维持高位
- CPI 8% + 仍在高位
- **判断**：利率未见顶，长久期时机未到

**评分**（2022-10）：
- 久期匹配：5/25（利率还在上行，不匹配）
- 信用质量：25/25（国债满分）
- YTM 吸引力：12/25（4.5% 不算低但利率还要涨）
- 流动性：25/25（TLT 极其流动）
- **总分**：67/25+25+25=67 → 观察档

**决策**：观望，等利率见顶信号（失业率 / CPI 拐点 / Fed 鸽派转向）

### 6.3 2024-10 时点重新分析（Fed FFR 4.75%，开始降息）

**Step 2 久期**：18，YTM ~4.3%（从 5%+ 下降）

**Step 5 宏观锚**（2024 年视角）：
- Fed 已降息 50 bps（9 月 50 / 11 月 25 / 12 月 25）
- 市场预期 2025 继续降息 2-4 次
- 通胀接近 2% 目标
- **判断**：利率已见顶 → 长久期时机来了

**评分**（2024-10）：
- 久期匹配：20/25（降息周期 + 长久期）
- 信用质量：25/25
- YTM 吸引力：18/25（4.3% 相对历史仍较高）
- 流动性：25/25
- **总分**：88/100 → 核心固收

**决策**：
- 买入 TLT 15-20% 组合权重
- 目标：若 Fed 降息 200 bps 累计 → TLT 理论上涨 20-30%（久期 18 × 1.5-2% 降息幅度）
- 止损：若 Fed 转鹰 / 通胀反弹 → -10% 减仓

### 6.4 事后回溯（2025 年视角）

**实际走势**：
- 2024-10 $95 → 2025-01 $89（Fed 放鹰 "higher for longer"）→ 2025-03 $93
- 降息幅度不及预期 → TLT 上涨不及预期
- 利率波动导致净值波动大

**教训**：
- 宏观判断对了（利率见顶），但节奏判断不到位（降息速度比想象慢）
- 久期 18 是双刃剑：**降息慢 = 长久期风险持续**
- **改进**：类似决策未来可以选 IEF（久期 8）取一半弹性 + 一半保护

**归因**：D3 预判不具体（应该预判"降息多少次 / 到多少 BPS"，不是笼统"降息"）

## 7. 可转债特殊分析（中国市场）

### 7.1 可转债本质

```
可转债 = 股性（转股价值）+ 债性（到期债底）+ 期权价值（转股权）
```

### 7.2 三个关键指标

| 指标 | 公式 | 判断 |
|------|------|------|
| **转股价值** | (正股价 / 转股价) × 100 | 越接近 100 = 越平衡 / > 130 = 偏股 / < 70 = 偏债 |
| **纯债价值** | PV(未来票息 + 本金, YTM) | 债底，转股失败时的保本价 |
| **转股溢价率** | (转债价 - 转股价值) / 转股价值 × 100% | < 0 = 折价 / 0-20% 合理 / > 50% 溢价过高 |

### 7.3 可转债决策矩阵

| 场景 | 决策 |
|------|------|
| 转股价值 > 120 + 溢价 < 10% | **跟正股强，偏股策略** |
| 转股价值 80-120 + 溢价 10-30% | **股债平衡，适合持有** |
| 转股价值 < 80 + 债底稳 | **偏债底策略，持有到期** |
| 溢价 > 50% + 正股弱 | **估值过高，避雷** |

## 8. 执行纪律

### 红线（违反 = 重做）

1. **忽视久期**："国债 4% 稳健"没看久期 = 2022 年踩坑
2. **只看高 YTM** 不看违约率 = 垃圾债陷阱
3. **不看宏观利率路径** = 买在利率顶部 / 卖在底部
4. **高息债 7-10% 单一重仓** = 信用事件一次清零
5. **可转债只看股性 / 只看债底** = 错过平衡配置
6. **债基无视久期 beta** = 以为债基都稳健
7. **忽略买卖价差** = 场内债小单成本吞掉几年利息

### 反模式

❌ **"债券稳健"迷思**：忽视加息周期的价格冲击
❌ **高息即好**：不算违约率补偿是否到位
❌ **短期交易债券** = 交易成本大于利息
❌ **美债 = 无风险** = 真忘了久期 + 汇率风险
❌ **不做通胀调整** = 4% YTM 减 3% 通胀实际仅 1%
❌ **可转债忽视正股** = 偏股时债性不能保护
❌ **买单券不买 ETF** = 散户分不够分散

### 核查清单

- [ ] 分类识别清楚（利率 / 信用 / 可转 / HY / TIPS）
- [ ] 久期 + 利率预期匹配
- [ ] 信用评级 + 违约率检查（非国债）
- [ ] 利差位置 + 走势预期
- [ ] 宏观锚（利率 / 信用周期 / 通胀）
- [ ] 流动性 + 费率合理
- [ ] 税后实际 YTM > 0
- [ ] 仓位 + 久期配比明确

## 9. 失效场景

### 9.1 极端利率冲击

**场景**：疫情 / 黑天鹅导致利率一夜暴跌或暴涨。

**退路**：
1. 久期 beta 短期会完全主导价格
2. 基本面 / 评级短期失效
3. 观察 3-6 个月稳定后再评估

### 9.2 信用事件全面爆发

**场景**：类 2008 金融危机，信用利差全面走阔。

**退路**：
1. 所有非国债大幅折价
2. 即使 AAA 公司债也会被拖累
3. 国债 + 现金为王，等待稳定

### 9.3 通胀失控

**场景**：通胀大幅超出 Fed 目标（> 8%）。

**退路**：
1. 名义债券全面下跌
2. 转 TIPS / 抗通胀资产
3. 缩短久期避免利率冲击

### 9.4 可转债正股退市风险

**场景**：可转债对应正股 ST / 退市。

**退路**：
1. 债底都可能失效
2. 立即减仓，不抄底

### 9.5 流动性枯竭

**场景**：特定单券成交 < 1 万元 / 日。

**退路**：
1. 即使估值便宜，退不出
2. 改买 ETF 或主流券

## 11. Disclaimer

- 债券的 YTM 和当前价格含预期，未来实际回报可能偏差
- 信用评级是历史 + 模型判断，可能突然下调
- 利率预期可能突变（黑天鹅 / 政策转向）
- 可转债含衍生品特征，风险高于普通债券
- 税务影响（市政债 / 不同国籍）因人而异
- 最终决策需结合个人税务 / 期限 / 流动性需求

**"债券不是保本产品，是利率和信用的投资品"** — 理解这点，才配买债券。
