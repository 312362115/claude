# 选股框架（Stock Selection）

> **alpha-generation 的可执行简化版**。不做方法论深讲，只做"从池子到待研究清单"的可执行流程。
>
> **核心动作**：4 步筛选器（赛道 → 龙头 → 质量 → 成长）+ 30/60/100 打分 → Top-N 待研究清单。
>
> **和 alpha-generation 的边界**：
> - `alpha-generation.md`：讲"为什么 Alpha 在执行不在发现"+ 5 执行差方法论（精度/时点/仓位/持有/顶点）
> - `stock-selection.md`（本文件）：讲"从赛道池怎么筛到 N 只待研究"+ 可执行清单

---

## 1. 核心定位

### 适用场景

- 用户给赛道/池子，要求筛出 Top-N（如"A 股 AI 算力里选 3 只"、"纳斯达克 100 里谁最值得研究"）
- 主题投资的标的选取（AI / 机器人 / 新能源 / 创新药）
- 量化筛选前的"定性 + 半定量"漏斗
- 组合重平衡：候选池 → 新增仓位候选

### 不适用场景

- **单股深度分析**（已锁定标的） → 走 `valuation-relative` / `valuation-dcf` / `alpha-generation`
- **全市场扫描**（量化层） → Tier 2 本地栈 + Python 脚本（见 `integrations/local-data-stack.md`）
- **事件驱动机会**（并购 / 拆分 / IPO） → `event-driven`（0.4.0）或 `us-ipo-arbitrage`
- **纯技术面筛选**（趋势 / 突破） → `technical`（0.4.0）

### 产出边界

本 framework 产出"**待研究清单**"，不产出"**买入清单**"。清单中 Top-N 需要进一步：
1. 每只跑 `valuation-relative` / `valuation-dcf` → 估值判断
2. 每只跑 `command-driven-analysis` → 递归下钻关键业务 / 订单 / 催化剂
3. 综合后出决策卡（买 / 持 / 观望）

## 2. 核心方法论：4 步筛选 + 打分

```
输入：赛道 + 池子
  │
  ├─ Step 1  赛道筛（边界定义 + 候选集构建）
  │   └─ 从池子中留下"真正属于赛道的" → 通常剩 30-80 只
  │
  ├─ Step 2  龙头筛（市占率 / Layer 卡位）
  │   └─ 留下 Top 3 + 特定 Layer 高卡位的 → 通常剩 10-20 只
  │
  ├─ Step 3  质量筛（ROE / 现金流 / 商誉 / 治理）
  │   └─ 留下财务健康的 → 通常剩 5-10 只
  │
  ├─ Step 4  成长筛（营收/利润 CAGR + forward 指引）
  │   └─ 留下有未来的 → 通常剩 5-8 只
  │
  ├─ 打分  30/60/100（四维各 25 分）
  │
  └─ 输出  Top-N（默认 3-5 只，每只附下一步研究方向）
```

### 设计原则

1. **每步必须可筛掉 50%+** — 否则该步没有意义
2. **每步阈值量化** — 不是"感觉龙头"，而是"市占率 ≥ 10%"
3. **红线严格** — 任一维度得分 < 10（满分 25）直接淘汰
4. **不做买入决策** — 只给候选清单 + 研究方向

## 3. 决策树

```
命题输入：从 <池子> 中筛选 <赛道> 的 Top-N
  │
  ├─ Step 0: 命题解析
  │   ├─ 赛道边界：什么算"属于这个赛道"
  │   ├─ 池子来源：交易所 / 指数成分股 / 产业链图谱 / ETF 持仓
  │   └─ Top-N：用户要几只（默认 3-5）
  │
  ├─ Step 1: 赛道筛（通过标准：≥ 2 项满足）
  │   ├─ 营收占比 ≥ 30% 来自该赛道
  │   ├─ 核心业务明确定位在该赛道（公司年报自述）
  │   ├─ 产业链上下游紧密关联（供应链 / 客户）
  │   └─ 主题资金流向明确（龙虎榜 / 北向资金）
  │
  ├─ Step 2: 龙头筛（通过标准：≥ 1 项满足）
  │   ├─ 市占率 Top 3（按全球 / 区域 / 细分场景）
  │   ├─ 技术代际领先（如光刻机 / HBM / CUDA）
  │   ├─ 客户 / 生态卡位（如苹果供应链 / NVDA 伙伴）
  │   └─ 独家商业模型（Layer 级卡位，参见 ai-thematic-investing）
  │
  ├─ Step 3: 质量筛（通过标准：4 项全满足）
  │   ├─ ROE 3 年均值 ≥ 行业中位数
  │   ├─ 经营现金流 / 净利 ≥ 80%（非一次性调节）
  │   ├─ 商誉 / 净资产 ≤ 30%（无商誉爆雷隐患）
  │   └─ 无审计师变更 / 无大额关联交易 / 无控股股东质押 > 50%
  │
  ├─ Step 4: 成长筛（通过标准：≥ 2 项满足）
  │   ├─ 过去 3 年营收 CAGR ≥ 15%（成熟业）或 ≥ 40%（新兴业）
  │   ├─ 过去 3 年净利 CAGR ≥ 营收 CAGR（或利润率改善）
  │   ├─ 最新季度 YoY > 过去 3 年 CAGR（加速信号）
  │   └─ 卖方一致预期未来 2 年营收增速 ≥ 20%
  │
  ├─ Step 5: 30/60/100 打分（四维各 25 分）
  │   ├─ 赛道分（主题热度 / 空间 / 可持续）
  │   ├─ 龙头分（市占率 / 护城河 / 稀缺性）
  │   ├─ 质量分（财务健康 / 治理 / 信披）
  │   └─ 成长分（增速 / 加速度 / 一致预期）
  │
  └─ Step 6: 输出 Top-N 清单
      ├─ 按总分降序
      ├─ 每只附"下一步研究方向"（估值 / 产业链 / 催化剂）
      └─ 标注"谁适合做 Core、谁适合做 Satellite"
```

## 4. 量化信号（每步阈值）

### Step 1 赛道筛

| 信号 | 阈值 | 来源 |
|------|------|------|
| 赛道营收占比 | ≥ 30% | 年报 / 季报 |
| 官方定位 | "核心业务 X" | 公司年报 / 投资者关系 |
| 产业链关联 | 核心客户 / 核心供应商在赛道 | S-1 / 招股书 / 公开披露 |
| 主题资金 | 近 3 月主题 ETF 持有 | 交易所数据 |

### Step 2 龙头筛

| 信号 | 阈值 | 来源 |
|------|------|------|
| 市占率 | ≥ 10%（全球）/ ≥ 15%（区域） | 行业报告（Gartner / IDC / Counterpoint） |
| 技术领先 | 有独家专利 / 独家工艺 | 公司披露 + 公开报道 |
| 客户卡位 | Top 3 大客户 名单 | 年报 / 招股书 |
| Layer 卡位 | 位于 AI 8 层产业链的稀缺 Layer | `ai-thematic-investing.md` 参考 |

### Step 3 质量筛（4 项全通过）

| 信号 | 阈值 | 来源 |
|------|------|------|
| ROE 3Y 均值 | ≥ 行业中位数 | 财务数据 |
| 经营 CF / 净利 | ≥ 80% | 现金流量表 |
| 商誉 / 净资产 | ≤ 30% | 资产负债表 |
| 治理红线 | 无审计师变更 / 无大额关联 / 质押 ≤ 50% | 公告 |

### Step 4 成长筛

| 信号 | 阈值（成熟业）| 阈值（新兴业）|
|------|-------------|-------------|
| 过去 3Y 营收 CAGR | ≥ 15% | ≥ 40% |
| 过去 3Y 净利 CAGR | ≥ 营收 CAGR | 不亏损或减亏 |
| 最新季度 YoY | > 过去 3Y CAGR | > 100% |
| 卖方 forward 2Y 增速 | ≥ 20% | ≥ 50% |

## 5. 打分模型：30/60/100

### 5.1 四维度各 25 分

**赛道分（25）**：

| 子项 | 分值 | 评分标准 |
|------|-----|---------|
| 主题热度 | 10 | 主线主题 10 / 细分主题 7 / 无主题 3 |
| 空间 | 8 | TAM > $1T 得 8 / $100B-1T 得 5 / < $100B 得 2 |
| 可持续 | 7 | 5 年以上主线 7 / 2-5 年 4 / < 2 年 1 |

**龙头分（25）**：

| 子项 | 分值 | 评分标准 |
|------|-----|---------|
| 市占率 | 10 | ≥ 30% 得 10 / 10-30% 得 7 / 5-10% 得 4 / < 5% 得 0 |
| 护城河 | 8 | 多重护城河 8 / 单一护城河 5 / 弱护城河 2 |
| 稀缺性 | 7 | 全球独家 7 / 区域独家 5 / 同质竞争 2 |

**质量分（25）**：

| 子项 | 分值 | 评分标准 |
|------|-----|---------|
| 财务健康 | 10 | ROE ≥ 20% 得 10 / 15-20% 得 7 / 10-15% 得 4 / < 10% 得 0 |
| 现金流质量 | 8 | CF/净利 > 100% 得 8 / 80-100% 得 5 / 50-80% 得 2 / < 50% 得 0 |
| 治理 | 7 | 无红线 7 / 有 1 项 4 / 有 2 项 0 |

**成长分（25）**：

| 子项 | 分值 | 评分标准 |
|------|-----|---------|
| 过去增速 | 10 | 营收 CAGR ≥ 30% 得 10 / 15-30% 得 7 / 5-15% 得 4 / < 5% 得 0 |
| 加速度 | 8 | 最新季度 YoY > 过去 CAGR 1.5x 得 8 / 1-1.5x 得 5 / < 1x 得 2 |
| 一致预期 | 7 | forward 增速 > 过去 CAGR 得 7 / 持平 得 4 / 下滑 得 0 |

### 5.2 红线（任一维度 < 10 直接淘汰）

- 赛道分 < 10 → 赛道逻辑不成立，不值得研究
- 龙头分 < 10 → 跟风选手，无核心价值
- 质量分 < 10 → 财务或治理有隐患，不碰
- 成长分 < 10 → 没有未来，不属于"待研究"

### 5.3 总分分档

| 总分 | 档位 | 处置 |
|------|------|------|
| 85-100 | **核心待研究** | 进入 Top-3，必做深度研究 |
| 70-85 | **备选待研究** | 进入 Top-5，可选深度研究 |
| 55-70 | **观察池** | 暂不研究，下季度 refresh |
| < 55 | **淘汰** | 不进入候选 |

## 6. Worked Example：A 股动力电池赛道筛选（2026-04）

### 6.1 命题

- 池子：A 股全部上市公司 + 港股同类
- 赛道：动力电池（新能源车 + 储能）
- Top-N：3 只

### 6.2 Step 1 赛道筛

从 A 股 + 港股约 5000 只中筛选"营收占比 ≥ 30% 来自动力电池"的：

**通过候选**（示例 8 只）：
- CATL（宁德时代 300750）
- 比亚迪（002594，电池业务 + 车双主业）
- 国轩高科（002074）
- 欣旺达（300207）
- 亿纬锂能（300014）
- 孚能科技（688567）
- 鹏辉能源（300438）
- 珠海冠宇（688772）

### 6.3 Step 2 龙头筛

| 候选 | 市占率 | 技术 / 卡位 | 通过？ |
|------|-------|------------|-------|
| CATL | 全球 37% | 麒麟 / 神行 / 钠电多线 | ✓ |
| 比亚迪 | 全球 16%（自供 + 外供） | 刀片电池 + 集成度高 | ✓ |
| 国轩高科 | 中国 5% | 磷酸铁锂工艺 | ✓（下限） |
| 欣旺达 | 消费电池为主，动力电池 < 5% | - | ✗ |
| 亿纬锂能 | 中国 3% | 消费 + 动力 + 储能 | ✓（下限） |
| 孚能科技 | 中国 < 2% | - | ✗ |
| 鹏辉能源 | 中国 < 2% | 储能 | ✗ |
| 珠海冠宇 | 消费为主 | - | ✗ |

通过：CATL / 比亚迪 / 国轩 / 亿纬（4 只）

### 6.4 Step 3 质量筛

| 候选 | ROE 3Y | CF / 净利 | 商誉 / 净资产 | 治理 | 通过？ |
|------|-------|----------|-------------|------|-------|
| CATL | 25% | 130% | < 5% | 无红线 | ✓ |
| 比亚迪 | 20% | 110% | < 10% | 无红线 | ✓ |
| 国轩 | 5% | 60% | 15% | 无红线 | ✗（ROE + CF） |
| 亿纬锂能 | 18% | 95% | 10% | 无红线 | ✓ |

通过：CATL / 比亚迪 / 亿纬（3 只）

### 6.5 Step 4 成长筛

| 候选 | 3Y 营收 CAGR | 净利 CAGR | Q1 YoY | forward 2Y | 通过？ |
|------|------------|----------|--------|-----------|-------|
| CATL | 45% | 50% | 60% | 30% | ✓ |
| 比亚迪 | 65% | 80% | 45% | 25% | ✓ |
| 亿纬锂能 | 55% | 40% | 70% | 35% | ✓ |

通过：CATL / 比亚迪 / 亿纬（3 只全部）

### 6.6 Step 5 30/60/100 打分

| 候选 | 赛道分 | 龙头分 | 质量分 | 成长分 | 总分 | 档位 |
|------|-------|-------|-------|-------|------|------|
| **CATL** | 22 | 25 | 23 | 22 | **92** | 核心 |
| **比亚迪** | 22 | 20 | 21 | 24 | **87** | 核心 |
| **亿纬锂能** | 22 | 15 | 19 | 23 | **79** | 备选 |

**打分说明**：
- CATL 龙头分满分 25：全球 37% 市占率 + 麒麟/钠电多技术路线 + 全球独家龙头地位
- 比亚迪赛道分略低：电池 + 车双主业，电池纯度不如 CATL
- 亿纬锂能龙头分仅 15：市占率偏低，虽然三业务布局但都不是龙头

### 6.7 Step 6 输出 Top-3 清单

| 排名 | 标的 | 总分 | 下一步研究方向 | Core / Satellite |
|------|------|------|---------------|------------------|
| 1 | CATL（300750） | 92 | 估值（forward PE vs 出口倍数）/ 麒麟量产进度 / 钠电商业化 | Core |
| 2 | 比亚迪（002594） | 87 | 电池 vs 车业务分拆估值 / 海外市场进展 | Core |
| 3 | 亿纬锂能（300014） | 79 | 三业务利润贡献拆分 / 储能订单 / 消费电池周期拐点 | Satellite |

## 7. 执行纪律

### 红线（违反 = 重做）

1. **池子无边界**："A 股"太泛，必须缩小到"沪深 300"或"相关指数成分"
2. **赛道筛阈值跳过** → 30% 营收占比是硬门槛
3. **龙头筛只看市值**（不看市占率） → 市值大不等于龙头（如比亚迪电池业务外供 < 自供）
4. **质量筛 4 项放宽到 2 项** → 质量门槛不能降
5. **成长筛用 TTM PE 代替 CAGR** → 方法错
6. **打分主观拍** → 每分都要有依据
7. **总分 < 55 强行进清单** → 降低标准 = 清单失效
8. **不给"下一步研究方向"** → 清单没有执行价值

### 反模式

❌ **"龙头当次选"陷阱**：比亚迪赛道分拉满（当做纯电池股） — 实际电池外供少
❌ **"主题 = 龙头"陷阱**：鹏辉能源占储能主题就进清单 — 市占率太低
❌ **"营收规模 = 龙头"陷阱**：按营收排名而非市占率
❌ **质量门槛滑动**："ROE 只有 5% 但成长快" — 质量红线不能让
❌ **成长用 TTM**：TTM PE 低不代表成长好，CAGR 才说明问题
❌ **打分四舍五入作弊**：赛道 9.5 凑到 10 让它通过红线
❌ **不分 Core / Satellite**：3 只都当核心持仓 → 失去仓位结构

### 核查清单

- [ ] 池子边界明确吗？
- [ ] 4 步筛选每步都做了量化阈值判断吗？
- [ ] 每步筛掉 50%+ 吗？
- [ ] 质量 4 项全过，没有放宽吗？
- [ ] 打分每个子项有数据依据吗？
- [ ] 任一维度 < 10 的淘汰了吗？
- [ ] Top-N 每只都附了研究方向吗？
- [ ] 标注了 Core / Satellite 吗？

## 8. 失效场景

### 8.1 赛道边界模糊

**场景**："AI 股"的边界问题（硬件 / 软件 / 应用都算 AI 股吗？）。

**退路**：
1. 强制限定赛道（如"AI 算力芯片"而非"AI 股"）
2. 或拆成多个子赛道分别筛

### 8.2 池子数据不全

**场景**：研究对象在港美股 + A 股混合池，数据标准不一致。

**退路**：
1. 按单一市场筛选
2. 跨市场对比只做定性（不做打分）

### 8.3 赛道处于初创期

**场景**：赛道太新（如 2026 年的 AI Agent），上市公司少，都没利润。

**退路**：
1. 成长筛阈值放宽到"不亏损或减亏"
2. 或不用本 framework，走 deep-research 做产业链分析

### 8.4 红线冲突

**场景**：某只标的赛道/龙头分高，但质量分 < 10。

**退路**：**不破例**。红线就是红线。标注"观察池，等质量改善"。

### 8.5 主题退潮期

**场景**：曾经主线主题现在已经变为"逆主题"（如 2023 年的元宇宙）。

**退路**：
1. 赛道分降至 < 10 直接淘汰
2. 等新主题起来重新筛

## 9. 相关 template 和 framework 组合

### Template
- 主产出：`templates/stock-screening-card.md`（候选集 → 4 步结果 → 打分 → Top-N）
- 辅助：`templates/alpha-thesis-card.md`（对 Top-3 每只做论点卡）

### Framework 组合

| 命题 | 加载顺序 |
|------|---------|
| "从赛道筛 3 只" | `stock-selection` → 每只 `valuation-relative` / `valuation-dcf` |
| "主题龙头是谁" | `stock-selection` + `ai-thematic-investing`（主题层判断） |
| "选股 + 估值" | `stock-selection` → `valuation-relative` / `valuation-dcf` |
| "选股 + 执行纪律" | `stock-selection` → `alpha-generation`（5 执行差） |

### 与 alpha-generation 的协作

```
stock-selection  → 待研究清单（候选 Top-N）
  ↓
alpha-generation → 决策阶段（5 执行差：精度 / 时点 / 仓位 / 持有 / 顶点）
  ↓
valuation-*      → 目标价 + 止损
  ↓
决策卡 + 仓位建议
```

## 10. Disclaimer

- 本 framework 是筛选流程，不是买入建议
- 打分含主观判断，建议每季度 refresh
- Top-N 清单进入研究阶段，不等于买入
- 数据依赖公开披露，财报质量和一致预期可能过时
- 赛道热度可能反转（主题退潮），要定期重评
- 最终决策需结合个人风险偏好、仓位结构、流动性需求

**清单只是起点，研究才是决策的根据** — 不要跳过估值和深度研究直接买 Top-N。
