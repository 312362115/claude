# 商业模式分析框架

> 调研涉及盈利模式、变现策略、单元经济学等命题时，叠加以下框架。

## 核心框架

### 商业模式画布（9 要素）

- **适用场景**：全景梳理商业模式、对标竞品、模式创新
- **操作步骤**：逐一填写 9 大模块

| 模块 | 核心问题 | 互联网关注点 |
|------|---------|------------|
| 客户细分 | 为谁创造价值？ | 用户 vs 客户分离（用户免费，广告主付费） |
| 价值主张 | 解决什么问题？ | 10x 改进还是渐进优化？ |
| 渠道通路 | 如何触达？ | 自然增长 vs 付费获客 vs 渠道合作 |
| 客户关系 | 如何维系？ | 自助 / 社区 / 专属顾问 |
| 收入来源 | 怎么收钱？ | 订阅/抽成/广告/增值 |
| 核心资源 | 最重要资产？ | 数据、算法、用户关系、品牌 |
| 关键活动 | 必须做好什么？ | 产品开发、用户运营、内容生产 |
| 关键伙伴 | 需要谁合作？ | 云服务商、支付渠道、内容供应商 |
| 成本结构 | 主要成本？ | 固定 vs 边际成本、CAC 占比 |

- **产出格式**：一页商业模式画布图 + 关键要素分析
- **常见误用**：填完就完事不做深度分析；不与竞品对比

### 单元经济学（Unit Economics）

- **适用场景**：盈利可持续性判断、增长质量评估
- **操作步骤**：
  1. 计算核心指标，与基准对比
  2. 按时间序列观察趋势
  3. 找出改善杠杆点

| 指标 | 公式 | 健康基准 | 危险信号 |
|------|------|---------|---------|
| LTV | ARPU × 毛利率 × 客户生命周期 | — | LTV 持续下降 |
| CAC | 销售营销支出 / 新增客户 | — | CAC 持续上升 |
| **LTV:CAC** | LTV / CAC | **>= 3:1** | < 1:1 |
| CAC 回收期 | CAC / 月度毛利 | **<= 12 个月** | > 18 个月 |
| 毛利率 | (收入-COGS)/收入 | SaaS>=70%, 电商>=30% | 低于行业均值 |

- **产出格式**：单元经济学仪表盘 + 行业对比 + 改进路径

### 收入模型类型学

- **适用场景**：变现策略选择、商业模式对标

| 模型 | 代表公司 | 关键指标 |
|------|---------|---------|
| SaaS 订阅 | Salesforce/Notion | MRR/ARR, NDR, Churn |
| 交易抽成 | 美团/Uber/Airbnb | GMV, Take Rate, 频次 |
| 广告模式 | Google/Facebook | DAU, ARPU, Ad Load |
| Freemium | Spotify/Dropbox | 转化率, ARPPU |
| 硬件+服务 | Apple/Peloton | 设备激活量, 服务收入占比 |

### 商业模式可行性判断

```
Step 1: LTV > 3x CAC？ → 否则重新设计收入模型或获客策略
Step 2: 毛利率够高？ → 否则检查成本结构
Step 3: 增长飞轮存在？ → 否则增长将严重依赖营销投入
Step 4: 护城河能建多深？ → 深=值得大投入 / 浅=利基策略
```

## 典型报告骨架

```
1. 执行摘要
2. 商业模式画布（一页视觉总览）
3. 价值主张深入（用户画像 + 痛点 + 验证数据）
4. 收入模型分析（收入来源 + 定价 + 预测）
5. 单元经济学（LTV/CAC/回收期 + 行业对比 + 改进路径）
6. 增长引擎（获客渠道 + 留存曲线 + 飞轮效应）
7. 成本结构（固定/变动 + 规模效应 + 盈亏平衡点）
8. 风险与假设（关键假设 + 敏感性分析）
9. 结论与建议
```

## 数据采集指南

### 核心数据源
- 对标公司财报：毛利率、CAC 基准
- 行业基准：SaaS Capital Survey、KeyBanc SaaS Survey、OpenView SaaS Benchmarks
- 用户研究：Van Westendorp 价格敏感度测试
- 竞品定价：官网、G2 Crowd、Product Hunt

## 常见反模式

- **只看收入不看成本**：GMV 增长但单位经济学恶化
- **忽略 CAC 趋势**：获客成本持续上升是危险信号
- **飞轮幻觉**：声称有飞轮但每个环节没有因果强化关系
- **对标错位**：拿消费品的毛利率基准评估 SaaS 企业
