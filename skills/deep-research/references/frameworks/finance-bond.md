# 固收/债券分析框架

> 调研涉及债券投资分析时使用。

## 核心分析框架

### 利率分析

- **适用场景**：利率债投资、久期管理
- **核心逻辑**：经济基本面 + 货币政策 → 利率走势 → 债券价格

| 指标 | 含义 | 信号 |
|------|------|------|
| 10Y 国债收益率 | 无风险利率锚 | 趋势判断债市方向 |
| 期限利差（10Y-2Y） | 经济预期 | 收窄/倒挂 = 衰退预警 |
| DR007 | 银行间流动性 | 偏离政策利率 = 流动性松紧 |
| 社融/M2 | 信用扩张 | 社融放量 → 利率上行压力 |

### 信用分析（五维度模型）

- **适用场景**：信用债投资、信用风险评估

| 维度 | 关键指标 | 判断标准 |
|------|---------|---------|
| 行业风险 | 景气度、政策支持度 | 产能过剩行业风险高 |
| 企业基本面 | 资产负债率、利息覆盖倍数 | 利息覆盖 < 1.5x 高危 |
| 现金流 | 经营现金流/总债务 | < 10% 警惕偿债能力 |
| 外部支持 | 政府/母公司支持 | 城投 → 地方政府信用 |
| 信用评级 | 主体评级 + 债项评级 | AA+ 以上为投资级 |

### 久期-凸性管理

- **适用场景**：债券组合管理
- **核心概念**：
  - 久期：价格对利率的一阶敏感度。久期 5 年 = 利率↑1%，价格↓约 5%
  - 凸性：二阶敏感度。凸性大的债券"涨多跌少"
- **策略**：预期利率↓ → 拉长久期；预期利率↑ → 缩短久期；不确定 → 选高凸性

### 利率期限结构

- **因子分解**：水平因子（整体平移）、斜率因子（陡峭/平坦化）、曲率因子（中段凸起/凹陷）
- **理论基础**：预期理论、流动性溢价理论、市场分割理论

## 估值方法

| 指标 | 用途 |
|------|------|
| 到期收益率（YTM） | 持有至到期综合收益 |
| 当期收益率 | 当期利息回报 |
| 信用利差 | 信用风险溢价（= 信用债 YTM - 同期限国债 YTM） |
| 久期/凸性 | 利率敏感度 |
| OAS（期权调整利差） | 含权债券真实信用溢价 |

**推理链**：宏观判断(经济+通胀+政策) → 利率方向 → 久期策略 → 信用环境 → 信用下沉/防守 → 流动性 → 杠杆策略 → 组合构建

## 典型研报骨架

```
一、宏观经济与政策环境（GDP/CPI/货币政策/财政/海外）
二、利率走势分析（收益率曲线/关键利率/期限利差/预测）
三、信用市场分析（信用利差/评级迁移/违约跟踪/重点发行人）
四、投资策略建议（久期策略/信用下沉/杠杆策略/骑乘策略）
五、风险提示
```

## 数据来源

| 平台 | 功能 |
|------|------|
| 中国债券信息网（中债） | 中债估值曲线、债券信息（利率债定价基准） |
| 中央结算公司（CCDC） | 债券登记托管数据 |
| 上海清算所 | 银行间市场交易数据 |
| 外汇交易中心 | 银行间债券市场实时行情 |
| Wind/Bloomberg | 全品种债券数据+分析工具 |

## 风险分析要点

| 风险类型 | 评估方法 |
|---------|---------|
| 利率风险 | 久期/凸性分析、VaR |
| 信用风险 | 信用分析五维度模型 |
| 流动性风险 | 换手率、买卖价差 |
| 再投资风险 | 情景分析 |
| 提前偿还风险 | OAS 分析 |
| 通胀风险 | 实际利率/TIPS 利差 |
