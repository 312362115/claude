# 运营职能分析框架

> 调研涉及用户增长、留存优化、精细化运营等命题时，叠加以下框架。

## 核心框架

### AARRR 海盗模型

- **适用场景**：增长策略制定、全链路运营分析、初创期产品
- **操作步骤**：
  1. 搭建全链路数据看板（每阶段核心指标）
  2. 找到转化率最低的环节（瓶颈）
  3. 针对瓶颈设计实验方案
  4. AB 测试验证效果
  5. 沉淀有效策略，进入下一个瓶颈

| 阶段 | 核心指标 | 关键动作 |
|------|---------|---------|
| Acquisition | CAC、渠道转化率 | 渠道优化、投放策略 |
| Activation | 激活率、Aha Moment 到达率 | 新手引导优化 |
| Retention | 次日/7日/30日留存率 | 触达策略、习惯养成 |
| Revenue | ARPU、LTV、付费转化率 | 定价策略、付费引导 |
| Referral | K 因子、邀请转化率 | 裂变机制、口碑运营 |

- **产出格式**：AARRR 漏斗图 + 各环节转化率 + 瓶颈诊断 + 实验计划
- **常见误用**：只看注册量不看激活和留存（虚荣指标）

### RFM 模型（用户分层）

- **适用场景**：用户价值分层、精准营销、流失预警
- **操作步骤**：
  1. 提取用户 R（最近活跃距今天数）、F（消费/活跃次数）、M（消费/贡献金额）
  2. 各维度按分位数分高低两档
  3. 组合为 8 类用户分群，匹配差异化运营策略

| 分群 | R | F | M | 策略 |
|------|---|---|---|------|
| 重要价值客户 | 高 | 高 | 高 | VIP 维护、专属权益 |
| 重要发展客户 | 高 | 低 | 高 | 提升消费频次 |
| 重要保持客户 | 低 | 高 | 高 | 召回激活 |
| 重要挽留客户 | 低 | 低 | 高 | 高优先级召回 |

- **产出格式**：8 类用户分群表 + 各群体量级 + 对应运营策略
- **常见误用**：不同业务类型的 R/F/M 权重应不同

### 用户生命周期模型

- **适用场景**：全生命周期运营策略、LTV 计算
- **操作步骤**：
  1. 定义阶段划分标准：潜在→新增→活跃→成熟→衰退→流失
  2. 每阶段设定核心指标和运营策略
  3. 建立预警机制（活跃度下降趋势 → 自动干预）
- **产出格式**：生命周期阶段定义 + 各阶段指标/策略表 + 预警规则

### 北极星指标 + OSM 模型

- **适用场景**：运营目标拆解、跨团队对齐
- **操作步骤**：
  1. 确定北极星指标（如"周活跃用户总观看时长"）
  2. 拆解关键因子（O = 目标 → S = 策略 → M = 度量指标）
  3. 每个策略有明确度量和负责人

| 业务类型 | 北极星指标 | 拆解方向 |
|---------|-----------|---------|
| SaaS | Weekly Active Teams | 注册 × 激活率 × 周活跃率 |
| 电商 | 月交易额 GMV | 活跃买家 × 频次 × 客单价 |
| 内容平台 | 日均消费时长 | DAU × 打开次数 × 单次时长 |

- **产出格式**：北极星 + OSM 拆解树 + 各指标负责人

### 选择逻辑

| 场景 | 推荐框架 |
|------|---------|
| 增长期全链路分析 | AARRR |
| 用户精细化分层 | RFM |
| 全生命周期运营 | 用户生命周期 |
| 目标拆解与对齐 | 北极星 + OSM |

## 典型报告骨架

```
1. 核心指标概览（北极星表现 + AARRR 各环节 + 关键变动）
2. 用户增长（新增趋势分渠道 + CAC 变化 + 渠道 ROI）
3. 活跃与留存（DAU/MAU + 留存曲线按队列 + RFM 分群分布）
4. 商业化（收入趋势 + ARPU/LTV + 付费转化漏斗）
5. 运营动作复盘（动作清单 + 效果量化 + AB 结论）
6. 诊断与归因（指标异常归因 + 流失原因 + 竞品影响）
7. 下一周期策略（目标 + 优先级清单 + 资源需求）
```

## 数据采集指南

### 内部数据（优先）
- 埋点数据：用户行为分析（神策/GrowingIO/Amplitude）
- AB 实验：策略效果验证（火山引擎 DataTester/Statsig）
- BI 系统：历史数据看板（Metabase/自建 BI）

### 外部数据
- 行业基准：QuestMobile/TrustData
- 竞品活跃：七麦数据/App Annie
- 社交舆情：微博数据中心/抖音指数

## 常见反模式

- **虚荣指标**：只看注册量不看留存
- **归因偏差**：最后触点归因忽略前置渠道贡献
- **增长黑客迷信**：追求技巧忽视产品价值
- **过度补贴**：靠补贴拉来的用户留存极差
- **推送骚扰**：为拉活跃过度推送，导致卸载
