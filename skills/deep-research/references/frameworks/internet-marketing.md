# 市场职能分析框架

> 调研涉及营销策略、品牌建设、市场进入等命题时，叠加以下框架。

## 核心框架

### STP 分析框架

- **适用场景**：新产品市场进入、品牌定位、目标人群选择
- **操作步骤**：
  1. **Segmentation**：按人口统计/心理/行为/地理维度细分市场
  2. **Targeting**：评估每个细分市场吸引力（规模 × 增速 × 竞争 × 自身匹配），选 1-3 个
  3. **Positioning**：为每个目标市场制定差异化定位
  4. 验证定位是否被目标用户感知
- **产出格式**：细分市场地图 + 目标选择矩阵 + 定位声明
- **常见误用**：细分维度不够具体；定位只写内部视角不验证用户感知

### 4P 营销组合

- **适用场景**：营销策略制定、竞品营销分析
- **操作步骤**：
  1. 逐维度分析并制定策略

| 维度 | 传统 | 数字化升级 |
|------|------|-----------|
| Product | 产品特性、包装 | 数字产品体验、内容即产品 |
| Price | 定价策略 | 动态定价、订阅、Freemium |
| Place | 分销渠道 | 电商/社交电商/DTC |
| Promotion | 广告、公关 | 内容营销、KOL、精准投放 |

- **产出格式**：4P 策略表（当前 vs 竞品 vs 建议）
- **常见误用**：只做自身分析不做竞品对比

### PESO 媒体模型

- **适用场景**：传播策略制定、媒体渠道组合优化
- **操作步骤**：
  1. 评估四类媒体当前投入和效果
  2. 找出 ROI 最高和最低的渠道
  3. 优化组合配比

| 类型 | 定义 | 示例 | 核心指标 |
|------|------|------|---------|
| Paid | 付费曝光 | 信息流/SEM/KOL 付费 | CPM、CPC、ROAS |
| Earned | 第三方报道 | 新闻、测评、口碑 | 媒体提及量、SOV |
| Shared | 社交互动 | 微博/抖音/小红书讨论 | 互动率、UGC 量 |
| Owned | 自有渠道 | 官网、公众号、APP | 粉丝量、打开率 |

- **产出格式**：PESO 渠道效果矩阵 + 预算分配建议
- **常见误用**：过度依赖单一渠道（如只做信息流投放）

### 品牌健康度追踪

- **适用场景**：品牌建设效果评估、品牌策略调整
- **操作步骤**：
  1. 定期（季度/半年）追踪六维度：知名度、认知度、好感度、考虑度、忠诚度、SOV
  2. 与竞品对比，识别差距
  3. 基于数据调整品牌策略

| 维度 | 测量方式 |
|------|---------|
| 知名度 | 调研问卷（辅助/非辅助回忆） |
| 好感度 | NPS、品牌好感度评分 |
| 考虑度 | 购买意向调研 |
| SOV | 社交媒体 + 新闻监测 |

- **产出格式**：品牌健康度仪表盘 + 趋势 + 竞品对比

### 选择逻辑

| 场景 | 推荐框架 |
|------|---------|
| 市场进入策略 | STP |
| 营销方案制定 | 4P |
| 传播渠道规划 | PESO |
| 品牌效果评估 | 品牌健康度 |
| 营销活动分析 | 营销漏斗 + AIDA（Attention→Interest→Desire→Action） |

## 典型报告骨架

```
1. 执行摘要（核心发现 + 战略建议）
2. 市场概况（规模 + 细分格局 + 趋势）
3. 竞争分析（份额 + SWOT + 4P 对比 + 差异化机会）
4. 目标用户（STP 结果 + 决策旅程 + 关键触点）
5. 品牌健康度（六维度得分 + SOV + 竞品对比）
6. 营销效果（PESO 各渠道 + 漏斗转化 + ROI）
7. 策略建议（定位 + 渠道优化 + 内容方向 + 预算分配）
```

## 数据采集指南

### 内部数据（优先）
- 广告投放数据：投放效果（巨量引擎/腾讯广告/Google Ads）
- 品牌调研：健康度追踪（问卷星/Panel 调研）
- 社交媒体监测：品牌舆情、SOV（新榜/蝉妈妈/Brandwatch）

### 外部数据
- 行业报告：市场规模（艾瑞/eMarketer/Statista）
- 竞品投放：广告创意库（AppGrowing/有米有数）
- 搜索数据：品牌热度（百度指数/Google Trends）

## 常见反模式

- **品效割裂**：品牌团队和效果团队各自为政
- **SOV ≠ SOW**：声量占比高不等于市场占比高
- **短期主义**：只追短期 ROI 不做品牌投入
- **调研偏差**：社会期望偏差导致用户说的和做的不一样
- **数据孤岛**：各渠道数据无法打通，无法全链路归因
