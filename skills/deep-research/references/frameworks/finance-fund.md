# 基金分析框架

> 调研涉及基金投资分析时使用。

## 核心分析框架

### Brinson 业绩归因

- **适用场景**：评价基金经理投资能力
- **核心原理**：将超额收益（相对基准）分解为：
  - **配置效应**：是否在表现好的行业上超配？
  - **选择效应**：在每个行业内是否选到了更好的标的？
  - **交叉效应**：配置与选择的交叉影响
- **解读**：配置效应高 → 自上而下能力强；选择效应高 → 自下而上选股能力强

### 因子归因

- **适用场景**：量化基金、风格分析
- **核心模型**：
  - Fama-French 三因子：市场(MKT) + 规模(SMB) + 价值(HML)
  - Carhart 四因子：三因子 + 动量(MOM)
  - Barra 多因子：10+ 风格因子
- **解读**：alpha > 0 = 有真正选股能力；beta 暴露 = 风格偏好

### 晨星风格箱

- **适用场景**：基金分类、风格漂移监测

|  | 价值 | 混合 | 成长 |
|------|------|------|------|
| **大盘** | 大盘价值 | 大盘混合 | 大盘成长 |
| **中盘** | 中盘价值 | 中盘混合 | 中盘成长 |
| **小盘** | 小盘价值 | 小盘混合 | 小盘成长 |

- **判断**：持仓加权 PE/PB → 价值-成长维度；持仓加权市值 → 大-小盘维度
- **风格漂移**：季度间位置变化 > 2 格需关注

### 择时能力分析

- **T-M 模型**：gamma > 0 且显著 → 有择时能力
- **H-M 模型**：区分牛熊市 beta
- **CL 模型**：分别估计上涨/下跌市场的 beta

### 风险调整收益指标

| 指标 | 含义 | 判断标准 |
|------|------|---------|
| Sharpe Ratio | 每单位总风险的超额收益 | > 1 优秀，> 2 卓越 |
| Information Ratio | 每单位跟踪误差的超额收益 | > 0.5 良好，> 1 优秀 |
| Sortino Ratio | 每单位下行风险的超额收益 | 比 Sharpe 更关注下行 |
| Calmar Ratio | 年化收益 / 最大回撤 | > 1 良好 |
| 最大回撤 | 峰谷最大跌幅 | 偏股型 < 30% 可控 |

## 评价体系

基金无传统"估值"，核心是多维度评价：

| 维度 | 方法 | 关键指标 |
|------|------|---------|
| 收益能力 | 绝对收益、同类排名 | 年化收益率、百分位 |
| 风险控制 | 波动率、回撤 | 最大回撤、下行偏差 |
| 风险收益比 | 风险调整指标 | Sharpe、Calmar、IR |
| 选股能力 | Brinson 归因 | 选择效应 > 0 |
| 择时能力 | T-M/H-M 模型 | gamma 显著为正 |
| 风格稳定性 | 风格箱漂移 | 季度偏移 < 2 格 |
| 规模影响 | 规模-业绩关系 | 偏股型 > 100 亿警惕稀释 |

**推理链**：确定投资目标(收益+风险) → 资产配置方案 → 基金筛选(定量+定性) → 持续监控(风格漂移+业绩持续性+规模) → 动态调整

## 典型研报骨架

```
一、基金概况（类型/规模/费率/经理/策略/基准）
二、业绩分析（绝对收益/相对超额/同类排名/风险调整指标）
三、持仓分析（行业配置/重仓股/集中度/换手率）
四、风格分析（风格箱定位/因子暴露/风格漂移）
五、业绩归因（Brinson配置vs选股/个股贡献/择时能力）
六、基金经理评价（经验/历史业绩/管理规模/风格一致性）
七、投资建议（适合投资者/配置建议/风险提示）
```

### 晨星评级
基于风险调整后收益（MRAR），同类排名：前 10% = 5 星，后 10% = 1 星，每季度更新

## 风险分析要点

| 风险类型 | 评估方法 |
|---------|---------|
| 市场风险 | Beta、最大回撤 |
| 风格风险 | 因子暴露分析 |
| 流动性风险 | 机构持有比例、规模变化 |
| 经理风险 | 基金经理任期、变更公告 |
| 规模风险 | 规模增长率、换手率变化 |
| 费率风险 | 总费率对比 |
