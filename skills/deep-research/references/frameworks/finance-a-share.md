# A 股分析框架

> 调研涉及 A 股（中国大陆）投资分析时使用。

## 核心分析框架

### 自上而下宏观驱动

- **适用场景**：策略研究、行业配置、大类资产择时
- **操作步骤**：
  1. 划分宏观经济情景（增长 × 通胀 × 流动性 = 8 种组合）
  2. 映射每种情景下历史表现最好的行业板块
  3. 根据当前情景做行业配置权重调整
  4. 在优选行业内选择个股
- **核心指标**：GDP 增速、PMI、M2 增速、社融规模、DR007、两融余额、北向资金
- **A 股特殊性**：政策驱动是最强 alpha 来源。重大会议（两会/中央经济工作会议/政治局会议）措辞变化直接影响市场预期

### 景气度投资

- **适用场景**：行业轮动、成长股投资
- **核心逻辑**：行业 ROE 趋势 → 景气周期位置 → 股价超额收益
- **关键指标**：ROE(TTM) 连续 2 季度环比上升 = 景气上行；营收增速加速；毛利率扩张；资本开支/折旧 > 1.5 为扩张期
- **机构主流范式**：选择当年或预期未来业绩增速最靠前的行业，结合 PEG

### 行业轮动

- **适用场景**：ETF 配置、行业主题投资
- **四种思路**：宏观映射（美林时钟）、中观景气（利润增速排名）、微观切割（产业链利润传导）、量价技术（行业动量/反转）

### 主题/事件驱动

- **适用场景**：政策主题投资（如"新质生产力"、"国产替代"）
- **核心要素**：政策催化剂识别 → 受益标的筛选 → 主题生命周期判断（预期 → 验证 → 兑现 → 退潮）

## 估值方法

| 估值方法 | 适用行业 | 不适用场景 |
|---------|---------|-----------|
| PE | 消费、医药、TMT | 周期股亏损年份 |
| PB | 银行、保险、周期股 | 轻资产公司 |
| PEG | 高成长（TMT/生物医药） | 低增长/负增长 |
| EV/EBITDA | 电信、公用事业 | 资本密集度低的行业 |
| DCF | 现金流稳定企业 | 早期/周期性企业 |
| NAV | 房地产 | 非地产 |

**A 股特殊估值考量**：壳价值（随注册制推进弱化）、国企改革溢价、政策主题溢价

**推理链**：宏观环境 → 行业景气度排序 → 公司基本面筛选 → 估值合理性检验 → 催化剂识别 → 风险收益比 → 仓位管理

## 典型研报骨架

```
封面：评级 + 目标价
核心摘要（1页）：评级、3-5条核心逻辑、盈利预测表
一、公司概况（业务构成/股权/发展阶段）
二、行业分析（规模/竞争格局/驱动因素）
三、核心竞争力（技术壁垒/客户粘性/成本优势）
四、业绩驱动与盈利预测（收入拆分/利润率/三表预测）
五、估值与投资建议（估值方法/可比对标/目标价）
六、风险提示（量化影响）
附录：财务报表预测明细
```

## 数据来源

### 核心平台
- **Wind（万得）**：机构标配，数据最全（年 2.7-4 万）
- **Choice（东财）**：高性价比，95% 功能覆盖（年 0.5-1 万）
- **东方财富网/雪球**：免费，基础行情+社区

### 另类数据
- 卫星图像（港口/停车场密度）、电商销量数据、社交媒体情绪、产业链草根调研

## 风险分析要点

| 风险类型 | 评估方法 |
|---------|---------|
| 政策风险（如教培双减） | 政策信号跟踪、政策敏感度分析 |
| 退市风险 | ST/*ST 标记、财务连续亏损预警 |
| 大股东风险 | 质押比例 > 50% 高危、解禁日历 |
| 地缘风险 | 制裁清单监测、出口管制跟踪 |

**宏观传导**：美联储加息 → 中美利差扩大 → 北向资金流出 → A 股承压；国内降准降息 → 流动性宽松 → 利好成长股/小盘股
