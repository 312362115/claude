# Deep Research 报告支持 HTML 格式输出

## 背景

当前调研报告只支持 Markdown 格式输出。MD 在样式表现力上有限（配色、布局、排版），且图表需要导出为 PNG 再嵌入，流程割裂。

## 需求

增加可选的 HTML 格式输出，与现有 MD 格式并存，用户可选择：

1. **富样式渲染**：HTML 格式下可自由控制配色、字体、布局、间距等视觉效果，报告更专业美观
2. **图表内嵌**：图表直接以 HTML/SVG/JS 内嵌到报告中，无需导出 PNG 再插入，交互体验更好（如 hover 提示、缩放等）
3. **自包含文件**：单个 HTML 文件包含所有样式和图表，可直接浏览器打开，方便分享

## 关键点

- 复用 diagram skill 现有的 HTML/SVG 图表能力，图表直接嵌入而非走 PNG 导出流程
- MD 格式保持不变，HTML 为可选增强
- 考虑报告模板的 HTML 版本设计（目录导航、章节锚点、响应式布局等）
