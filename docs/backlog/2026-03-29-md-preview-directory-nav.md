# MD Preview 增加目录导航能力

## 需求描述

当前 `preview-md` skill 只支持单个文件预览，需要增加**目录级浏览**能力：

1. **目录侧边栏**：展示当前文件所在目录的所有 `.md` 文件列表
2. **点击切换**：点击侧边栏中的文件名即可预览该文件，无需重新调用命令
3. **当前文件高亮**：侧边栏中当前正在预览的文件有高亮标识

## 演进方向

向 **MD 文档管理系统** 方向发展，后续可考虑：
- 多级目录树导航
- 文件搜索/过滤
- 文件间链接跳转（`[xxx](./other.md)` 点击直接预览）
- 最近访问记录
