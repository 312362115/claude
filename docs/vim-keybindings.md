## 三、Vim 入门路线图

### 第一周：活下来（只需记住 10 个操作）

别想一次学完，先用这些替代鼠标操作：

```
i          → 进入编辑（在光标前插入）
kj 或 Esc  → 退回普通模式
h j k l    → 左 下 上 右（不用方向键也行，但 hjkl 更快）
,w         → 保存
,q         → 退出
Ctrl+p     → 打开文件
Ctrl+b     → 开关文件树
u          → 撤销
/关键词     → 搜索
```

> 要点：随时按 `kj` 回到普通模式，这是你的"安全状态"。
> 不确定自己在什么模式？看左下角状态栏。

### 第二周：提速（学会"动词+名词"思维）

Vim 的操作像造句：**动词 + 范围**

| 动词 | 含义 |
|------|------|
| `d` | 删除 |
| `c` | 修改（删除并进入插入模式） |
| `y` | 复制 |
| `v` | 选中 |

| 范围 | 含义 |
|------|------|
| `w` | 一个单词 |
| `iw` | 整个单词（光标在中间也行） |
| `i"` | 双引号内的内容 |
| `i(` | 括号内的内容 |
| `ip` | 整个段落 |
| `$` | 到行尾 |
| `G` | 到文件末尾 |

**组合示例：**

```
dw     → 删除一个单词
diw    → 删除光标所在的整个单词
ci"    → 修改双引号里的内容（超常用）
yi(    → 复制括号里的内容
vip    → 选中整个段落
d$     → 删除到行尾
dG     → 删除到文件末尾
```

> 这个组合逻辑是 Vim 的精髓。学会后操作速度会指数级提升。

### 第三周：用上 IDE 功能

开始用 coc.nvim 的代码智能：

```
gd         → 跳转到定义（看源码必备）
gr         → 查找谁在用这个函数
K          → 看函数文档
F2         → 重命名变量（批量改名）
,fm        → 格式化代码
Tab        → 补全
Ctrl+/     → 注释切换
```

### 第四周：工作流完善

```
,rg        → 全项目搜内容（找代码必备）
,f         → 文件树中定位当前文件
,v         → 左右分屏对比
Ctrl+h/l   → 左右窗口切换
]c / [c    → 跳转 Git 修改
```

## 四、效率技巧与最佳实践

### 移动

| 技巧 | 说明 |
|------|------|
| `f{字符}` | 跳到当前行的某个字符上（`;` 重复） |
| `%` | 跳到匹配的括号 |
| `gg` / `G` | 文件头 / 文件尾 |
| `{数字}G` 或 `:{数字}` | 跳到指定行 |
| `Ctrl+o` / `Ctrl+i` | 跳转历史：回退 / 前进 |
| `*` | 搜索光标下的词（快速看哪里用了） |
| `H` / `L` | 行首 / 行尾（已映射） |

### 编辑

| 技巧 | 说明 |
|------|------|
| `ciw` | 改掉光标所在的词（最常用的编辑操作） |
| `ci"` / `ci'` / `ci(` | 改掉引号/括号里的内容 |
| `.` | 重复上一次操作（超级强大） |
| `A` | 在行尾插入（比 `$a` 快） |
| `I` | 在行首插入（比 `^i` 快） |
| `o` / `O` | 在下方/上方新开一行 |
| `xp` | 交换两个字符（快速修正 typo） |
| `ddp` | 交换两行 |

### surround 插件（处理包裹符号）

| 操作 | 说明 | 示例 |
|------|------|------|
| `cs"'` | 把双引号改成单引号 | `"hello"` → `'hello'` |
| `ds"` | 删除双引号 | `"hello"` → `hello` |
| `ysiw"` | 给单词加双引号 | `hello` → `"hello"` |
| `ysiw)` | 给单词加括号 | `hello` → `(hello)` |
| `S"` (可视模式) | 给选中内容加双引号 | |

### 搜索替换

```vim
/keyword              " 搜索
:%s/old/new/g         " 全文替换
:%s/old/new/gc        " 全文替换（逐个确认）
:5,20s/old/new/g      " 只在第5-20行替换
```

### 多文件操作

```
Ctrl+p → 输入文件名 → Enter      " 打开文件
,b → 选择 Buffer                  " 切换已打开的文件
,v → 左右分屏                     " 对比两个文件
Ctrl+h / Ctrl+l                   " 在分屏间跳转
```

### 实用命令

```vim
:w                 " 保存（或用 ,w）
:q                 " 退出（或用 ,q）
:wq                " 保存并退出
:qa!               " 强制退出所有（不保存）
:e filename        " 打开文件
:vs filename       " 垂直分屏打开
:term              " 打开内置终端
```

## 六、进阶建议

1. **不要死记硬背** — 每天只学一个新操作，刻意使用直到形成肌肉记忆
2. **`.` 命令是最强的** — 设计你的操作使其可被 `.` 重复（比如 `ciw新词Esc`，然后 `n.n.n.` 批量替换）
3. **文本对象是核心** — 把 `i` (inner) 和 `a` (around) 系列学熟：`iw`, `i"`, `i(`, `ip`, `it`（标签）
4. **用 `vimtutor`** — 终端输入 `vimtutor` 有官方交互教程，30 分钟走一遍
5. **遇到重复操作就想有没有更快的方式** — 有重复就有优化空间
6. **别急着加插件** — 先把内置功能吃透，当前配置已经够用了
