## 摘要

本报告对金融投资领域六大方向进行系统性调研，梳理每个领域的主流分析框架、典型研报结构、数据来源与工具链、估值方法论、风险评估体系和业界最佳实践。核心发现：(1) A 股分析高度依赖政策与资金面，"自上而下"框架是机构主流范式；(2) 美股以 DCF 和可比公司分析为核心，卖方研报体系最为成熟；(3) 港股兼具 A 股政策影响和海外流动性特征，AH 溢价是独特分析维度；(4) 加密货币已形成链上分析、叙事周期、代币经济学三大独立框架；(5) 固收分析以利率期限结构和信用分析为双主线；(6) 基金分析以业绩归因（Brinson 模型）和风格分析为核心方法论。

# 第一部分：A 股（中国大陆）

## 1. 核心分析框架

### 框架一：自上而下宏观驱动框架

**适用场景**：策略研究、行业配置、大类资产择时

**核心逻辑**：宏观经济情景 --> 流动性环境 --> 行业配置 --> 个股选择

**核心指标**：

| 维度 | 关键指标 | 判断标准 |
|------|---------|---------|
| 经济增长 | GDP 增速、PMI、工业增加值 | PMI > 50 为扩张，连续 3 月趋势判断拐点 |
| 流动性 | M2 增速、社融规模、DR007 | M2 - 名义 GDP = 剩余流动性，正值利好股市 |
| 政策面 | 国常会/政治局会议定调、产业政策 | 政策转向信号（如"稳增长"vs"防风险"） |
| 市场情绪 | 两融余额、北向资金、新基金发行 | 两融占流通市值 > 2.5% 为过热信号 |

**使用步骤**：
1. 划分宏观经济情景（增长 x 通胀 x 流动性 = 8 种组合）
2. 映射每种情景下历史上表现最好的行业板块
3. 根据当前情景做行业配置权重调整
4. 在优选行业内选择个股

**A 股特殊性**：政策驱动是 A 股最强的 alpha 来源之一。重大会议（两会、中央经济工作会议、政治局会议）的措辞变化直接影响市场预期。机构投资者通常在会议前后密集发布策略报告。

### 框架二：自下而上景气度投资框架

**适用场景**：行业轮动、成长股投资

**核心逻辑**：行业 ROE 趋势 --> 景气周期位置 --> 股价超额收益

**核心指标**：

| 指标 | 含义 | 判断标准 |
|------|------|---------|
| ROE（TTM） | 净资产收益率 | ROE 连续 2 个季度环比上升 = 景气上行 |
| 营收增速 | 收入端景气 | 季度同比增速加速为积极信号 |
| 毛利率变化 | 盈利质量 | 毛利率扩张 = 量价齐升或成本下降 |
| 资本开支/折旧 | 产能周期 | > 1.5 为扩张期，< 1 为收缩期 |

**机构 vs 个人区别**：
- **机构**（公募/私募）：以景气度投资为主流范式，选择当年或预期未来业绩增速最靠前的行业，结合估值性价比（PEG）[^1]
- **个人投资者**：更多依赖技术分析和消息面，缺乏系统的景气度跟踪能力

### 框架三：行业轮动框架

**适用场景**：ETF 配置、行业主题投资

**四种研究思路**[^2]：
1. **宏观映射**：美林时钟（经济周期四象限 --> 对应行业）
2. **中观景气**：行业利润增速排名动态调整
3. **微观切割**：产业链上下游利润传导分析
4. **量价技术**：行业指数动量/反转策略

### 框架四：主题/事件驱动框架

**适用场景**：政策主题投资（如"新质生产力"、"国产替代"）

**核心要素**：
- 政策催化剂识别（文件级别、执行力度、资金配套）
- 受益标的筛选（直接受益 vs 间接受益）
- 主题生命周期判断（预期 --> 验证 --> 兑现 --> 退潮）

### 框架五：量化因子框架

**适用场景**：量化选股、对冲基金

**A 股有效因子**（经实证检验）：

| 因子 | 年化超额 | 适用市场环境 |
|------|---------|------------|
| 小市值 | 8-12% | 流动性宽松期 |
| 低估值（EP） | 5-8% | 价值回归期 |
| 盈利质量（ROE） | 6-10% | 全市场 |
| 动量（近 6 月） | 4-7% | 趋势市 |
| 反转（近 1 月） | 3-5% | 震荡市 |

### 不同类型研报对比

| 类型 | 篇幅 | 核心内容 | 时效性 |
|------|------|---------|--------|
| **深度报告** | 30-60 页 | 全面分析，首次覆盖或重大变化 | 长期有效 |
| **点评报告** | 3-8 页 | 财报/事件快评，调整盈利预测 | 1-2 周 |
| **行业报告** | 20-40 页 | 行业格局、趋势、选股逻辑 | 季度级 |
| **策略报告** | 10-30 页 | 市场观点、行业配置、风险提示 | 周/月度 |

### 好研报 vs 差研报

| 维度 | 好研报 | 差研报 |
|------|--------|--------|
| 逻辑 | 有清晰的因果推理链 | 堆砌数据无结论 |
| 预测 | 盈利预测有详细假设拆分 | 拍脑袋给数字 |
| 差异化 | 有独立观点和边际信息 | 人云亦云的共识观点 |
| 风险 | 风险提示具体且量化 | 模板化的万能风险提示 |
| 时效 | 抓住关键时间窗口发布 | 滞后于市场反应 |

## 4. 估值与分析方法

### A 股核心估值方法与行业适用性

| 估值方法 | 适用行业 | 不适用场景 | 核心公式 |
|---------|---------|-----------|---------|
| **PE（市盈率）** | 消费、医药、TMT | 周期股亏损年份 | 股价 / EPS |
| **PB（市净率）** | 银行、保险、周期股 | 轻资产公司 | 股价 / 每股净资产 |
| **PS（市销率）** | 亏损期科技公司、零售 | 利润率差异大的行业 | 市值 / 营收 |
| **EV/EBITDA** | 电信、公用事业、高杠杆 | 资本密集度低的行业 | 企业价值 / EBITDA |
| **PEG** | 高成长行业（TMT/生物医药） | 低增长/负增长 | PE / 盈利增速 |
| **DCF** | 现金流稳定企业 | 早期/周期性企业 | 自由现金流折现 |
| **DDM** | 高股息蓝筹 | 不分红公司 | 股利折现 |
| **NAV** | 房地产 | 非地产 | 重估净资产 |
| **SOTP（分部估值）** | 多元化集团 | 单一业务 | 各业务分别估值加总 |

### A 股特殊估值考量

- **壳价值**：A 股 IPO 审核制下，小市值公司隐含壳资源价值（随注册制推进正在弱化）
- **国企改革溢价**：国企混改、员工持股带来的估值提升
- **政策主题溢价**：政策风口行业短期估值可大幅偏离基本面

### 从分析到投资结论的推理链

```
宏观环境判断 --> 行业景气度排序 --> 公司基本面筛选 --> 估值合理性检验
--> 催化剂识别 --> 风险收益比评估 --> 买入/卖出决策 --> 仓位管理
```

## 5. 风险分析方法

### A 股特有风险类型

| 风险类型 | 描述 | 评估方法 |
|---------|------|---------|
| **政策风险** | 产业政策突变（如教培双减） | 政策信号跟踪、政策敏感度分析 |
| **退市风险** | 注册制下退市加速 | ST/*ST 标记、财务连续亏损预警 |
| **大股东风险** | 质押爆仓、违规减持 | 质押比例（> 50% 高危）、解禁日历 |
| **地缘风险** | 中美关系对科技/军工的影响 | 制裁清单监测、出口管制跟踪 |
| **流动性风险** | 限售股解禁、IPO 节奏 | 解禁规模日历、IPO 审批节奏 |

### 宏观风险传导路径

```
美联储加息 --> 中美利差扩大 --> 北向资金流出 --> A 股承压
         \--> 人民币贬值 --> 出口企业受益 / 进口依赖型企业受损

国内降准降息 --> 流动性宽松 --> 资金面改善 --> 利好成长股/小盘股
           \--> 利率下行 --> 利好债市 / 利空银行净息差
```

### 研报中风险提示的呈现

通常放在报告末尾，标准格式为宏观经济增速不及预期、行业竞争加剧、原材料价格大幅波动、新产品/新业务拓展不及预期、政策变化风险等。**好的风险提示**应量化影响：如"若原材料价格上涨 10%，预计影响毛利率 2 个百分点，对应净利润下降约 15%"。

# 第二部分：美股

## 1. 核心分析框架

### 框架一：基本面分析（Fundamental Analysis）

**适用场景**：中长期投资、价值投资

**核心指标**：

| 维度 | 关键指标 | 判断标准 |
|------|---------|---------|
| 盈利能力 | EPS 增速、ROE、ROIC | ROIC > WACC 为价值创造 |
| 现金流 | FCF Yield、OCF/Net Income | FCF Yield > 5% 为高自由现金流 |
| 资产负债 | Debt/Equity、Interest Coverage | Interest Coverage < 3x 需警惕 |
| 增长 | Revenue CAGR、TAM 渗透率 | TAM 渗透率 < 10% 为早期高增长 |

### 框架二：DCF 内在价值框架

**适用场景**：所有有正现金流的成熟企业

**使用步骤**[^4]：
1. 预测未来 5-10 年的自由现金流（FCFF 或 FCFE）
2. 确定折现率（WACC，通常 8-12%）
3. 计算终值（Terminal Value，永续增长法或退出倍数法）
4. 折现求和得到企业价值
5. 减去净债务得到股权价值，除以总股数得到每股内在价值

**关键假设敏感度**：终值通常占 DCF 总价值的 60-80%，因此永续增长率（g）和 WACC 的微小变化会显著影响结果。

### 框架三：可比公司分析（Comps）

**适用场景**：快速估值、IPO 定价

**核心步骤**：
1. 选取 5-10 家可比公司（同行业、相近规模、相似增速）
2. 计算关键倍数：EV/EBITDA、P/E、P/S、EV/Revenue
3. 取中位数或均值作为基准倍数
4. 将基准倍数应用于目标公司的对应指标

### 框架四：GARP（Growth at a Reasonable Price）

**适用场景**：成长股投资

**核心指标**：PEG Ratio = P/E / EPS Growth Rate
- PEG < 1：被低估
- PEG = 1-1.5：合理
- PEG > 2：可能过高

### 框架五：技术分析与量化策略

**适用场景**：短线交易、对冲基金

**常用指标**：RSI、MACD、布林带、成交量分析、期权 Greeks

**机构 vs 个人**：
- **华尔街卖方**：以 DCF + Comps 为标配，强调 12 个月目标价
- **买方（对冲基金）**：多因子模型 + 另类数据 + 事件驱动
- **个人投资者**：更多使用 Finviz 等免费筛选工具 + 简化估值指标

## 2. 典型研报结构

### 华尔街卖方研报标准骨架[^5]

```
Cover Page: Rating (Buy/Hold/Sell) + Price Target + Key Metrics
Executive Summary (1 page):
  - Investment Rating & Price Target
  - Key Thesis Points (3-5 bullets)
  - Financial Snapshot Table

I. Company Overview
  1.1 Business Description & Revenue Segments
  1.2 Management Team & Corporate Governance
  1.3 Industry Positioning

II. Industry Analysis
  2.1 Total Addressable Market (TAM/SAM/SOM)
  2.2 Competitive Landscape & Market Share
  2.3 Industry Trends & Secular Drivers

III. Investment Thesis
  3.1 Bull Case / Base Case / Bear Case
  3.2 Key Catalysts (upcoming earnings, product launches, M&A)
  3.3 Competitive Moat Analysis

IV. Financial Analysis
  4.1 Revenue Build (segment-by-segment)
  4.2 Margin Analysis (gross/operating/net)
  4.3 Balance Sheet & Cash Flow Analysis
  4.4 Three-Statement Model (IS/BS/CF)

V. Valuation
  5.1 DCF Analysis (with sensitivity table)
  5.2 Comparable Company Analysis
  5.3 Precedent Transactions (if applicable)
  5.4 Football Field / Valuation Range Summary

VI. Risks
  - Macro/industry/company-specific risks
  - Bear case scenario quantification

Appendix: Detailed Financial Projections
Disclosures & Disclaimers
```

### 美股研报特色

- **Football Field Chart**：将多种估值方法的结果放在同一张横向柱状图中，展示估值区间
- **Bull/Base/Bear 三情景**：分别给出乐观/基准/悲观情景下的目标价和概率加权
- **Sensitivity Table**：DCF 中 WACC 和永续增长率的敏感度矩阵

## 4. 估值与分析方法

### 美股各行业主流估值方法[^7]

| 行业 | 首选估值方法 | 辅助方法 | 原因 |
|------|------------|---------|------|
| 科技（SaaS） | EV/Revenue、Rule of 40 | DCF | 高增长期利润不稳定 |
| 消费品 | P/E、EV/EBITDA | DCF | 盈利稳定可预测 |
| 金融 | P/B、P/E | DDM | 杠杆经营，净资产是核心 |
| 医药/生物科技 | Pipeline DCF（rNPV） | EV/Revenue | 管线项目概率加权 |
| 能源 | EV/EBITDA、NAV | DCF | 重资产，折旧影响大 |
| REIT | P/AFFO、Cap Rate | NAV | FFO/AFFO 替代净利润 |
| 电信 | EV/EBITDA | DCF | 高折旧，EBITDA 更能反映经营 |
| 工业 | P/E、EV/EBITDA | DCF | 周期性需关注正常化盈利 |

研究发现，P/E 在 25 个行业组中的 20 个为首选估值模型，EV/EBITDA 在电信、能源和材料行业更受青睐，P/AFFO 在 REITs 行业最常使用[^7]。

### 推理链：从分析到决策

```
行业趋势（TAM 增长） --> 公司护城河（竞争优势可持续性）
--> 财务模型（收入/利润预测） --> 估值（DCF + Comps 交叉验证）
--> 催化剂（Earnings/Product Launch） --> 风险收益比（Upside/Downside）
--> 仓位决策
```

# 第三部分：港股

## 1. 核心分析框架

### 框架一：AH 溢价分析框架

**适用场景**：两地上市公司的相对价值判断

**核心指标**：
- **恒生 AH 股溢价指数**：2025 年从 140+ 降至 126 点位，创 5 年新低[^8]
- 指数 = 100 表示 A/H 股等价；> 100 表示 A 股溢价
- 溢价主要来源：流动性差异、交易成本差异（港股通 20% 红利税）、投资者结构差异

**使用逻辑**：
- 溢价率过高时（> 150），H 股具有更好的性价比
- 溢价率收敛时，关注收敛的驱动因素（政策、资金流向、估值修复）

### 框架二：流动性双驱动框架

**核心逻辑**：港股同时受国内基本面和海外流动性双重驱动

| 驱动因素 | 影响路径 | 关键指标 |
|---------|---------|---------|
| 国内经济 | 企业盈利 --> 港股 EPS | 中国 PMI、社零、出口 |
| 美国利率 | 贴现率 + 资金流向 | 10Y UST、联邦基金利率 |
| 南向资金 | 增量资金推动估值 | 港股通净买入额 |
| 全球风险偏好 | 新兴市场配置 | VIX、EM Fund Flow |

### 框架三：港股市值分层分析

| 层级 | 特征 | 分析重点 |
|------|------|---------|
| 大型蓝筹（恒指成分） | 流动性好，机构主导 | 基本面+估值 |
| 中型股 | 弹性大，研究覆盖不足 | 景气度+催化剂 |
| 小型股/细价股 | 流动性差，"老千股"风险 | 流动性筛选+治理结构 |

### 港股特殊风险：老千股识别

**识别特征**[^9]：
- 长期不盈利或微利，极少分红
- 频繁合股后再配股/供股（稀释小股东权益）
- 频繁更换公司名称和股票代码
- 大股东通过关联交易转移资产

**防范方法**：查看近 5 年损益表 --> 检查股本变化历史 --> 查分红记录 --> 查大股东持股变化

# 第四部分：加密货币

## 1. 核心分析框架

### 框架一：链上分析框架（On-Chain Analysis）

**适用场景**：BTC/ETH 等主链资产的市场周期判断

**核心指标**[^10]：

| 指标 | 含义 | 信号判断 |
|------|------|---------|
| **MVRV Ratio** | 市值 / 已实现市值 | > 3.5 过热（卖出区间），< 1 极度低估（买入区间） |
| **MVRV Z-Score** | MVRV 的标准化偏离 | > 7 泡沫顶部，< 0 底部区间 |
| **NVT Ratio** | 网络价值 / 链上交易量 | > 100 高估（类似 P/E 过高），< 50 低估 |
| **SOPR** | 已花费输出利润率 | > 1 持有者整体盈利，< 1 整体亏损 |
| **NUPL** | 净未实现利润/亏损 | > 0.75 极度贪婪，< 0 投降 |
| **Exchange Balance** | 交易所持币量 | 持续流出 = 长期持有信号 |

### 框架二：叙事周期框架（Narrative Cycle）

**适用场景**：板块轮动、新币/新赛道投资

**叙事周期四阶段**：
1. **创新叙事**：新技术/新概念出现（如 DeFi Summer 2020、AI Agent 2025）
2. **资金涌入**：叙事扩散 --> VC 投资 --> 代币发行 --> 散户 FOMO
3. **泡沫膨胀**：估值脱离基本面，空气币泛滥
4. **叙事破灭**：黑天鹅事件或自然消退 --> 价值回归

**判断工具**：社交媒体热度（Twitter/CT）、Google Trends、VC 投资数据、TVL 变化

### 框架三：代币经济学分析框架（Tokenomics）

**适用场景**：新项目/代币投资评估

| 维度 | 关键问题 | 红旗信号 |
|------|---------|---------|
| 供给机制 | 总量固定还是通胀？解锁计划？ | 大量代币即将解锁（悬崖解锁） |
| 分配结构 | 团队/投资者/社区各占比？ | 团队+VC 占比 > 50% |
| 效用价值 | 代币在生态中的实际用途？ | 纯治理代币无现金流支撑 |
| 价值捕获 | 协议收入如何回流代币持有者？ | 无回购/销毁/分红机制 |
| 流动性 | 交易深度如何？上了哪些交易所？ | 仅在 DEX，深度极浅 |

### 框架四：宏观-加密相关性框架

**核心逻辑**：BTC 与全球流动性高度相关

| 宏观变量 | 对加密市场的影响 |
|---------|----------------|
| 美联储资产负债表规模 | 扩表利好，缩表利空 |

| 实际利率 | 负实际利率推动加密配置 |
| 全球 M2 | M2 扩张 --> 风险偏好上升 |

### 框架五：技术分析

加密市场 24/7 交易，技术分析广泛使用：
- 支撑/阻力位、趋势线
- RSI、MACD、布林带
- 资金费率（Funding Rate）：> 0.1% 表示多头过度杠杆
- 未平仓合约（Open Interest）：暴涨时需防范强平连锁

### S2F 模型现状（2025 年）

Bitcoin S2F 模型由 PlanB 于 2019 年提出，公式为 Price = 0.18 x (S/F)^3.3。2024 年减半后，至 2025 年 10 月 BTC 涨至约 $126,000，较减半时涨约 100%，远低于前几轮减半后 300%+ 的涨幅[^12]。大多数分析师现将 S2F 视为理解稀缺性的概念框架，而非精确预测模型。

# 第五部分：固收/债券

## 1. 核心分析框架

### 框架一：利率分析框架

**适用场景**：利率债投资、久期管理

**核心逻辑**：经济基本面 + 货币政策 --> 利率走势 --> 债券价格

| 指标 | 含义 | 信号 |
|------|------|------|
| 10Y 国债收益率 | 无风险利率锚 | 趋势判断债市方向 |
| 期限利差（10Y-2Y） | 经济预期 | 利差收窄/倒挂 = 衰退预警 |
| DR007 | 银行间市场流动性 | 偏离政策利率 = 流动性松紧 |
| 社融/M2 | 信用扩张 | 社融放量 --> 利率上行压力 |

### 框架二：信用分析框架

**适用场景**：信用债投资、信用风险评估

**分析五维度**[^13]：

| 维度 | 关键指标 | 判断标准 |
|------|---------|---------|
| 行业风险 | 行业景气度、政策支持度 | 产能过剩行业风险高 |
| 企业基本面 | 资产负债率、利息覆盖倍数、EBITDA | 利息覆盖 < 1.5x 为高危 |
| 现金流 | 经营现金流/总债务 | < 10% 需警惕偿债能力 |
| 外部支持 | 政府/母公司支持意愿和能力 | 城投平台 --> 地方政府信用 |
| 信用评级 | 主体评级 + 债项评级 | AA+ 以上为投资级 |

### 框架三：久期-凸性管理框架

**适用场景**：债券组合管理

**核心概念**[^14]：
- **久期（Duration）**：债券价格对利率变动的一阶敏感度。久期 5 年 = 利率上升 1%，价格下降约 5%
- **修正久期**：= 麦考利久期 / (1 + YTM)，更准确衡量价格敏感度
- **凸性（Convexity）**：二阶敏感度。凸性大的债券"涨多跌少"
- **价格变动公式**：delta P/P 约等于 -D x delta y + 0.5 x C x (delta y)^2

**使用策略**：
- 预期利率下行 --> 拉长久期（加大利率敏感度以获取更多资本利得）
- 预期利率上行 --> 缩短久期（减少价格波动损失）
- 利率不确定 --> 选择高凸性债券（同等久期下获得更好的凸性保护）

### 框架四：利率期限结构分析

**核心理论**：
- **预期理论**：长期利率 = 预期未来短期利率的平均值
- **流动性溢价理论**：长端利率包含期限溢价
- **市场分割理论**：不同期限的供需独立定价

**因子分解**：
- **水平因子**（Level）：收益率曲线整体平移
- **斜率因子**（Slope）：曲线陡峭化/平坦化
- **曲率因子**（Curvature）：中段凸起/凹陷

| **到期收益率（YTM）** | 使未来现金流现值 = 当前价格的折现率 | 衡量持有至到期的综合收益 |
| **当期收益率** | 年息票 / 债券市价 | 衡量当期利息回报 |
| **信用利差** | 信用债 YTM - 同期限国债 YTM | 衡量信用风险溢价 |
| **久期** | 现金流现值加权平均到期时间 | 衡量利率敏感度 |
| **凸性** | 价格对利率的二阶导数 | 修正久期估算的偏差 |
| **OAS（期权调整利差）** | 扣除内含期权后的利差 | 含权债券真实信用溢价 |

### 投资决策推理链

```
宏观判断（经济+通胀+政策） --> 利率方向判断 --> 久期策略
信用环境判断（违约率+利差走势） --> 信用下沉 or 防守
流动性判断（资金面松紧） --> 杠杆策略
--> 组合构建（期限+信用+杠杆配置比例）
```

# 第六部分：基金

## 1. 核心分析框架

### 框架一：业绩归因分析（Brinson 模型）

**适用场景**：评价基金经理的投资能力

**核心原理**[^17]：将基金超额收益（相对基准）分解为：
- **资产配置效应（Allocation Effect）**：是否在表现好的行业/资产类别上超配？
- **个股选择效应（Selection Effect）**：在每个行业/类别内是否选到了更好的标的？
- **交叉效应（Interaction Effect）**：配置与选择的交叉影响

**公式**：
```
超额收益 = 配置效应 + 选择效应 + 交叉效应
配置效应 = SUM[ (wp,i - wb,i) x (Rb,i - Rb) ]
选择效应 = SUM[ wb,i x (Rp,i - Rb,i) ]
交叉效应 = SUM[ (wp,i - wb,i) x (Rp,i - Rb,i) ]
```

其中 wp,i 为组合在行业 i 的权重，wb,i 为基准在行业 i 的权重，Rp,i 为组合在行业 i 的收益，Rb,i 为基准在行业 i 的收益。

### 框架二：因子归因分析

**适用场景**：量化基金、风格分析

**核心模型**：
- **Fama-French 三因子**：市场（MKT）、规模（SMB）、价值（HML）
- **Carhart 四因子**：三因子 + 动量（MOM）
- **Barra 多因子**：10+ 风格因子（规模/价值/动量/波动率/流动性/盈利等）

**使用逻辑**：
```
基金收益 = alpha + beta1 x 市场 + beta2 x 规模 + beta3 x 价值 + beta4 x 动量 + ... + epsilon
- alpha > 0：基金经理有真正的选股能力
- beta 暴露：基金经理的风格偏好（大盘/小盘、价值/成长）
- epsilon：残差，不可解释部分
```

### 框架三：风格分析（晨星投资风格箱）

**适用场景**：基金分类、风格漂移监测

**晨星风格箱**[^18]：

|  | 价值 | 混合 | 成长 |
|------|------|------|------|
| **大盘** | 大盘价值 | 大盘混合 | 大盘成长 |
| **中盘** | 中盘价值 | 中盘混合 | 中盘成长 |
| **小盘** | 小盘价值 | 小盘混合 | 小盘成长 |

**判断方法**：
- 通过持仓加权平均的市盈率/市净率确定价值-成长维度
- 通过持仓加权平均的市值确定大-小盘维度
- 风格偏移（Style Drift）：季度间风格箱位置变化 > 2 格需关注

### 框架四：择时与选股能力分析

**核心模型**：
- **T-M 模型**（Treynor-Mazuy）：Rp - Rf = alpha + beta(Rm - Rf) + gamma(Rm - Rf)^2 + epsilon，gamma > 0 且统计显著 --> 有择时能力
- **H-M 模型**（Henriksson-Merton）：用虚拟变量区分牛熊市的 beta
- **CL 模型**（Chang-Lewellen）：分别估计上涨/下跌市场的 beta

### 框架五：风险调整收益指标

| 指标 | 公式 | 含义 | 判断标准 |
|------|------|------|---------|
| **Sharpe Ratio** | (Rp - Rf) / sigma_p | 每单位总风险的超额收益 | > 1 优秀，> 2 卓越 |
| **Information Ratio** | (Rp - Rb) / TE | 每单位跟踪误差的超额收益 | > 0.5 良好，> 1 优秀 |
| **Sortino Ratio** | (Rp - Rf) / sigma_d | 每单位下行风险的超额收益 | 比 Sharpe 更关注下行 |
| **Calmar Ratio** | 年化收益 / 最大回撤 | 收益与极端风险的平衡 | > 1 为良好 |
| **最大回撤** | 峰谷最大跌幅 | 最坏情况损失 | 偏股型 < 30% 为可控 |

**机构 vs 个人**：
- **机构（FOF 管理人）**：使用 Brinson 归因 + 因子分析 + 风格箱，关注 alpha 的持续性和可解释性
- **个人投资者**：主要看历史收益率排名、最大回撤、晨星星级评级

## 1. 头部机构研究方法

### 桥水基金（Bridgewater Associates）

- **全天候策略（All Weather）**[^20]：不预测经济环境，而是构建在所有环境中都有资产表现的组合
- **风险平价（Risk Parity）**：按风险贡献而非资金贡献做配置，使每类资产的风险权重相等
- **四象限框架**：经济增长（上升/下降）x 通胀（上升/下降）= 4 种环境，每种环境配置对应表现好的资产
- **方法论核心**：基于资产与经济环境的结构性关系，而非不稳定的历史相关性

### 高盛（Goldman Sachs）

- **GS SUSTAIN**：ESG 整合投资框架，将可持续发展因素纳入传统估值
- **Top of Mind 系列**：宏观主题研究，整合经济学家+策略师+外部专家观点
- **量化研究**：大规模因子模型+另类数据+机器学习信号生成

### 中金公司（CICC）

- **自上而下框架**：宏观策略团队产出 --> 行业研究团队细化 --> 形成完整的"宏观-中观-微观"研究链
- **深度产业研究**：以产业链视角做行业分析，注重一手调研数据
- **中国特色分析**：对政策解读和政府行为模式的理解是核心竞争力

### 高瓴资本（Hillhouse Capital）

- **长期结构性价值投资**：聚焦"长坡厚雪"赛道[^21]
- **深度研究驱动**：重仓持有前进行 6-12 个月的深度研究
- **关注护城河**：网络效应、转换成本、规模效应、品牌
- **一二级市场联动**：VC/PE/二级市场统一视角

### ARK Invest

- **自上而下 + 自下而上结合**[^22]
- **破坏性创新框架**：聚焦五大创新平台（AI/Robotics/Energy Storage/Blockchain/Multi-omic Sequencing）
- **开放研究**：研究报告和模型公开发布
- **2025-2026 年新实践**：整合 Kalshi 预测市场数据，将事件合约（宏观数据、公司业绩、监管决策的概率定价）纳入投资研究流程

## 3. 常见分析误区与反模式

### 通用误区

| 误区 | 描述 | 正确做法 |
|------|------|---------|
| **幸存者偏差** | 只看成功案例，忽略失败样本 | 纳入退市/清盘样本做全样本分析 |
| **后视偏差** | 事后看一切"显而易见" | 基于当时可得信息做前瞻判断 |
| **锚定效应** | 被历史估值/价格锚定 | 基于当前基本面重新评估 |
| **确认偏差** | 只找支持自己观点的证据 | 主动寻找反面证据（kill the idea） |
| **过度自信** | 对预测精度过于乐观 | 使用概率区间而非点估计 |

### A 股特有误区

| 误区 | 描述 |

## 结论与建议

### 核心结论

1. **框架选择因市场而异**：A 股的政策驱动特征使"自上而下"框架更为有效；美股市场效率更高，"自下而上"的基本面分析更为主流；加密货币需要全新的链上分析思维
2. **数据能力是核心壁垒**：从 Wind 到 Bloomberg 到 Glassnode，数据获取能力和处理能力直接决定分析质量
3. **估值方法没有"万能钥匙"**：每种估值方法都有适用条件和局限性，交叉验证是基本纪律
4. **风险分析常被低估**：好的分析不仅要算出目标价，更要清晰识别和量化风险
5. **另类数据正在改变游戏规则**：65% 的对冲基金已使用另类数据，91% 的基金管理人正在使用或计划使用 AI 辅助投资研究

### 行动建议

- **个人投资者**：从掌握一个市场的核心框架开始，而非试图覆盖所有市场。优先学习 Sharpe Ratio、最大回撤等风险指标
- **从业人员**：在掌握通用方法论的基础上，深耕一个行业/领域建立比较优势。重视数据工具链的熟练度
- **机构投资者**：投资数据基础设施和另类数据能力，整合 AI 工具提升研究效率
